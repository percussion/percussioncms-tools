Shortcut Using the Anchor Tag

The following examples show the use of the ID attribute in the Anchor tag.E

Example 1:

ANCHOR: <a href="samplelinkURL" id="psx-fieldname">

The field named in id="psx-fieldname" replaces the value of the 'href' attribute at run time.

 X-SpLitGenerated XML:

<fieldname>samplelinkURL</fieldname>

Example 2:

ANCHOR: <a href="person.htm" id="psx-person/link">

The value of person/link replaces the 'href' attribute at run time.

 XSpLit-generated XML:

<person>

<link>person.htm</link>

</person>
