Using Exposed and Hidden Syntax Together

You can use both Exposed and Hidden syntax in the same document. For example:

 Design time HTML:

<a href="person.htm" id="psx-person/link"><font size="6" psx-size="psx-person/size">ID:<span>psx-person/empid</span></font></a>

 XSpLit-generated XML:

<person>

<link>person.htm</link>

<size>6</size>

<empid></empid>

</person>

 Input XML Data:

person/link = "person.htm?empid=345"

person/empid = "345"

person/size = "12"

 Output HTML at run time:

<a href="person.htm?empid=345"><font size="12">ID: <span>345</span></font></a>
