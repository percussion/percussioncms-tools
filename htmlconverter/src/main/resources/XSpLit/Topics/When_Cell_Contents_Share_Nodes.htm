When Cell Contents Share Nodes

If fields in different cells of the same row share a common node, the row repeats once for each occurrence of the common node.

For example, if you had the fields psx-product/name and psx-product/feature included in the following HTML code:

<table border="1" width="50%">
<tr bgcolor="lightgrey">
<th align="left">Product Name</th>
<th align="left">Product Features</th>
</tr>
<tr>
<td>psx-product/name</td>
<td>psx-product/feature</td>
</tr>
</table>
</pre>
</body>

At run time, the results would look like the following example. Note that the fields share a common node (product) so the row repeats once for each occurrence of product.

Content: XML

Output: HTML

<products>
<product>
<name>Widget</name>

</product>

<feature>It's fun!</feature>
</product>
<product>
<name>Gizmo</name>
<feature>Low maintenance</feature>
</product>
</products>
