Basic List Rule

About the Root Node

A list item repeats once for each time the first node (left to right) repeats. Any static text that is part of the list item repeats when the list item repeats. Any list item attributes also repeat.

For example, if you had the field names psx-name/last and psx-name/first in the following HTML code:

<ul>
<li><span>psx-name/last</span>, <span>psx-name/first</span></li>
</ul>

The list item contains two fields that share the name node. Each item in the list repeats once per occurrence of name.
