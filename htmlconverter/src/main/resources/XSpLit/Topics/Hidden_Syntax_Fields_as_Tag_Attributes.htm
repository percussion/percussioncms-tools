Hidden Syntax Fields as Tag Attributes

Place fields used for tag attributes in a prefixed attribute added into the HTML tag.

<tagname attribute="dummy attribute value" psx-attribute="psx-fieldname">

The prefixed psx-attribute and its dynamic value from psx-fieldname replaces the dummy attribute and value when XSpLit generates the style sheet.

Example 1:

<font size="6" psx-size="psx-Order/Pointsize">Hello world.</font>

The value of the Order/Pointsize field replaces the 'size' attribute of the <font> tag when the style sheet is merged with generated XML data.

 XSpLit-generated XML:

<Order>

<Pointsize>6</Pointsize>

</Order>

Example 2:

<meta name="keywords" content="to be determined" psx-content="psx-page/Content">

The value of the page/Content field replaces the content attribute of the meta tag when the style sheet is merged with the generated XML data.

 XSpLit-generated XML:

<page>

<Content>to be determined</Content>

</page>
