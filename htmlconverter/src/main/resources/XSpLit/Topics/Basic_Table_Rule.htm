Basic Table Rule

About the Root Node

A table row repeats once for each time the first node (reading the field left to right) is shared by all repeating fields in the row. Any static text in the row repeats when the row repeats. All row and cell attributes (such as border and background color) also repeat.

Note: XSpLit ignores the rowspan and colspan attributes for repeated table rows (but preserves them for non-repeated rows). Use nested tables to achieve the same effect where necessary.
