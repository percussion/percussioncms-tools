Advanced Example

In this example, we have the fields psx-product/name and psx-product/number in the following code:

<form method="POST">
<font size="2">Choose a product:<br></font>
<table>
<tr>
<td><input type="radio" name="R1" size="20" value="none" checked>none<td>
</tr>
<tr>
<td>
<input type="radio" name="R1" value="psx-product/number"/>
<span>psx-product/name</span> (<span>psx-product/number</span>)
</td>
</tr>
</table>
</form>

The Input element repeats once per product, the first shared node. The value of the value attribute will be the value of the products/product/number field. Since the Input field is enclosed in a table cell, all surrounding text and tags (including the two repeating fields product/name and product/number) are repeated as each Input element is repeated.

Since the text block includes both static text and dynamic fields, each dynamic field must also be enclosed by their own SPAN tags. See the Exposed syntax rules for dynamic fields.
