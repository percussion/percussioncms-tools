Exposed Syntax

Use Exposed Syntax if you want to see where dynamic content is and you do not need to see the final page format until run time.

To use Exposed Syntax, insert fields directly inside existing HTML tags or tag attributes. The field names are visible in the HTML page during design time.

You can use both the Exposed and Hidden syntax together in the same HTML page.

Note: When using Exposed syntax, the sample XML file generated by XSpLit contains XML tags without content.
