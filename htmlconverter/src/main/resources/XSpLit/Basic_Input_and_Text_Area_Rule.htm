Basic Input and Text Area Rule

About the Root Node

In a form, an Input or a Textarea element repeats once for the first shared node (going left to right) of the repeating fields in the Input tag.

For example, suppose you have the field psx-product/name in the following HTML code:

<form method="POST">
<p><input type="text" name="T1" size="20" value="psx-product/name" ></p>
</form>

The Input field of type text repeats once per product. The value of the input field is products/product/name.

At run time, the output would look like this:

<product>
<name>Widget</name>
</product>

</products>
