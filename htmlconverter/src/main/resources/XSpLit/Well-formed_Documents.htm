Well-formed Documents

A well-formed document is a document that conforms to the rules for being well-formed. These rules include:

  • All content is contained within the root element.
  • No attribute appears more than once in the same tag.
  • All markups are properly closed after they are opened.
  • All tags are properly nested. In other words, start and end tags do not overlap each other.

For more information about well-formed documents, visit the World Wide Web Consortium website (www.w3.org) or consult an XML reference.
