Shortcut Using the Form Tag

The following examples show the use of the ID attribute in the FORM tag.

Example 1:

FORM: <form action="sampleactionURL" id="psx-fieldname">

The field name in id="psx-fieldname" replaces the value of the 'action' attribute at run time.

XSpLit-generated XML: 

<fieldname>sampleactionURL</fieldname>

Example 2:

FORM: <form action="person.htm" id="psx-person/actiontype">click here</a>

The value of person/actiontype replaces the 'action' attribute at run time.

 XSpLit-generated XML:

<person>

<actiontype>person.htm</actiontype>

</person>
