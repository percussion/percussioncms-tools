Hidden Syntax: Fields as Tag Content

Place fields used for generic content tags inside the ID attribute of the HTML <SPAN> tag.

<span ID="psx-fieldname">sample content</span>

NOTE: Nest the SPAN tag inside other tags to produce the formatting you want.

Example 1:

<P><span id="psx-Name/First">Bob</span></P>

Only "Bob" appears in the previewed page at design time.

XSpLit-generated XML:

<Name>

<First>Bob</First>

</Name>

Example 2:

<td><span id="psx-Product/Part/Number">1234</span></td>

Only "1234" appears in the previewed page at design time.

XSpLit-generated XML:

<Product>

<Number>1234</Number>

</Part>

</Product>
