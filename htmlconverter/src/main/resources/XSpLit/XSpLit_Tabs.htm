XSpLit Tabs

After running XSpLit, you can view and/or edit generated output text.

Tab

What you see

Source HTML

The original HTML file.

Tidied HTML

A HTML document based on the original document you opened. XSpLit generates this document as a basis for the creation of the DTD, XSL, and XML documents. It ensures that the split documents are well-formed. Documents in XML format (in other words, documents that begin <?xml;) are bypassed. The code assumes these documents are already well-formed.

Note that the currently emerging markup languages, such as Handheld Device Markup Language (HDML), Wireless Markup Language (WML) and Telephony Markup Language (TML) are all based on the XML format.  XHTML is a version of HTML adapted to XML standards.

Log

A log of all processing activity, including Error and Warning messages.

DTD

The original DTD derived from the original HTML file. See About XML fields and DTDs.

Form fields DTD

A DTD based on fields defined in HTML forms.

Format : XSL

The style sheet describing data formatting.

Content: XML

A sample data document.

Output HTML

HTML created from the combined XSL and XML data.

Note: In the View menu, you can toggle the tab visibility. (All tabs are visible by default.)
