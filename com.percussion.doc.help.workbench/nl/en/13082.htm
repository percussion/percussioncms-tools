Editing a Field Validation

When you edit a field validation you can add, remove, or change the order of any of the Rules listed in the table and change any of the Rule Details that correspond to the Rules.

For a graphic of the Field Validation editor and definitions of the fields discussed below, see Field Validation Editor.

To edit a field validation:

  1. In the Field Properties editor for the field, click [Validation]

    OR

    from Field and Field Sets editor, select the field and click [Validation].

    The Field Validation editor opens.

  2. To change the order in which the rules are executed, change the position of the rules in the Rule table. Click on each rule that you want to move and click or .
  3. To add a rule to the Rule table, click on the row where you want to add the rule or click on the rule that follows the rule that you want to add and click to create a new row. Add the rule as specified in Adding a Field Validation.
  4. To delete a rule from the Rule table, click on the rule and click .
  5. To change the details (parameters) for a rule, click on the rule in the Rule table and modify the parameters. For information about the parameters, see Default Validations and their Parameters. To apply the modified details, click [Apply].
  6. To change the Validation failure message, enter a new message.
  7. Click [OK].
