Extension Registration Editor for JavaScript

The handler for an extension is defined as Java or JavaScript in the New Extension Registration wizard. If the handler is defined as JavaScript, the following editor opens.

Screenshot of Extension Registration Editor for JavaScript

Field Definitions

JavaScript extension name - Read only. Name for the extension.  Must be unique within the handler and unique across all handlers.

Description - Optional description of the extension.

Version - Version of extension (optional).  Rhythmyx saves each version of an extension.

Context - Category of the extension.

Parameters - Table of parameters that the extension uses.

Parameter Name - Parameter name.

Parameter Type - Parameter type (string, number, Boolean, date).

Parameter Description - Description of the parameter.

Body - JavaScript that performs the function of the extension.

Summary - Description of extension.
