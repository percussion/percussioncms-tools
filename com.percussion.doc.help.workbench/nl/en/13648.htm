External Editors

xxx This is written from preliminary notes in the spec. I sense that this procedure may change xxxx

In Rhythmyx Workbench views, you can open objects that are stored as files in external applications, such as a graphics application or an XML editor.

To open an object in an external editor:

  1. Right-click the object and choose xx Open with external editor xx.

    A dialog prompts the user to enter a name to be given to the edited file. xx why?? doesn't it already have a name ?? xx

  2. Enter a name for the edited file in the dialog and click [OK].

    The object is locked to other operations and users, and it opens in the external application.

  3. Edit and the file in the external application.
  4. Save the file and close the external application.
  5. In the Rhythmyx Workbench, right-click on the object and choose either:
    • Finished editing with external editor - saves edits to object
    • Release lock - does not save edits to object
