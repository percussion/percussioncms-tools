Adding Properties to a JavaScript Extension Registration

For a graphic of the Extension Registration editor for JavaScript and definitions of the fields discussed below, see Extension Registration Editor for JavaScript.

To add properties JavaScript extension:

  1. Open the Extension Registration editor for the JavaScript extension.

    The Description and Context fields are filled in from the wizard.

  2. In Version, optionally enter the version of the extension.
  3. In the Parameters table, define any parameters that the extension uses. In the row for each parameter:
    1. Type in a Name.
    2. Choose a data type in the Type drop list.
    3. Optionally, enter a Description.
  4. In Body, enter the JavaScript that performs the extension's function.
  5. Save and close the editor.
