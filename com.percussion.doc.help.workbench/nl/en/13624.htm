Maintaining Object Access

Access control allows the server to give different permission levels to different users or groups of users. The Rhythmyx Workbench lets you assign access control permissions to each type of object in the system for separate Roles, Communities and Users.

  • Use the Object ACL dialog to set permissions for individual objects.
  • Use the Security dialog to set default permissions for all objects.
  • Use the Add ACL Entry dialog to add Roles, Communities and Users to an ACL.

With the ACL dialogs you can:

  • Set global permissions for designing objects in the Rhythmyx Workbench or for viewing them when they run (for example, for viewing a search in Content Explorer).
  • Set permissions for designing specific objects in the Rhythmyx Workbench or for viewing them when they run (for example, for viewing a search in Content Explorer).
  • Add entries to the ACL for an object.
