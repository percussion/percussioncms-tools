New Template Output Dialog

The New Template wizard displays this dialog after the New Template Type dialog if you chose Local or Shared as the Template Type. If you chose Global or Database publishing XML as the Template Type, the wizard skips this dialog.

This dialog specifies assembler extension used to format the output, and the output form of the Template.

Screenshot of the New Template Output Form dialog

Field Descriptions

Assembler Specifies the assembler extension to use when assembling the output. Options include:

Velocity Assembler

Generates text output (including both HTML and XML) using the Apache Velocity technology. For details about Velocity, see http://jakarta.apache.org/velocity, or the following books:

Joseph D. Gradeck, Mastering Apache Velocity

Rob Harrop, Pro Jakarta Velocity

 

Binary Assembler

Extracts binary files from the Repository.

Dispatch Assembler

Used to calculate which Template to use when more than one Template could be used.

Output: Specifies the output for the Template. The following options are available:

Snippet

The Template produces a Snippet, which will be merged into another output.

Text Page

The Template produces a complete text page, such as an HTML page. If you select this option, you must specify the Global Template option to be used for the output pages. Options include:

Default - Search up the Folder tree and use the first global template found; otherwise, use the global template specified for the Site.

Specified - Always use the specified global template with this Template.

None - Do not use a global template with this Template.

Binary

The Template produces a binary output.
