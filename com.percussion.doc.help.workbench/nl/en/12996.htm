Rhythmyx Wizards

Rhythmyx Workbench wizards enable you to create new instances of objects that appear in the appropriate CMS Navigator view.

A Workbench wizard opens in a separate window from the Workbench. Depending on the amount of information required, the wizard can consist of one dialog or a sequence of dialogs. A box at the top of each dialog includes instructions and displays error information if invalid or insufficient data is entered. When the correct information is entered, the [Finish] button is enabled.

Screenshot of Rhythmyx Wizard

Clicking [Finish]:

  • Creates the object in the appropriate view.
  • Closes the wizard and causes the editor for the object to open in a window on the Workbench. The editor may include fields that are not available in the wizard.

To access the wizard for an object:

EITHER

  1. In the CMS Navigator view for the object, right-click the folder for the object type and choose New > [object name].

    The wizard for the object opens.

OR

  1. In the Menu bar choose File > New.

    The menu displays a submenu that lets you choose a commonly created object or Other to view all other objects. They commonly created objects listed are Content Type, Slot, and Template.

    1. If you want to create one of the commonly created objects (Content Type, Slot, and Template), click the object name.

      The first page of the wizard for the object opens.

    2. If the object name does not appear in the upper part of the submenu, click Other.

      The Select a wizard dialog opens.

      Choose the object under Rhythmyx CMS and click [Next >].

      The wizard for the object opens.

Note: For instructions on using the wizard for a specific object, see the Wizard and Creating topics in the section for the object in this Help.
