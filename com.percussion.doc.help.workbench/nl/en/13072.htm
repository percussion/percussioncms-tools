Maintaining Field Validations

Field validations check the format or value of data entered into a field. For example, a field validation could check if a date field is numeric and uses the format yyyymmdd. Rhythmyx performs a field validation each time it inserts or updates a Content Item.

A field validation is part of the field's definition. If the field is a shared field, the validation will occur in any Content Editor that includes the field.

  • Use the Field Validation editor to create and maintain field validations.

With the editor you can:

  • Create new field validations;
  • Edit field validations; or
  • Remove field validations.
