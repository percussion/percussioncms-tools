Maintaining Content Types

A Content Type defines a specific Content Item's format. Content Types may include local fields that are specific to the Content Type, shared fields that are common to multiple Content Types, and system fields that the CMS defines. The Content Type also defines the Content Editor that displays the Content Type's fields to users for creating or editing a Content Item.

  • Use the New Content Type wizard to create a new Content Type.
  • Use the Content Type editor to complete and maintain Content Types.

You can:

  • Create a new Content Type;
  • Add fields to the Content Type;
  • Set up Item Transforms and Item Validation Rules;
  • Edit the Content Type;
  • Enable and disable Content Types or
  • Delete the Content Type.
