Registering an Extension

To register an extension for use with Rhythmyx, complete the New Extension Registration Wizard.

For a graphic of the wizard and definitions of the fields discussed below, see New Extension Registration wizard.

To register an extension:

  1. In System Design view, right-click on Extensions and choose New > Extension.

    The New Extension Registration wizard opens.

  2. In Extension name, enter a name for the extension.
  3. Optionally, in Description, enter a description.
  4. In Context, choose the appropriate category for the extension.
  5. In Handler choose Java or JavaScript, depending on how your extension is coded. If you choose JavaScript, Supported interfaces and Class name are disabled.
  6. If you have chosen Java as the Handler:
    1. In Supported interfaces choose an interface in the drop list and click [+].

      The interface is added to the box below the drop list.

    2. Choose any number of Supported interfaces.
    3. In Class name, enter the name of the Java class associated with the extension.
  7. Click [Finish].

    The extension registration is complete and a node for the extension appears in the System Design view. The New Extension Registration wizard closes and the Extension Registration editor opens.

Now you can:

  • add properties to the Java extension or add properties and code to the JavaScript extension.
  • save and close the Extension registration editor and continue configuring the extension registration at a later time if required.
