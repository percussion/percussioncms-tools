Content Type Editor - Content Type Tab

The Content Type (first) tab of the Content Type editor displays the fields and field sets in the Content Editor for the Content Type.

Use this tab to:

  • Add fields to the Content Type
  • Edit fields of the Content Type

Field Descriptions

Shared and system fields - Shared fields and system fields that you can add to the Content Type. Shared fields are globally available to all Content Editors and are grouped according to field sets; they are user-defined and usually store data that is common to many Content Types. System fields are globally available to all Content Editors; they hold system maintained data such as the date and the login user.

Fields and Field Sets table - Fields or field sets in the Content Type. Add local fields by entering a name in a cell under Name.

For a description of the remainder of the fields in the tab, see Fields and Field Sets Editor.
