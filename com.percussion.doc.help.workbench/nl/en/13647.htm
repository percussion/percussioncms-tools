File Editors

In Rhythmyx Workbench views, you can open objects that are stored as files in file editors, such as text editors or XML editors that are included in Eclipse or that you include as plugins.

To open an object in a file editor:

  1. Right-click the object and choose Open with.

    A submenu of available file editors opens.

  2. In the submenu, choose an editor.

    The file for the object opens in the editor.
