New Community Wizard

The New Community wizard creates new Communities in Security Design view.

To access the New Community wizard:

  • In Security Design view, right-click on the Communities folder and choose New > Community.
  • From the Menu bar choose File > New > Other. In the Select a Wizard dialog, choose Community.

    Screenshot of New Community Wizard

Field Descriptions

Community name - The name of the Community.

Description - An explanation of the Community and its purpose.

Available roles - The roles that are accessible for assignment to the Community.

Assigned roles - The roles that are allocated to the Community.
