New Extension Registration Wizard

The New Extension Registration wizard allows you to register a Java extension that will appear in the System Design view's Extension section. A Java extension that is registered in the wizard may be functional but a JavaScript extension cannot be functional since you must use the Extension editor to enter the JavaScript code.

To access the New Extension Registration wizard:

  • In System Design view right-click on the Extensions folder and choose New > Extension.
  • From the Menu bar choose File > New > Other. In the Select a Wizard dialog choose Extension.

    Screenshot of New Extension Registration Wizard

Field Definitions

Extension name - Name of the extension. Must be unique within the Handler and Type.

Description - Optional. Description of the extension.

Context - Category of the extension.

Handler - Java class that processes the code in an extension. Choices are Java and JavaScript. If you choose JavaScript, the Supported Interfaces and Class Name fields are disabled.

Supported Interfaces - Interfaces that define the Java extension. Disabled if the Handler is JavaScript.

Class Name - Name of the Java class associated with the extension. Disabled if the Handler is JavaScript.
