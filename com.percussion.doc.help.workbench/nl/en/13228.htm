Editing a Keyword

To change the name of the Keyword, you must access the Keyword's right-click menu in Content Design view. To change Keyword choices use the Keyword editor.

To change a Keyword name:

  1. In Content Design view, under the Keywords node, right-click the keyword, and choose Rename.

    The Keyword name is selected.

  2. Type in the new name.
  3. Click [ENTER].

    The keyword is listed under the new name. The keyword choices are unchanged.

To edit a keyword's choices:

  1. In any view where the Keyword appears, right-click on the keyword and choose Open.

    OR

    Double-click the Keyword.

    The Keyword editor opens with data filled in for the keyword.

  2. Change any of the Labels, Values, or Descriptions in the Choices table.
  3. To change the order of the Keywords, select a Keyword and click to move it up in the list or click to move it down in the list.
  4. To add a keyword choice either click on the first empty row or insert a row by clicking on the row that you want below it and click . Enter a Label, Value, and optionally, a Description for the new keyword.
  5. To delete a keyword choice, click on its row and click .
  6. Save and close the Keyword editor.
