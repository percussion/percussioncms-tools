Deleting Slots

If you delete a Slot, be sure the change the value of any slotname parameter in any #slots Velocity macros that match the name of the Slot you deleted. If you do not change the value of the slotname parameter of these macros, assembly of the Template containing that macro will fail, causing the failure of any page containing that Template.

Any XSL Variants that use a Slot that has been deleted will also fail. so be sure to update any XSL templates that contains the name of a Slot you delete.

To delete a Slot, in Assembly Design view, select the slot you want to delete, right-click, and from the popup menu choose Delete.

If you delete a Slot, assembly of any Template or Variant that includes that Slot will fail, and any Templates or Variants that contain the Slot that failed will also fail.
