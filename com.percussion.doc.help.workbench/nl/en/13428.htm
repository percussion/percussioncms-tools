Creating a Custom AuthType

To create a custom AuthType:

  1. Define a new Keyword value under Authorization Types for the new AuthType.
  2. Add a new value to the AuthType properties file. This value must match the Choice Value of the new Authorization Type Keyword. The format is:

    authtype.[numeric value for authtype]= [application name]/[resource name]

  3. In System Design view, expand the Configurations folder and open the Navigation object.

    Add the following line to the opened file:

    authtype.[numeric value for authtype].validflags=i,y,s

  4. Create an application with a resource to define the LinkURL for the related content.
