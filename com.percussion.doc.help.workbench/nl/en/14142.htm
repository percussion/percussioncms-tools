Default System Fields

sys_communityid - The Community associated with the Content Item. Usually the login user's Community ID.

sys_contentexpirydate - The date the Content Item is to be removed from a Site.

sys_contentstartdate - The date the Content Item is to be published to a Site.

sys_currentview - [field used for internal processing]

sys_hibernateversion - [field used for internal processing]

sys_lang - The Locale associated with the Content Item. Usually the login user's Locale.

sys_pathname - The name and path to use when publishing the file.

sys_pubdate - The date the Content Item was published.

sys_reminderdate - A date that a reminder (of any type) is to be emailed.

sys_suffix - The file suffix to use when publishing the Content Item.

sys_title - The system title of the Content Item.

sys_workflowid - The Workflow associated with the Content Item.
