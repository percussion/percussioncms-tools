$rx.ext

The method of this function allows you to call an extension.

$rx.ext.call

Returns Object

Calls the specified extension, with the specified parameters. Returns whatever the requested extension returns.

Name

Type

Description

name

String

Name of the extension to call

parameters

String

Parameters to submit to the requested extension. Up to 20 parameters can be submitted as strings.
