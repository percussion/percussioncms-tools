CMS Navigator - Assembly View

The Assembly Design view of CMS Navigator displays Assembly objects and the sub-categories and objects beneath them.

The main objects in the CMS Navigator Assembly view are Slots, Templates, Global Templates, and Item Filters. Below, descriptions of these objects and their children follow the graphical representation of the Assembly view.

Screenshot of Assembly Design View

Slots Folder

Slots folder includes folders representing categories of Slots and any Slots that are not inserted in folders. Inline Slots are also included; they cannot be edited, but you can modify the templates that they contain. The Slot folders are:

  • user-defined folders - Any folders that users create to contain user-defined categories of Slots. Our example includes the user-defined category Images, which holds Slots that would contain image Templates.

Each Slot can include:

  • Allowed Content Types - Includes references to Content Types allowed in the Slot. Referenced Content Type objects appear under the Content Types node in the same view. To add a reference to a Content Type under a Slot, add the reference in the Slot Editor or drag the Content Type from the Content Types node in Content Design view, and drop it on the Allowed Content Type for the Slot.

    Each Content Type reference can include:

    • Allowed Templates - Includes references to Templates allowed in the Content Type. To add a reference to a Template under a Content Type, add the reference in the Slot Editor or drag the Template from the Templates node in Assembly Design view, and drop it on the Allowed Templates folder for the Content Type.

Templates Folder

Templates folder includes folders that represent categories of Templates:

  • Shared folder - Provided by default; contains Templates that are not defined in association with a specific Content Type. The Shared folder may include user-defined subfolders.

    Each Shared Template can include:

    • Allowed Content Types - Includes references to Content Types that are associated with the Template. To add a reference to a Content Type, drag the Content Type from Content Design view and drop it on the Associated Content Types folder (or copy it and paste it).
    • Contained Slots - Includes references to Slots included in the Template. To add a reference to a contained Slot, add it in the Templates editor or drag it from the Slots folder in Assembly Design view and drop it on the Contained Slots folder.
  • Type Specific folder - Provided by default; contains templates that are defined in association with a specific Content Type.

    Each Type Specific Template includes:

    • Allowed Content Type - Includes a reference to the Content Type that is associated with the Template.
    • Contained Slots - Includes references to Slots included in the Template. To add a reference to a contained Slot, add it in the Templates editor or drag it from the Slots folder in Assembly Design view and drop it on the Contained Slots folder

Global Templates Folder

Global Templates folder contains templates that define the material surrounding a Web Page, such as navigation panels and banners.

Each Global Template can include sub folders for:

  • Contained Slots - Includes references to Slots included in the Template. To add a reference to a Slot, either add it in the Template editor or drag it from the Slots folder in Assembly Design view and drop it on the Contained Slots folder.

Item Filters Folder

Item Filters folder includes all available Item Filters. Item Filters filter the list of Content Items to be published.
