Creating an AuthType Keyword

When creating a custom AuthType, you will need to add a choice for the AuthType to the Authorization Type Keyword. For details about adding a new Keyword choice see (link to appropriate topic).

The value in the Choice Value field is used to choose the Rhythmyx resource to process the AuthType. Rhythmyx tests this value against the values defined in the Authtypes.properties field to select the correct resource. Best practice is to use a value over 300 to ensure that your value does not conflict with a value already defined in the system.
