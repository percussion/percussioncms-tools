Application Editor

The Workbench displays the Application editor when you double-click on an XML Application object in the XMLS Server Explorer. Use the Application editor to create and maintain XML applications.

NEED SCREENSHOT OF APPLICATION EDITOR

When you open the Application editor, it automatically opens the Main tab, which displays all of the resources in the application. You can add new Resources to the application in this tab. Different icons represent query (NEED NEW SCREENSHOT) and update NEED NEW SCREENSHOT resources.

To access a Resource, double-click on it. The Workbench opens it in a new tab.

NEED SCREENSHOT OF APPLICATION EDITOR SHOWING A RESOURCE TAB SELECTED

Any XSL stylesheets or HTML files associated with the application are displayed in separate tabs.
