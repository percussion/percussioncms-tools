New Keyword Wizard Choices Dialog

The Choices dialog is the second dialog of the New Keyword wizard. Use this dialog to specify the choices for the Keyword.

Screenshot of New Keyword Wizard, Choices dialog

Field Descriptions

Choices - Table of keyword choices.

Label - Name of keyword choice displayed to users.

Value (table) - Value of keyword choice stored in Rhythmyx.

Description - Optional. Description of keyword choice.

Value (box) - Read only. Same as Value of the choice selected in the table.
