Creating a Keyword

To create a keyword and keyword choices, complete the New Keyword wizard.

For a graphic of the wizard and definitions of the fields discussed below, see New Keyword Wizard.

To create a keyword and keyword choices:

  1. In Content Design View, right-click on the Keyword node and choose, New > Keyword.

    The Keyword wizard opens to the first page (Keyword dialog).

  2. In Keyword name, enter the name for the keyword.
  3. Optionally, in Description, enter a description of the keyword.
  4. Click [Next].

    The Keyword wizard opens to the second page (Choices dialog).

  5. Optionally, in the Choices table, enter keyword choices. Keyword choices appear in drop lists and other controls in the order in which you enter them.

    For each keyword choice:

    1. Under Label, enter the name for the keyword choice that will appear to users.
    2. Under Value, enter the value for the keyword choice that Rhythmyx will store.
    3. Under Description, optionally enter a description of the keyword choice.
  6. Click [Finish]

    The Keyword editor opens. The keyword and keyword choices appear in Content View under the Keyword node.

Now you can:

  • In the Keyword editor, make changes to the keyword choices.
  • Add the keyword to a field control.
