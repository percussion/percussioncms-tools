Using the WHERE Table to Select Data

To select data using the WHERE table:

  1. Click the Use WHERE table radio button. (NOTE: This is the default option).
  2. Click in the Variable column of the first empty row.

    Rhythmyx displays a popup menu with the options Function and Single Value.

  3. To enter a single value, choose Single Value. Rhythmyx displays the Value Selector, which you can use to choose the value you want to enter. When entering values for this column of the selector, you will generally choose a Backend Column.
  4. To enter a database function, choose Function. Rhythmyx displays the Function Properties dialog.

    Select the function you want to use from the Select function drop list. If the function requires parameters, click the Value column for each parameter to display the Value Selector to enter the value for the parameter. The following functions are available:

    • DAY
    • CURRENT_DATE
    • CURRENT_TIMESTAMP
    • LEFT
    • LOWER
    • LTRIM
    • MONTH
    • RIGHT
    • RTRIM
    • SOUNDEX
    • SUBSTRING
    • TRIM
    • UPPER
    • YEAR
    • IN-NUMBER
    • IN-TEXT
    • IN-NUMBER-ARRAY
  5. Click in the Op column and choose an operator for the row.
  6. Enter a Value to limit the data selected.
  7. If you selecting multiple columns, enter a Boolean operator in the Bool column. Options are AND and OR.
  8. To skip the row if the data in the specified database column is null, check the Omit if Null checkbox.
  9. To remove redundant rows from Selector results, check the Make entries distinct checkbox.
  10. Click the [OK] button to save your selection criteria.
