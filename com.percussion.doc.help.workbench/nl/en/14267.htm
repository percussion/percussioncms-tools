Locating Object IDs

Most objects in Rhythmyx have an ID number. In some cases, you may need to know the ID number. For example, if you are using WebDAV you must know the Content Type ID and the Community ID. Or, if you are using Multi-Server Manager, knowing an object's ID number helps you to identify the ID type.

You can locate an object ID by viewing it either in Properties View or the Object Sorter.

To view an object's ID number:

  1. Select the object in any view.
  2. In the Menu bar, choose Window > Show view > Other.

    The Show View dialog opens.

    [graphic]

  3. Under the Basic folder, choose Properties.
  4. Click [OK].

    The Properties view opens. It displays properties for the object.

    The ID is displayed in the format: <host ID>-<type ID>-<object ID>. The portion of the ID that you use in Rhythmyx applications is the object ID (in the above example, 304).

OR

  1. Right-click the object in any view and choose Show Object Sorter.

    After several seconds the Object Sorter opens.

    graphic

  2. Locate the object in the Object Sorter.

    By default, the fourth column displays the ID in the format: <host ID>-<type ID>-<object ID>. The portion of the ID that you use in Rhythmyx applications is the object ID (in the above example, 304).
