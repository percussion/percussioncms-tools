Viewing or Editing Files in Applications

You can view files in the sys_resources folder, rx_resources folder, or any application folder that you have added to the Resources section.

You can edit a file in the rx_resources folder or in any application folder that you have added to the Resources section. Although you can edit files in the sys_resources folder, it is not recommended since the file is overwritten on upgrade.

To view a file:

  1. Right-click the file and choose Open With <editor>.
  2. View the file.
  3. Close the file.

To edit a resource file:

  1. Right-click the file and choose Open With <editor>.
  2. Edit the file.
  3. Save and close the file.
