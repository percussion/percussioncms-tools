Deleting a Content Type

To delete a Content Type:

  1. In Content Design View, right-click on the Content Type and choose Delete.

    A dialog appears asking you to confirm the deletion.

  2. Click [OK].

    The Content Type is deleted.

    • Any existing Content Items of that Content Type can no longer be seen in Content Explorer. You cannot create new Content Items of that Content Type or publishing existing ones.
    • In general, Templates that previously included Content Items of the deleted Content Type assemble without errors. However, automated indexes that include the deleted Content Type continue to search for Content Items of the deleted types and return error messages.
