Conditional Properties Dialog

Use the Conditional Properties dialog to define a set of conditions that must be satisfied before a specific mapping can occur at runtime. Specify conditional mappings if you need to map different records to different XML fields depending on the data contained in those records.

To access the Conditional Properties dialog, on the Mapper, in the row to which you want to assign a conditional mapping, click the condition button , the click on the browse button next to it.

Field Descriptions

Variable That variable whose value you want to test.

Op The comparison operation for the condition. Options include

  • =
  • !=
  • <
  • <=
  • >
  • >=
  • IS NULL
  • IS NOT NULL
  • BETWEEN
  • NOT BETWEEN
  • IN
  • NOT IN
  • LIKE
  • NOT LIKE

Value The value to which the variable will be compared.

bool Boolean operator used to link multiple conditions. Options include AND and OR.
