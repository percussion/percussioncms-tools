Function Properties Dialog

Use the Function Properties dialog to specify the parameter values for a User Defined Function. The Rhythmyx Workbench displays this dialog when you add a new User Defined Function to a mapping. You can also access this dialog by double-clicking on a User Defined Function mapping.

Field Descriptions

Function Read-only. The name of the User Defined Function.

Parameters: Name Read-only. The name of the function parameter.

Parameters: Value The value of the function parameter. Use the Value Selector to specify the value.
