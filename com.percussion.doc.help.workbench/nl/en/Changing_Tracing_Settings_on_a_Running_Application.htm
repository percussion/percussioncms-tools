Changing Tracing Settings on a Running Application

To change the tracing settings on a running application:

  1. Open the application for which you want to change tracing settings.
  2. In the Menu bar, choose Tools > Debugging.

    Rhythmyx will display the Trace Setup dialog, with the name of the application in the Title bar, and showing the current settings.

  3. Change the trace options to match the settings you want.

    Note that you cannot modify the Output Column Width and Enable timestamp only tracing fields on running applications.  To modify these options you must click [OK], save the application, then restart it.

  4. Click [Apply].  This button is only available if an application is running with tracing.

NOTE:  These changes are temporary unless you click [OK] then save the application.
