Window Style Dialog

The Window Style dialog lets you create a style for the window launched when you choose a menu entry. The dialog is launched when you check Launch new window in the General tab of the Menu and Menu Entry editor and click [...] beside Style.

Screenshot of Window Style dialog

Field Definitions

Target name - Name for internally referencing the window.

Height - Height of the window.

Width - Width of the window.

Status bar - Whether or not a status bar (information bar at the bottom of the window) is included.

Menu bar - Whether or not a menu bar is included.

Tool bar - Whether or not a tool bar is included. Default is checked (yes).

Scroll bars - Whether or not scroll bars are included on the sides of the window.

Location - Whether or not a field for entering a URL is included.

Resizable - Whether or not users can resize the window.
