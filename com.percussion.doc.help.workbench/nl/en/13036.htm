Content Type Editor - Field Set Tabs

A field that uses the sys_table control is a field set. A field set contains a set of sub-fields. For example, a Contact field set might include sub-fields for first name, last name, telephone number, and address. For each field set entered in the Content Type tab the editor adds a Field Set tab. The Field Set tabs of the Content Type let you maintain the sub-fields in the field set.

Use this tab to:

  • Edit the Content Type's field sets
