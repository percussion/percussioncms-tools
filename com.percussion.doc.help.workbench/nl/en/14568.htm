Maintaining Shared Field Definitions

A shared field is a Content Type field that you can include in any number of Content Types. Shared fields are stored in shared field sets. Any number of shared field sets compose are defined in a shared field definition file. You can create any number of share field definition files in your system.

  • Use the New Shared Field Definition File wizard to create a new shared field definition file.
  • Use the Shared Field editor to complete and maintain shared field sets and their fields.

You can:

  • Create a new shared filed definition file;
  • Create new fields and field sets;
  • Configure additional search and display properties;
  • Edit a field or
  • Delete shared field definition files and shared field sets.
