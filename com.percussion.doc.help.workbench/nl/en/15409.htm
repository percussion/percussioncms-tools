Adding/Removing an Object from a Community

The types of objects that can be visible to a Community are Content Types, Display Formats, Menus and Menu Entries, Searches, Sites, Templates and XSL Variants, Views, and Workflows. In general, objects stored in files (with the exception of applications) cannot be associated with Communities.

To add or remove an object from one or more Communities:

  1. Open a view that displays the object.
  2. Right-click the object and choose Security.

    The Object ACL Dialog opens.

  3. Follow the instructions in Editing Object Access for changing the object's association with one or more Communities.

To add or remove several objects from a single Community, it may be easier to use the Community Visibility view:

To add several objects:

  1. Open Community Visibility view.
  2. Multiple-select the objects and drag and drop or copy and paste them onto the Community.

To remove several objects:

  1. Multiple-select the references to objects.
  2. Right-click, and choose Delete.
