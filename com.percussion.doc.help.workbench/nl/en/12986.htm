Default Rhythmyx Perspective

A Rhythmyx perspective is a particular selection of Rhythmyx Workbench screens that you can view through Eclipse. The default Rhythmyx perspective appears as follows:

Screenshot of Default Rhythmyx Perspective

Note: If you have installed the Rhythmyx Workbench as part of an existing Eclipse environment, your default Rhythmyx perspective will appear differently.

The default Rhythmyx perspective includes five main CMS views or windows which represent the structure of the Rhythmyx System. Collectively, we refer to these views as the CMS Navigator. Each view uses a tree format to display a category of CMS objects and the relationships of the objects within the category. The CMS Navigator presents a model of the CMS in which various CMS elements fit into the categories:

  • Content Design
  • Assembly Design
  • UI Elements Design
  • Security Design
  • System Design

The default Rhythmyx Perspective includes additional views for looking at the system's Communities and for performing other Rhythmyx Workbench functions. To modify the default Rhythmyx perspective or create additional perspectives, see the section Working with Perspectives in the Eclipse help.

Views display CMS objects in a manner which displays their relationships to one another. Note that some of the views contain references to objects in other views rather than the objects themselves. References to objects cannot be edited. To access functions that you can perform on an object, right-click on the object to open its object right-click menu (context menu).
