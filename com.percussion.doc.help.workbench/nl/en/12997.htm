Help

Various forms of help are available to Rhythmyx Workbench users:

Form of Help

Description

How to Access

Rhythmyx Workbench Help

Online Help specific to the Rhythmyx Workbench.
  1. From the Rhythmyx Workbench, click Help > Help Contents or Help > Dynamic Help.

    The Help opens in a new screen or in the right panel of the interface.

  2. To access Rhythmyx Workbench topics, click the first node, Rhythmyx Workbench Help.

Eclipse Workbench User Guide

Eclipse Online Help
  1. From the Rhythmyx Workbench, click Help > Help Contents or Help > Dynamic Help.

    The Help opens in a new screen or in the right panel of the interface.

  2. To access Eclipse Workbench topics, click the second node, Eclipse Workbench Help.

Context Sensitive Help

Help about the current window or dialog.

Press F1.

Rhythmyx Implementation Guide

Instructions for implementing your Rhythmyx CMS using Content Explorer and the Rhythmyx Workbench.

On your installation CD, open Rhythmyx Implementation Guide.pdf.
