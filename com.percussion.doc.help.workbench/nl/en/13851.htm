Modify a Custom Search

You can modify any of the information for a standard search accessible through the Search Editor for standard searches.

For a graphic of the editor and definitions of the fields discussed below, see Search Editor, Custom Search.

To modify a custom search, change any of the following:

  1. Custom Search label

    Description

    Display Format

    Check or uncheck Unlimited or change the maximum results returned in Max rows returned.

    Click [Customize] to add or remove custom fields that you have included under Search criteria.

  2. If you have changed the location of the URL of the application that performs the query, or you want to use a different query, change the value in Custom search URL.
  3. Save and close the editor.
