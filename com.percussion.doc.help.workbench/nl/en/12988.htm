Database Explorer

Database Explorer displays databases connected to your system. To connect a database to your system, create a data source for the database. Then expand the database to display its tables, and drag and drop any of the tables on applications which can use them when performing queries.

Screenshot of Database Explorer

Database Explorer is visible in the default Rhythmyx perspective for upgrade users. Users who are new to Versions 6.0 and higher of Rhythmyx must open Database Explorer by choosing Window > Show view > Other > Database Explorer in the Menu bar.
