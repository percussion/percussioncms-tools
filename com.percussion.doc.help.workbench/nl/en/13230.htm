Maintaining Locale Objects

Locales enable the localization (L10N) or internationalization (I18N) of a Rhythmyx interface or a Site created in Rhythmyx.

A Locale specifies a language including its regional variation. A system's Locales are defined in Content Design View under the Locales folder of the Localization node. A system with more than one Locale is globalized or internationalized.

The system behaves according to the Locale of the login user. For example, if the login user's Locale is Canadian French, Content Explorer may display labels in Canadian French and automatically create copies of new Content Items in both Canadian English and Canadian French.

Locales use standardized language and country codes, and are written in the format language-country. For example, en-us represents US English.

  • Use the Locale Wizard to create a new Locale.
  • Use the Locale Editor to change the Locale properties.

You can:

  • Create a new Locale.
  • Edit Locale Properties.
  • Delete a Locale.
