Log Database

Rhythmyx can be configured to log various system and user activities.  This information is stored in a SQL database.

Rhythmyx ships with a simple database called InstantDB (idb) that can be used as the default log store.

If messages cannot be logged to the default log store, a back-up mechanism automatically begins.  This storage mechanism is an .XML file called log.xml, and is stored in the root Rhythmyx installation directory.
