Maintaining Item Filters

An Item Filter is a set of one or more Item Filter Rule extensions that filter the list of Content Items to be published. The following Item Filter Rules are provided by Percussion Software:

  • sys_filterByPublishableFlag

    Filters based on value of the Publishable Flag of the State of the Content Item. Requires the sys_flagValues parameter, which can take a single value ("y") or a comma-separated list of values ("y,i").

  • sys_filterByFolderPaths

    Filters based on the path of the Content Item. Requires the sys_folderPaths parameter,which can be a Folder path ("//Sites/EnterpriseInvestments"),or a semicolon-separated list of Folderpaths ("//Sites/EnterpriseInvestments;//Sites/CorporateInvestments"). Each Folderpath can include one or more wildcard characters (% matches 0 or morechatacters, _ matches exactly one character). To publish the contents of a Folder and all of its descendants, append the % character;for example, "//Sites/EnterpriseInvestments/%"would return the contents of the Enterprise Investments root folder and all of its Subfolders.

  • sys_incrementalPublish

    Filters for Content Items that have been modified since the last publishing run. Requires the sys_siteid parameter, which specifies the Site to publish.

  • sys_filterBySitefolder

    Used for cross-site linking. Checks the Folder specified to ensure that the target Content Item for the link being published exists in that Folder, then checks to confirm that the Path to that Folder includes the Site Root for the specified Site. If this test fails, checks whether the Content Item exists on the specified Site at all. If so, the matching path will be published. Otherwise, the link is not published.

Note that you can also write your own Item Filter Rule extensions. For details about writing an Item Filter Rule extension, see the Rhythmyx Technical Reference Manual.

Rules are processed in the order specified.

An Item Filter can refer to, or inherit,another Item Filter.

Use the Item Filter wizard to create a new Item Filter.

Use the Item Filter editor to maintain an existing Item Filter.

You can:

  • create new Item Filters;
  • modify existing Item Filters; or
  • delete Item Filters.
