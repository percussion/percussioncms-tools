Conditional Property dialog

Use the Conditional Property dialog to specify the conditions for a rule on the Rule Editor. The Conditional Property dialog is only available if the Type of the Rule is Condition.

To access the Conditional Property dialog, in the Rule column of the Rule Editor, click the browse button (...)

Columns

Variable The variable being tested. Use the Value Selector to specify the variable.

Op The boolean operator for the condition.

Value The value against which you are testing the Variable. Use the Value Selector to specify the value.

bool Boolean operator joining and evaluating multiple conditions. Options are AND or OR, or null.
