Adding Properties to a Java Extension Registration

For a graphic of the Extension Registration editor for Java and definitions of the fields discussed below, see Extension Registration Editor for Java.

To add properties Java extension:

  1. Open the Extension Registration editor for the Java extension.

    All of the fields have been entered in the wizard except Parameters, Required files, and Required applications.

  2. In the Parameters table, define any parameters that the extension uses. In the row for each parameter:
    1. Type in a Name.
    2. Choose a data type in the Type drop list.
    3. Optionally, enter a Description.
  3. In Required files, enter any number of files that the extension uses:
    1. Either type in a file name or click [...] to browse for a file.
    2. Click [+] to add the file to the box below.
  4. In Required applications, choose all of the applications that use the extension. If you fail to choose an application that uses the extension, you will not effect the running of the extension; however, the Rhythmyx Multi-Server Manager will not automatically package the extension with the application. For more information see the Rhythmyx Multi-Server Management document:
    1. Either type in a file name or click [...] to browse for a file.
    2. Click [+] to add the file to the box below.
  5. Save and close the editor.
