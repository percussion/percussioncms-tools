Keyword Editor

The Keyword editor allows you to modify keyword choices.

To access the Keyword editor:

  • After finishing the New Keyword wizard, click [Finish].
  • In any view that displays Keywords, right-click a Keyword and choose Open.
  • In any view that displays Keywords, double-click a Keyword..

    Screenshot of Keyword Editor

Field Definitions

Keyword name - Read only. The name of the keyword.

Description - Optional. A description of the keyword.

Choices - Table of keyword choices.

Label (table) - Name for keyword choice displayed to users.

Value (table) - Value for keyword choice stored in Rhythmyx.

Description (table) - Optional. Description of keyword choice.

Value (box) - Read only. Same as Value of the choice selected in the table.
