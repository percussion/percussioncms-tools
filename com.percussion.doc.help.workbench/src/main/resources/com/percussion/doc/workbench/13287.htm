Editing a Java Extension Registration

When you edit a Java extension registration, you can modify any of the information accessible from the Extension Registration Editor for Java except the extension's name.

For a graphic of the Extension Registration editor for Java and definitions of the fields discussed below, see Extension Registration Editor for Java.

To edit properties of a Java extension:

  1. Open the Extension Registration editor for the Java extension.
  2. Change the Description, Class name, Context, or Parameters.
  3. In Supported interfaces, Required files, or Required applications, choose any additional items from the drop lists and click [+] to add them to the text boxes below the fields.
  4. In the text boxes below the fields for Supported interfaces, Required files, and Required applications, click any item and click [X] to delete the item from the list.
  5. Save and close the editor.
