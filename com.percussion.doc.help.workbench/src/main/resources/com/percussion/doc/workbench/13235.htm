Maintaining Translation Settings

A translation setting defines a Community and Workflow that are automatically assigned to Content Items of a Content Type when they are created in a certain Locale. Translation settings are used when translation copies of a Content Type require Community or Workflow properties that are different from the Content Type's defaults.

Note: Translation copies of Content Items are copies of Content Items that have a different Locale and are frequently translated into a different language. For more about Automated Translation and Translation copies see the document Internationalizing and Localizing Rhythmyx.

  • Use the Translation Settings editor to enter and change translation settings.

With the Translation Settings editor you can

  • Create a new translation setting.
  • Modify a translation setting.
  • Remove a translation setting.
