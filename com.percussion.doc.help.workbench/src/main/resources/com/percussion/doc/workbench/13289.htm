Maintaining Configuration Files

Configuration files specify the default settings or properties for features in the Rhythmyx Server or Workbench. The Configurations section of System Design view lists the configuration files that are available for viewing or editing. To alter your set-up options or create custom settings for features in your system, edit the appropriate configuration file.

To view or edit a configuration file:

  1. In System Design view, under the Configurations node, right-click the configuration file that you want to view or edit and choose Open With > [editor].
  2. The file opens in the editor that you have chosen.
  3. View or make edits to the file.
  4. Close or save and close the file.
  5. Depending on the file that you have changed, you may have to restart the Rhythmyx Server or the Workbench to view your changes.

Now you can:

  • Test the new configuration. For example, if you have changed your logging settings, you may choose to run an application and then check its log files.
