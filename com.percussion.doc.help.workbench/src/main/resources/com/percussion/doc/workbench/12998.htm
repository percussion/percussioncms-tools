Setting Rhythmyx Workbench Preferences

The Preferences dialog is an Eclipse dialog for setting user preferences. In Rhythmyx the Preferences dialog includes a Rhythmyx node with a Security sub-node.

To set Rhythmyx Workbench Preferences:

  1. In the Menu bar, choose Window > Preferences.

    The Preferences dialog opens.

  2. Click the Rhythmyx node to display general Rhythmyx preferences.

    Screenshot of Preferences dialog

Field Definitions

Show hidden applications - Show any applications that are hidden by default. Unchecked by default.

Show warning for read-only objects on open - When users attempt to open objects to which they do not have write-access, a dialog asks them if they want to open the object as read-only. To automatically open the object in read-only mode without displaying the dialog, uncheck this option. Editors displaying a read-only object include (read-only) in the title name. Checked by default.

Show deprecated functionality - When checked, deprecated java extensions are shown. Unchecked by default.

Automatically connect when workbench opens - If a connection in the Connection dialog is marked as default, logs user in to the Workbench without opening the Connections dialog. Unchecked by default.

Default Content Editor date format - The default Content Editor date format appears as mm/dd/yyyy. Choose any option in the drop list to change the default format. Unchecked by default.

Click on the Security subnode to display the Security preferences page.
