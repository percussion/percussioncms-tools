Changing Package Association to Communities

To change package associations in the "Show by Packages" view:

  1. Select the package whose associations you want to change.
  2. Click the [Edit Assocations] button

    Package Manager displays the Edit Community Associations dialog.

  3. In the Available list, choose the Communities you want to associate with the package. Standard multi-select options are available.
  4. Click the (arrow graphic) to move the Communities to the Selected list. You can also use drag and drop to move Communities.
  5. To remove Community associations, move them from the Selected list to the Available list.
  6. Click the [OK] button.

    The system saves your changes.
