Changing Community Association to Packages

To change package associations in the "Show by Communities" view:

  1. Select the Community whose associations you want to change.
  2. Click the [Edit Associations] button.

    Package Manager displays the Edit Package Associations dialog.

  3. In the Available list, choose the packages you want to associate with the Community. Standard multi-select options are available.
  4. Click the (arrow graphic) to move the packages to the Selected list. You can also use drag and drop to move packages.
  5. To remove package associations, move them from the Selected list to the Available list.
  6. Click the [OK] button.

    The system saves your changes.
