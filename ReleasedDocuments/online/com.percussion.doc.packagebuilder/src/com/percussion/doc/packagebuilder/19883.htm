Generating a Configuration Definition File

You can generate a basic configuration definition file for your package. The file will include a configuration bean for each design object specified in the descriptor, with default configurations for each configurable property of each design object

To generate a configuration definition file:

  1. On the Package Descriptor List dialog, select the descriptor for which you want to create a configuration definition.
  2. In the Menu bar, choose Package > Export.

    The Package Builder displays the Select Operation and Target Directory dialog.

  3. Select the Create Config Def radio button.
  4. Enter the Directory where you want to create the configuration definition. Use the browse button to activate a browse dialog you can use to choose the directory.
  5. Click [OK].

    Package Builder creates the configuration definition. When this process is complete, Package Builder returns a message notifying you that the file has been created in the location you specified.

  6. Click [OK].
