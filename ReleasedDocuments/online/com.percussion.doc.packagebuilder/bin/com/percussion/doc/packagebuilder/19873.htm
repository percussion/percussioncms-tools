Descriptor Definition Dialog General Tab

Use the General tab of the Descriptor Definition dialog to maintain general data for the descriptor.

Field Descriptions

Package Name Name of the package that will be built from this descriptor. Also used as the name of the descriptor.

Description Free-form description of the package.

Version The Version of the package built from the descriptor.

CMS Minimum Version The minimum version of the Percussion CMS Server to which the package built from this descriptor can be installed.

CMS Maximum Version The maximum version of the Percussion CMS Server to which the package built from this descriptor can be installed.

Publisher Name Name of the organization providing the package built from this descriptor.

Publisher URL URL of the Web site of the organization providing the package built from this descroptor.
