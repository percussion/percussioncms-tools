Descriptor Definition Dialog Dependencies Tab

Use the Dependencies tab to review the packages required to support the package built from the descriptor, and to specify any additional packages your want to add to support the package built from the descriptor.

Field Descriptions

Dependent Packages for Selected Elements List of packages that must be installed to ensure that the design elements in the package built from this descriptor function correctly. These packages must be installed on the target server before you can install the package built from this descriptor.

Additional Required Packages List of packages available on the source server. Packages that have been checked must be installed on the target server before you can install the package built from this descriptor.
