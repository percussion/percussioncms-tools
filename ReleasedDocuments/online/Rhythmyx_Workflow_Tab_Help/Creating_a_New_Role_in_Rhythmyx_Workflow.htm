Creating a New Role in Rhythmyx Workflow

To function properly, any Role defined in the Workflow Editor must also be defined in the Server Administrator. Defining the Role in the Server Administrator associates individual users with the Role so they can access content items.  To access Role Maintenance in the Server Administrator, click the button. For more details, see "Maintaining Server Roles"

To create a Role in Rhythmyx Workflow:

  1. On the Edit Workflow page click Role in the New box.

    Rhythmyx displays the New Role for Workflow page, which is a blank version of the Edit Role page.

  2. Enter the Name of the Role and optional Description.  
  3. Click [Save].

Rhythmyx adds the Role to the workflow.

To allow a Workflow to apply to more than one Community, do not assign Roles associated with Communities to Workflows. The user assigned to a Workflow Role may have a separate Community Role.
