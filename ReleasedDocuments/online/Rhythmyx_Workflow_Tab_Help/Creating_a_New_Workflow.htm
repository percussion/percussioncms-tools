Creating a New Workflow

To create a new Workflow:

  1. Access the Workflow Editor.
  2. Click New Workflow.
  3. Rhythmyx displays the New Workflow page, which is a blank version of the Edit Workflow page.
  4. Enter the Name of the new workflow and the Workflow Administrator Role.  You can also enter a free-form Description of the workflow.  
  5. Click [Save].

    Rhythmyx creates the workflow and returns to the Workflow Editor.  Edit the workflow to add States and Roles.
