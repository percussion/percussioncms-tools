Configure New Searches for Communities Dialog

The Configure New Searches for Communities dialog lets you specify the default search dialog that opens when users choose to create a New Search in Content Explorer and the default search dialog that opens when users perform an Active Assembly search. In addition, the Default Dialogs for New Search dialog lets you specify different default New Content Explorer searches and Active Assembly searches for each Community in your system.

To access the Default Dialog for New Search dialog:

  • In UI Elements Design view, right-click on the Searches folder and choose Assign New Searches by Community. Note: The option is disabled if any Search wizards or editors are open.

Using the Default Dialog for New Search dialog, you can

  • Set default Content Explorer New Searches and default Active Assembly searches.

Field Definitions

Default Searches - The default searches used in the system except for those overridden under Community Specific Searches for the specified Communities.

New CX Search - Required. Specifies the pre-defined search used by default in Content Explorer when a user initiates a new search. Options include all pre-defined searches in the system.

New AA Search - Required. Specifies the pre-defined search used by default when a user searches for related content for Active Assembly. Options include all pre-defined searches in the system.

Community Specific Searches - Any values entered here override the defaults specified above for the Community specified.

Community - Lists all Communities in the system

New CX Search - Specifies the pre-defined search used by default for the specified Community in Content Explorer when a user initiates a new search. Options include all pre-defined searches in the system. The default value is Use Default, which indicates the search specified under Default Searches: New CX Search, above.

New AA Search - Specifies the pre-defined search used by default for the specified Community when a user searches for related content for Active Assembly. Options include all pre-defined searches in the system. The default value is Use Default, which indicates the search specified under Default Searches: New AA Search, above.
