New Template Wizard

Use the New Template Wizard to create a new Template object in the Workbench.

To access the New Template Wizard:

  • From the Menu bar choose File > New > Template.
  • In Assembly View, select the Template node (or any of its children), right-click, and from the popup menu choose New > Template.

The New Template wizard consists of the following dialogs:

  • New Template Type
  • New Template Output
  • New Template Name
  • New Template Slots
  • New Template Content Types.
  • New Template Target Tables
