Removing a Read-only Rule

Since a read only rule is a set of rules that are applied to a field, you do not delete the read only rule, but remove all of the read only rules applied to the field.

For a graphic of the Field Editability editor and definitions of the fields discussed below, see Field Editability Editor.

To remove a read only rule:

  1. In the Field Properties Editor, click [Read Only].

    The Field Editability editor opens.

  2. Click on each Rule in the Rule table and click .
  3. Click [OK].
