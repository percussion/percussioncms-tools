Editing Properties of a Field's Control

You can edit a field control's parameters, properties, and choices.

For graphics of the Display Control Properties editor's tabs and definitions of the fields discussed below, see Display Control Properties Editor, Control Tab and Display Control Properties Editor, Choices Tab.

To edit a field's control:

  1. In the Fields and Field Sets editor for the Content Type, find the row for the field and choose the control in the Control field, and click [...] beside the Control field.

    The Display Control Properties editor opens.

  2. To edit parameters and properties, click the Control tab.
    1. Change or add any parameter Names.
    2. Change or add any parameter Values. For information about entering parameter Values, see Adding Properties to a Field's Control.
    3. To delete a parameter Name/Value pair, click on the row and click [Delete].
    4. To change any of the flags under Field Data Properties, check or uncheck the flags.
  3. To edit choices, click the Choices tab.
    1. To change the method you are using to define choices, click a different radio button, and fill in the fields following the radio button. For information about the different radio buttons and their required fields, see Adding Choices to a Field's Control.
    2. To change the choices retrieved or entered, change the fields that follow the selected radio button. For information about filling in the fields, see Adding Choices to a Field's Control.
  4. To change the order in which the control displays choices, change the Sort Order.
  5. To include or change text displayed to the user for an empty entry, enter or re-enter he the text in the Label and Value fields of Display text for empty entry. To change the Include value or the Sort Order value for the empty entry, change the option in the Include or Sort Order fields. For information about entering values in these fields, see Adding Properties to a Field's Control.
  6. Click [OK] to save the new properties and return to the Field and Field Sets editor.
