Adding an Application to CMS Files

To add an application to the CMS Files folder in System Design view:

  1. Create an application without a query, update or non-text resource. Add any static files to the application. See Creating a New Application for instructions on creating an application.
  2. The application and its contents appear under the CMS Files folder.
