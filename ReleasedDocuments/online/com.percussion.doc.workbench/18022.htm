Template Editor Target Table Tab

The Target Table tab is included in the Template editor when the Template is defined as a database publishing Template when it is created. Use this tab to specify the datasource that will be used to connect to the target database, and the tables to which Content Item data will be published.

Field Descriptions

Datasource The datasource that will be used by this Template to find the database to which to publish output.
