Modifying an Item Filter

To modify an Item Filter:

  1. In Assembly Design view, double-click on the Item Filter you want to edit.
  2. The Rhythmyx Workbench displays the Item Filter dialog.
  3. You can change the Description, change the value in the Inherit from drop list, or change the Associated Authorization Type. An Authorization Type should be associated with any Item Filter that will be used for publishing. Item filters used for unpublishing do not need an Authorization Type association.
  4. To add a Rule, double-click in an empty row of the Rules table, then click the drop arrow and select the Rule you want to add. Options include all Rules defined in the system.
  5. To add parameters to a Rule:
    1. Double-click on a Rule, the click on the browse button .

      The Rhythmyx Workbench displays the Extension Parameters dialog.

    2. Specify a value for each parameter.
    3. Click the [OK] button to return to the Item Filter dialog.
  6. The delete a Rule, select the Rule you want to delete and click the Delete button .
  7. On the Button bar of the Rhythmyx Workbench, click the save button to save the Item Filter.
