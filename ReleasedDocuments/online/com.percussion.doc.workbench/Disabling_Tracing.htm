Disabling Tracing

Disabling tracing is stopping tracing on an application that is not currently running.  To disable tracing:

  1. In the Rhythmyx Workbench, go to the XML Server view.
  2. Select the application for which you want to disable tracing, right-click, and from the popup menu, choose Debugging.

    The Rhythmyx Workbench displays the Trace Setup dialog, with the name of the application in the Title bar, and showing the current settings.

  3. Uncheck the Enable Tracing checkbox.
  4. Click [OK].
  5. Save the application.
