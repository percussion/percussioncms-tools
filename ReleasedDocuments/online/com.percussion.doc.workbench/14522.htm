Exit Properties Dialog

The Exit Properties dialog allows you to parameters for an extension.

Screenshot of Exit Properties dialog

To use the Exit Properties dialog:

  1. Fill in parameter values in the Parameter name column, and Values in the Value column. Rhythmyx interprets text that you enter directly in the Value column as a literal. Select the browse button to create an expression in the Value Selector.
  2. Enter a description of the parameter in the Parameter Description field. (Optional)
  3. Click [OK] to return to the previous screen.
