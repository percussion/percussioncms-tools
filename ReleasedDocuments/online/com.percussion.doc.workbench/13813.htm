Extension Registration Editor

Although an extension registration object is created in the wizard, you can enter additional properties in the editor. A JavaScript extension is not functional until you enter the JavaScript code in the editor. You also use the appropriate Extension Registration editor to edit existing extension registrations.

To access the Extension Registration editor:

  • After completing the New Extension Registration wizard click [Finish].
  • Right-click on the extension in any view that displays it and choose Open.
  • Double-click on the extension in any view that displays it.

A different Extension Registration editor opens depending on the handler specified:

  • Extension Registration editor for Java
  • Extension Registration editor for JavaScript
