Field Transform Editor, Output Tab

Use the Output tab of the Field Transform editor to define field output transforms:

Screenshot of Field Transform Editor, Output Tab

Field Descriptions

Transforms: Rule - List of Transforms selected in Transform type drop list under Rule Details. All transform types are extensions, although common transforms are listed by their function and others are listed under the general category Extensions. Transforms are listed in the Rule table the order in which they are selected.

Rule Details - Section listing the Transform type and the values of parameters, if any, associated with the Transform type. For information about specific transforms, see Default Transforms and their Parameters.

Transform type - Extensions that perform output transforms.

Transform failure message - Message that appears in field in Content Editor if transform fails.
