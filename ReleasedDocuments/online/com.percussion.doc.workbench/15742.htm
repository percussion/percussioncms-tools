Exporting and Importing Rhythmyx Objects

Certain Rhythmyx objects can be exported to files. These files can then be stored in source control to provide a degree of backup and source control protection for these objects. Objects that can be exported include:

  • Content Types

    Content Types are exported to file of the type contentType. The file is named using the name of the Content Type plus this extension. For example, if you exported the rffGeneric Content Type, the name of the export file would be rffGeneric.contentType.

  • Templates

    Templates are exported to files of the type template. The file is named using the name of the Template plus this extension. For example, if you exported the template rffBnBinary, the name of the export file would be rffBnBinary.template.

These files can be imported into the system to restore a deleted or corrupted object.
