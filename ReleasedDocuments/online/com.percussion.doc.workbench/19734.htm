Naming Objects

Spaces and the characters in the following table are not valid in the names of objects that you create in the Rhythmyx Workbench:

Name

Invalid Character

Ampersand

&

Apostrophe

`

At sign

@

Backslash

\

Caret

^

Colon

:

Curly braces

{ }

Equals sign

=

Forward slash

/

Greater than sign [or left angle bracket]

<

Less than sign [or right angle bracket]

>

Percent sign

%

Period or dot

.

Pipe

|

Pound sign (also known as number sign, hash, or octothorp)

#

Question mark

?

Semicolon

;

Square brackets

[ ]

Tilde

~

If you enter an invalid name when you are creating an object in a Wizard, the wizard displays an error message and does not allow you to continue until you correct it.

The object's Label can include spaces or the special characters.
