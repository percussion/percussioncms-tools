View Editor

The View Editor lets you complete or edit a View.

To access the View Editor:

  • After completing the New View wizard click [Finish].
  • Right-click on the View object in any view that displays it and select Open.
  • Double-click the View object in any view that displays it.

Depending on the Type of View you have chosen in the New View wizard, you will either see:

  • The Standard View editor
  • The Custom View editor
