Add ACL Entry dialog

The Add ACL Entry dialog allows you to add Roles, Communities, and users to the ACL for a certain object.

To access the Add ACL Entry dialog:

  • Click [Add ACL Entry] in the Object ACL dialog or the Security dialog.

    Screenshot of Add ACL Entry Dialog

Field Definitions

Roles - All Roles available in the system.

Communities - All Communities available in the system.

Add to ACL - Roles, Communities, and users to add to the ACL.

[Add User] - Clicking this button opens the Add User dialog.

[Delete] - Clicking this deletes the selected entry in the ACL. If the entry is a Role or a Community, it reappears in the Roles or Communities list.
