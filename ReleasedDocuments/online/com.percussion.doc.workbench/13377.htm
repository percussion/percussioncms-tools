Creating a Resource Based on a White Page Assembler

A white page assembler is an assembly query resource generated automatically from a Content Editor. It is the base resource for all assembly resources. Note that since assembly is by definition a query, a white page assembler can only be used to create a query resource.

To create a white page assembler:

  1. Display the Content Explorer.
  2. Find the Content Type for which you want to create the assembly resource.
  3. Right-click on the Content Type icon and from the popup menu, select Copy Source XML.
  4. In the application where you want to create the assembly, right-click and from the popup menu select Paste As Assembler. This menu has two options: Page and Snippet. Select the type of assembler you want to create.

    Rhythmyx adds the resource to the application. The generated resource already includes all mappings and required extensions. All you need to do it add the marked-up HTML page of XSL stylesheet to complete the assembler resource.
