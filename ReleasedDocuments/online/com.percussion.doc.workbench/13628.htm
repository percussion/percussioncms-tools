Item Filter Wizard

Use the Item Filter wizard to define the Name and Description of a new Item Filter. Note that you will need to complete the Item Filter using the Item Filter editor before you can use the Item Filter.

To access the Item Filter Wizard:

  • From the Menu bar, choose File > New > Other. In the Select a wizard dialog, choose Item Filter.
  • In Assembly View, right-click on the Item Filters Node (or one of its children) and from the popup menu, choose New > Item Filter.

    Screenshot of Item Filter Wizard

Field Descriptions

Filter name Name for the Item Filter. Used in internal processing. Must be unique among all Item Filters in the system.

Description Free-form description of the Item Filter.
