Maintaining the Control associated with the Field

A control is the mechanism or interface in a Content Editor through which users enter data into a field. Once the data is entered, the control displays it in the Content Editor. A control such as a rich text editor or a drop list may control aspects of the data's format. Rhythmyx supplies a set of default controls.

  • Use the Display Control Properties editor to add and maintain control properties

With the editor you can:

  • Add properties to a control;
  • Add choices to a control or
  • Edit the properties of a control.
