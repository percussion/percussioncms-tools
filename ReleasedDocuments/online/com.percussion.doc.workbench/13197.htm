New Variant Wizard

Use the New Variant wizard to define the basic data for the Variant.

To access the New Variant wizard:

  • From the Menu bar choose File > New > Other. In the Select a wizard dialog, choose XSL Variant.

    Screenshot of the New Variant Wizard

Field Descriptions

Variant name Name for the Variant. Used in internal processing. Must be unique among all Variant Names in the system.

Label Label displayed for the Variant in Content Explorer. Multiple Variants can share the same Label.

Description Free-form description of the Variant.

Content Type Specifies the Content Type of the Variant.

Output

Snippet

Variant outputs a Snippet. Snippets are assembled into other Snippets or into pages. Because they are merged into other XML files, Variant Snippets must be valid XML.

Text page

Variant outputs a formatted HTML page.

Binary page

Variant produces a binary output.
