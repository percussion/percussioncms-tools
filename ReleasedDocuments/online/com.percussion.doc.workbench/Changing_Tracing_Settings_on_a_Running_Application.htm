Changing Tracing Settings on a Running Application

To change the tracing settings on a running application:

  1. In the Rhythmyx Workbench, go to the XML Server view.
  2. Select the application whose tracing configuration you want to manage, right-click, and from the popup menu choose Debugging.

    The Rhythmyx Workbench displays the Trace Setup dialog, with the name of the application in the Title bar, and showing the current settings.

  3. Change the trace options to match the settings you want.

    Note that you cannot modify the Output Column Width and Enable timestamp only tracing fields on running applications.  To modify these options you must click [OK], save the application, then restart it.

  4. Click the [Apply] button.  This button is only available if an application is running with tracing.

These changes are temporary unless you save the application.
