Creating a Variant

The following procedure assumes that you have an HTML file for the Variant marked up with XSpLit markup.

To create a new Variant:

  1. In the Menu bar, choose File > New > Other. In the Select a wizard dialog, choose XSL Variant.

    The Workbench displays the New Variant Wizard.

  2. Enter a Variant name and Label, and, optionally, a free-form Description of the Variant.
  3. Choose the Content Type for the Variant. Options include all Content Types that exist in the system.
  4. Select the Output for the Variant. Options include:
    • Snippet: Variant outputs a Snippet. Snippets are assembled into other Snippets or into pages. Because they are merged into other XML files, Variant Snippets must be valid XML.
    • Text page: Variant outputs a formatted HTML page
    • Binary: Variant produces a binary output
  5. Click the [Finish] button.

    The Workbench displays the Variant editor with the General Tab selected.

  6. Select a Publish option:
    • Always Always publish this Variant.
    • Never Never publish this Variant.
    • Default Publish this Variant if it is the default Variant of the Content Type.
  7. Specify an Active Assembly Format. Options include:
    • Normal: Default. The variant does not require special treatment for Active Assembly.
    • Auto Index: The variant requires a hidden slot in Active Assembly mode.
    • Non-HTML: The variant should not be formatted for Active Assembly. Use this with content types that store images, file attachments, JavaScript files, and other file types that are not compatible with Active Assembly.
  8. Optionally, specify a Location Prefix or a Location Suffix, or both.
  9. Optionally, enter the name of the XSL Stylesheet that contains the formatting for the Variant.
  10. Enter the URL of the Rhythmyx application that assembles the Variant.
  11. To specify Slots for the Variant:
    1. To add one Slot to the Variant, select the Slot you want to add and click the add arrow .
    2. To filter the list of available Slots, enter a string in the Filter field. The Workbench displays only Slots that match your filter criteria.
    3. To add all listed Slots to the Variant, click the double-arrow button.
    4. You can also multi-select Slots by holding down the CTRL key on your keyboard and clicking on the Slots you want to select.
    5. Click the [Next] button.
  12. To specify the Sites that can use the Variant:
    1. To add one Site to the Variant, select the Site you want to add and click the add arrow .
    2. To filter the list of available Sites, enter a string in the Filter field. The Workbench displays only Sites that match your filter criteria.
    3. To add all listed Sites to the Variant, click the double-arrow button .
    4. You can also multi-select Sites by holding down the CTRL key on your keyboard and clicking on the Sites you want to select.
  13. In the Rhythmyx Workbench button bar, click the save button to save the Variant.
