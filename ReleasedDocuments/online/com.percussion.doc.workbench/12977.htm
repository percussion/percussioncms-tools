Connecting to a Rhythmyx Server

Starting a Rhythmyx Workbench session requires opening the Rhythmyx Workbench and connecting to a Rhythmyx Server. Once you are connected to the Server, you can view and edit the objects it contains.

The Workbench can only be connected to one Rhythmyx Server at a time; however, while working in the Rhythmyx Workbench, you can end your connection to one Rhythmyx Server and start a connection to another one.

If you install the Rhythmyx Workbench with the Rhythmyx Server, your default connection is configured to connect to a Rhythmyx Server using the default Port and user. The default server is localhost and the default port is 9992.The default User and Password are admin1/demo. You must change these values if your Rhythmyx server is not local or uses a different port, and enter your own username and password. Make Default Connection is checked. If you check Save password, this configuration checks you in automatically in the future.

Screenshot of Manage Connections dialog

To manage connections to Rhythmyx Servers:

  1. In the Rhythmyx Workbench, in the menu bar, choose File > Connect. The Connections dialog opens.

    Any existing Rhythmyx server configurations appear in the Name list box.

  2. To add a new Rhythmyx server configuration:
    1. Click [New].

      The name New_Connection for the configuration appears in the Name list box and the Name text box. Server is filled with localhost, and Port is automatically set to 9992. The other values are filled in with admin1 and demo. None of the checkboxes are checked.

    2. In Name, optionally enter a new name for the connection.
    3. In Server, enter the name of the Rhythmyx server that you want to connect to, or enter its IP address.
    4. In Port, leave 9992 or enter the Rhythmyx server's port number.
    5. In UID, enter your Rhythmyx user name.
    6. In Password, enter your Rhythmyx password.
    7. If you want the dialog to remember your Rhythmyx password for this server, check Save password.
    8. To make this the default connection, check Make this the default connection.

      If you check both Save password and Make this the default connection, as long as you have not unchecked Automatically connect when Rhythmyx Workbench opens in the Preferences dialog, the Workbench will automatically log you in to this connection in the future.

    9. To connect to the Server over SSL, check SSL.
    10. The default Timeout is 60 seconds. The Timeout defines how long the Workbench will attempt to establish a connection to the specified server before stopping and reporting a connection error. Change the timeout by entering a different amount of time in seconds in the Timeout field.
    11. Click [Apply] to save the configuration without closing the dialog.
  3. To modify an existing Rhythmyx server configuration, click the name of the server configuration in the Names list box and change the values of any of the fields. Click [Apply] to save the changes without closing the dialog.
  4. To delete an existing Rhythmyx server configuration, click the name of the server configuration in the Names list box and click [Delete].
  5. To connect to a configured Rhythmyx server, click the name of the server configuration in the Names list box and click [Connect].

    The Workbench connects to the Rhythmyx server and closes the dialog. The Workbench views now display the objects and settings for the connected server.
