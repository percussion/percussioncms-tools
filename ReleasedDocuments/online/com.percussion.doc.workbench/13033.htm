New Content Type Wizard

The New Content Type wizard allows you to create a new Content Type object that appears in the Content Design view; however, additional details of the Content Type are entered in the Content Type editor.

To access the New Content Type wizard:

  • From the Menu bar choose File > New > Content Type.
  • In Content Design view, right-click the Content Type folder or a folder under it and choose New > Content Type.

The New Content Type wizard consists of the following dialogs:

  • Communities Dialog
  • Workflows Dialog
