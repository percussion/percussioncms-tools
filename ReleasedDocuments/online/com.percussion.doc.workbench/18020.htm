New Template Target Tables Dialog

The New Template Target Tables dialog is only displayed in the wizard if you select the Database radio button on the New Template Type dialog. It is displayed after the General Properties dialog. Use this dialog to specify the datasource that will be used to publish to a database and to specify the target tables for the publishing output.

Screenshot of New Template Target Tables dialog

Field Descriptions

Datasource The datasource that will be used by this Template to find the database to which to publish output.
