Adding Choices to a Field's Control

When you define a field and specify its control, if the control requires choices (such as a drop list or a check box group) add choices to the control.

For a graphic of the Display Control Properties editor and definitions of the fields discussed below, see Display Control Properties Editor, Choices Tab.

To add choices to a field's control:

  1. After choosing a control in the Control field in the Fields and Field Sets editor that requires choices, click [...] beside the Control field.
  2. The Display Control Properties editor opens to the Control tab. Click the Choices tab.
  3. Click one of the radio button choices for methods of defining choices:
    • Click Use a keyword to use choices predefined in Rhythmyx.
    • Click Retrieve from table to use choices stored in a database table.
    • Click Retrieve from XML application to use choices generated by a Rhythmyx application.
    • Click Define choices for this control only to manually enter choices in this dialog.
  4. If you click Use a keyword, in the Name drop list choose a keyword that defines a list of choices. For example, the keyword Seasons might store the choices Spring, Summer, Winter, and Fall. To view the Keywords and keyword entries (choices) in your system, see the Keyword node in Content View.
  5. If you click Retrieve from table, fill in the database and table fields:
    1. In the Data source drop list, choose the data source for the database that you want to connect to. In most cases, the only option is <CMS Repository>.
    2. In Table, choose the database table that stores the choices.
    3. In Label column, choose the table column that stores the names of the choices that will be displayed to users.
    4. In Value column, choose the table column that stores the values of the choices that Rhythmyx will use for processing.
  6. If you click Retrieve from XML application, in URL, enter an application name or click [...]. If you click [...] the Create Choice Lookup Request dialog opens.

    1. In Application name, choose the choice lookup application.
    2. In Resource name, choose the resource that will perform the lookup.
    3. Enter any required parameters and their values in Param name and Value.
    4. Click [OK].
  7. If you click Define choices for this control only, manually enter any number of choices for the control.
    1. In Label, enter the name of the choice that will be displayed to users.
    2. In Value, enter the name of the choice that Rhythmyx will use for processing.
  8. If you have chosen Use a keyword or Define choices for this control only, Default values displays the values of the choices.
  9. Choose a Sort Order.
  10. If you want to have an empty entry and want to display text in it, Check Display text for empty entry.
    1. In Label, enter the text that will be displayed to users.
    2. In Value, enter the value that Rhythmyx will store when users choose the empty entry.
    3. In Include, choose Always to always display the empty entry with the other choices; choose Only if Null to display the empty entry if there are no other choices for the keyword.
    4. In Sort order, choose First to display the empty entry first in the list of choices; choose Last to display the empty entry last in the list of choices, or choose Sorted to display the empty entry according to the Sort order chosen for the choices.
  11. Click [OK] to save the properties and return to the Field and Field Sets editor.

Now you can:

  • Add additional fields in the Fields and Field Sets editor.
  • Click [All Properties] to add more properties to the Field in the Field Properties editor.
