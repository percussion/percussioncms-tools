Shared Field Editor

The Shared Field editor is used to maintain shared field sets and shared fields in shared field definition files.

To access the Shared Field editor.

  • After completing the New Shared Field Definition File wizard, click [Finish].
  • Right-click on the Shared field definition file object in any view that displays it and select Open. (Note that you must click on the Shared field definition file, not the contained Field Sets. The Rhythmyx Workbench does not respond if you click on the Field Sets.
  • Double-click on the Shared field definition file object in any view that displays it.

For a graphic and field definitions, see Fields and Field Sets Editor.
