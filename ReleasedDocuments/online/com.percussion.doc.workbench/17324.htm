Maintaining Read Only Rules

A read only rule is a condition or extension that specifies the conditions under which a field is read only in a Content Editor. For example, a read only rule can specify that the Workflow field is read only to users who are not in the Administrator Role.

  • Use the Field Editability editor to create and maintain read only rules.

With the editor you can:

  • Add a read only rule;
  • Edit a read only rule; or
  • Remove a read only rule.
