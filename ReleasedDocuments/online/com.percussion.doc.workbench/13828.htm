Editing a JavaScript Extension Registration

When you edit a JavaScript extension registration, you can modify any of the information accessible from the Extension Registration Editor for JavaScript except the extension's name.

For a graphic of the Extension Registration editor for JavaScript and definitions of the fields discussed below, see Extension Registration Editor for JavaScript.

To edit properties of a JavaScript extension:

  1. Open the Extension Registration editor for the JavaScript extension.
  2. Change the Description, Version, Context, Parameters, or the Body of the JavaScript.
  3. Save and close the editor.
