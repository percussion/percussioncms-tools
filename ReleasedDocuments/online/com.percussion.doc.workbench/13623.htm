Error Messages

When an error occurs in the Rhythmyx Workbench, a dialog displays all of the exceptions associated with the error. In addition the exceptions and additional error information are stored in an error log. The error log is located in <Rhythmyx root>\workspace\.metadata\.log.

Screenshot of Error Message Dialog

The Error Message dialog includes a message explaining the error and an Exceptions box listing the exceptions associated with the error. Exceptions are listed hierarchically; children of exceptions are listed under them. If you select an exception in the Exceptions box, explanatory information displays below the box.
