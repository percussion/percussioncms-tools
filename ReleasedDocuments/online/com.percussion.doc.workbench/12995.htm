Object Right-click Menus

You can right-click objects, references to objects, and folders in views to access menus with choices of actions that can be performed on the component. Common choices are listed in the following table in the order they appear in the menus.

Choice

Function

Open

Opens the Rhythmyx editor for editing the object, or if no Rhythmyx editor exists, an appropriate external editor for the file type. You can select multiple objects that use Rhythmyx editors; separate editors open for each object. (You cannot select multiple objects for editors that open in a separate window from the Rhythmyx Workbench.)

Open With

For objects represented by files. Displays a list of the possible editors for editing the object for the user to choose from.

Rename

Highlights the object or folder so that the user can enter a new name.

Security

Opens the Object ACL dialog for viewing or editing access to the object.

Copy

Copies the object to the clipboard.

Paste

Available when there is an object on the clipboard and and a folder or object is selected that can include the object or a reference to the object. Behavior depends on the objects and nodes selected. See Dragging and Dropping (Copying and Pasting) for more information.

Delete

Deletes the object from the system. If you delete a reference to an object, the specific reference is deleted, but the object and other references to the object are not deleted.

Refresh

Refreshes the contents of the selected node.

Some right-click menu choices are particular to certain objects or object folders. These choices are described in topics in the sections for the object in this Help.
