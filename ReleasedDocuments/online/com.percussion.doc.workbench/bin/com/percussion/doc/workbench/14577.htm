Creating a Shared Field Definition file

Use the New Shared Field Definition File wizard to create a new shared field definition file.

For a graphic of the wizard and definitions of the fields discussed below, see New Shared Field Definition File Wizard.

To create a new shared field definition file:

  1. In Content Design view, right-click on the Shared Fields folder and choose New > Shared Field File.

    The New Shared Field Definition File wizard opens.

  2. In Shared Definition file name, enter a name for the file. xml is automatically added as the filename suffix.
  3. In First group name, enter a name for the first shared field set.
  4. Click [Finish].

    The New Shared Field Definition File wizard closes and the Shared Field editor opens. The Shared field definition file appears under the Shared Fields folder in Content Design view.

Now you can:

  • Save and Close the Shared Field editor and add shared field sets later.
  • Create shared fields sets and their fields now.
