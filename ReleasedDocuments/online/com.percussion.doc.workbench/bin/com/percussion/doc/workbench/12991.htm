Object Sorter

The Object Sorter helps users locate objects that are embedded in the navigation tree of another view by sorting all of the objects by name and listing them at the same hierarchical level. Users can sort by any column in the view.

graphic

To use the Object Sorter:

  1. In one of the CMS Navigator views, right click on the top node or a sub-node and choose Show Object Sorter.

    If it is not already opened, the Object Sorter opens and displays the objects under the selected node at the same hierarchical level. By default the objects are sorted by the Name column.

  2. To sort by another column, click on the column. The Object Sorter will sort on this column in the future.
  3. Once you find the object that you are looking for, right-click on it to open a right-click menu of options for processing the object.
