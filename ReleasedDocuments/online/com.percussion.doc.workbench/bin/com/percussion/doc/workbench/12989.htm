Community Visibility

Use the Rhythmyx Workbench's Community Visibility view to view or edit the objects that are visible to the members of a Community. The Community Visibility view lists all Communities on the Rhythmyx Server and displays the objects to which each Community has access.

[graphic]

The types of objects that can be visible to a Community are Content Types, Display Formats, Menus and Menu Entries, Searches, Sites, Templates and XSL Variants, Views, and Workflows. If you have modify ACL access to the object, you can drag and drop it from the following views:
