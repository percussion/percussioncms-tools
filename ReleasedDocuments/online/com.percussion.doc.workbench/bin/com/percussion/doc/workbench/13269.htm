Creating a New Content Explorer View

To create a new Content Explorer View that appears in UI Elements Design view, complete the information in the New View Wizard.

For a graphic of the Wizard and definitions of the fields discussed below, see New View Wizard.

To create a Content Explorer View:

  1. In the Menu bar, choose File > New > Other. In the Select a Wizard dialog, choose View.

    The New View Wizard opens.

  2. In the View name field, enter an internal name for the view.

    The View name is automatically entered in Label.

  3. Optionally, change the value in Label.
  4. Optionally, in the Description field, enter a description of the view.
  5. In the Type drop down list, select the view type:
    • Select Standard to create a query view using the View Editor.
    • Select Url to specify a path, relative to the Rhythmyx root, to the Rhythmyx application and resource that generates the view.
  6. In Parent Category, choose a Rhythmyx view parent category. Choices are My Content, Community Content, All Content, and Other Content. The new view will appear in Content Explorer under the Parent view that you choose.
  7. Move any Communities that you do not want to be able to see the view from Visible to these Communities to Available to the Communities.
  8. To reduce the number of Communities displayed in Available Communities, enter a matching term in Filter.
  9. Click [Finish].

    The View is created and appears in UI Elements Design view under the Standard or Custom Search folder depending on the Type of search you have specified. The New Search wizard closes. If you have specified that the search Type is Standard, the Search Editor for standard searches opens. If you have specified that the search Type is Custom, the Search Editor for custom searches opens.

  10. Click [Finish].

    The View is created and can be found on the UI Elements Design view under the Standard or Custom sub-folder of the Views folder. Most of the Views properties are not configured and it is not functional. The New View wizard closes and the either the Standard View Editor or the Custom View Editor opens, depending on the Type selection you made in the within the New View wizard.

Now you can:

  • Configure the properties of the view in the View Editor.
  • Save and close the View Editor and continue configuring the view at a later time.
