Velocity Tab

The Velocity tab of the Template displays the default Velocity editor if the assembler extension for the Template is the Velocity assembler. If the assembler extension for the Templates is Binary or Dispatch, this tab is empty, since these assemblers use only bindings and doe not use Velocity markup.

The Velocity editor consists of two subtabs. The source subtab displays the source HTML for the Template:

Screenshot of Source subtab of the Velocity editor

The Preview subtab displays a formatted preview of the rendered HTML. The Velocity macros are rendered as text rather than calculated.
