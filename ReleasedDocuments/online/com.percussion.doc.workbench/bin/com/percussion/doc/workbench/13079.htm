Deleting a Shared Field Object or Shared Field Set

To delete a shared field definition file:

  1. In the Shared Fields node of Content Design view, right-click on the shared field object and choose Delete.

    The shared field object and all of its shared field sets are deleted.

    • Content Types that currently use any of the shared fields included in the deleted shared field object will not be affected.

To delete a shared field set:

  1. In the Shared Fields node of Content Design view, locate the shared field set under its shared field object.
  2. Right-click on the shared field set and choose Delete.

    The shared field set is deleted.

    • Content Types that currently use any of the shared fields in the deleted shared field set will not be affected.
