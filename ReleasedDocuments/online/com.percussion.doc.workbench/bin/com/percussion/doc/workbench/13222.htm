Adding Properties to a Field's Control

When you define a field and specify its control, add properties to the control if you want to change its default settings.

For a graphic of the Display Control Properties editor and definitions of the fields discussed below, see Display Control Properties Editor, Control Tab.

To add properties to a field's control:

  1. After choosing a control in the Control field in the Fields and Field Sets editor, click [...] beside the Control field.

    The Display Control Properties editor opens to the Control tab. The editor does not have a tab if this is its only window.

  2. In the Parameters table, no parameters are listed. The parameters for each control have default values . To override any of the default values for the parameters:
    1. In Name, choose the parameter whose value you want to override in the drop list.
    2. In Value:
      • Choose User Defined Function to choose a user-defined function that sets the value.
      • Choose Other Value to choose from various other methods for setting the value (such as using a system parameter or a macro).
  3. If you are using the sys_EditLive or sys_EditBox control and want to allow users to insert inline links in the body of the text, check Allow inline links.
  4. If you are using the sys_EditLive or sys_EditBox control and want to remove broken inline links, check Clean up broken inline links.
  5. If the value of the field may contain IDs of components in your Rhythmyx system, and you may deploy a Content Editor containing the field using Multi-Server Manager, check May contain identifiers.

Now you can:

  • Click [OK] to save the properties and return to the Field and Field Sets editor.
  • Click the Choices tab to define choices for a control such as a drop list or check box group.
