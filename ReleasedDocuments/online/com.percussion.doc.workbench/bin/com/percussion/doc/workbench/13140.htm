New Template Slots Dialog

The New Template wizard displays this dialog after the New Template Name dialog. Use this dialog to specify the Slots to include in the Template.

The Available slots column displays all Slots in the system not current in use by the Template. You can filter this list by entering a string in the Filter field.

The Contained slots column lists Slots that are assigned to the Template.
