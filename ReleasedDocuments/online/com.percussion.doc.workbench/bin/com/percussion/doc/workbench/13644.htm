Editors

Editors open in windows on the Rhythmyx Workbench. The Rhythmyx Workbench supports the following types of editors:

  • Rhythmyx editors - graphic editors designed specifically for editing Rhythmyx objects.
  • File editors - editors for editing objects that are stored as files (for example, text editors).

When you attempt to open an object that another user already has open in an editor, a dialog asks you if you want to open the object in read-only mode.
