Parameter definitions

To Lower - Converts characters in field to lowercase.
To Upper - Converts characters in field to uppercase.
ProperCase - Capitalizes the first letter of each word.

Trim - Removes all leading and trailing spaces.

Normalize Date - Converts the date to the format that you specify.

NOTE: You can use the input Normalize Date transform and the output Format Date transform in conjunction with one another to ensure that dates are always entered and displayed for viewing or editing in Content Editors in the same format.

Input format - Date format that you specify.

input

Search for - The search string.

Replace with - The string to replace the search string with.

Set Field - Changes the value of the field according the the specified JEXL expression. The JEXL expression can use the current field or any other field in the Content Editor.

input

JEXL expression - The JEXL expression that sets the value of the field.

[Editor] - Button for opening a JEXL editor.

Map - Maps the value entered in the Content Editor (Input Value) to a value that is stored in the Rhythmyx Repository (Stored Value) .

NOTE: You can use the input Map transform and the output Map transform in conjunction with one another to ensure that certain fields are always entered and displayed for viewing or editing in Content Editors in the same format.

input

Input Value - The value the user has entered in the field.

Stored Value - The value to store in the Rhythmyx Repository when the user enters the Input Value in the field.

Format Date - Converts the date stored in the Repository back to the input format or to the format specified under Use this format.

NOTE: You can use the input Normalize Date transform and the output Format Date transform in conjunction with one another to ensure that dates are always entered and displayed for viewing or editing in Content Editors in the same format.

output

Use inverse of input transform - Converts date retrieved from Rhythmyx Repository to format supplied in Input format field of Normalize date transform.

Use this format - Converts data retrieved from Rhythmyx Repository to format you specify in the text box.

Map - Displays a value in a Content Editor field depending on the value stored for the field in the Rhythmyx Repository. The display value is either the corresponding Input Value in the input Map transform, or it is the Output Value specified in the map for this transform.

NOTE: You can use the input Map transform and the output Map transform in conjunction with one another to ensure that certain fields are always entered and displayed for viewing or editing in Content Editors in the same format.

output

Use inverse of input transform - Converts value retrieved from Rhythmyx Repository to value supplied in Input Value field of input Map transform.

Use this map - Converts value retrieved from Rhythmyx repository to value specified in Output Value in the map in the dialog.

Stored Value - Value stored in Rhythmyx repository.

Output Value - Value to display in Content Editor.

Extension - Transforms the field value according to the specified extension.

input, output

Name - Extension that will transform value of field.
