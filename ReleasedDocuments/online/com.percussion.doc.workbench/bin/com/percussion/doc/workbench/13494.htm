Function Properties Dialog

The Function Properties dialog allows you to specify a user-defined function and its parameters.

To specify a user-defined function and its properties:

  1. Choose a User-Defined Function in the drop list.
  2. The default parameters for the function are entered under Name. Enter any additional parameters under Name. (Some functions do not require all parameters to be specified. See documentation of a specific function in the Technical Reference Manual for more details.)
  3. Under Value, enter literal values for the parameters or click in the row and click [...] to access the Value Selector to use other options to enter values.
  4. Click [OK].
