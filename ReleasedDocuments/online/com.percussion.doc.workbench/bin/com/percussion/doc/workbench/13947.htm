New Content Type Workflows Dialog

The Workflows dialog is the second dialog of the New Content Type Wizard. Use this dialog to specify the Workflows that users can associate with the Content Type and the default Workflow that is associated with the Content Type.

Field Descriptions

Available workflows - Workflows that can be associated with the Content Type.

Allowed workflows - At least one required. Workflows associated with the Content Type.

Filter - Pattern that each Workflow displayed under Available workflows must match.

Default workflow - Required. Workflow that is assigned to Content Items of the Content Type if no other Workflow is chosen when the Content Item is created. When a Workflow is chosen as the Default workflow, if it is not in the Allowed workflows list, it is automatically moved to Allowed workflows. If you move the Default workflow from Allowed workflows to Available workflows, the first Allowed workflow becomes the Default workflow.
