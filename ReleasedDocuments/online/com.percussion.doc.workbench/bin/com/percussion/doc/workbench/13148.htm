Template Editor Sites Tab

Use the Sites tab to specify the Sites on which the Template can be used.

The Hidden on these Sites column lists the Sites for which the Template is not currently available. You can filter this list by entering a string in the Filter field. The Visible on these Sites column lists the Site for which the Template is currently available.

Use the arrow buttons to move Sites from one column to the other.
