Search Editor, Standard Search

The Search Editor for standard searches lets you customize a standard search.

To access the Search Editor for standard searches:

  • After completing the New Search wizard (and selecting the Type Standard), click [Finish].
  • Right-click on the standard search object in any view that displays it and select Open.
  • Double-click the standard search object in any view that displays it.

Using the Search Editor for standard searches you can:

  • Customize a standard search
  • Modify a standard search

Depending on the search engine that your system uses, you will either see:

  • the Standard Database Search Editor or;
  • the Standard Full Text Search Editor
