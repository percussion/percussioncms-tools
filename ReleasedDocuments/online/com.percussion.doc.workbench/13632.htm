Deleting an Item Filter

You cannot delete an Item Filter that is currently used by a Content List. If you attempt to delete an Item Filter that is used by a Content List, they system will return an error message. Remove the Item Filter from all Content Lists before attempting to delete it.

To delete an Item Filter, right-click on the Item Filter you want to delete and from the popup menu, choose Delete.
