New Template General Properties Dialog

The New Template wizard displays this dialog after New Template Type dialog if you chose Global on that dialog; otherwise, this dialog is displayed after the New Template Output dialog.

Screenshot of New Template General Properties dialog

Field Descriptions

Template name Required Name for the Template. Used in internal processing.

Label Label displayed for the Template in Content Explorer.

Description Free-form description of the Template.

Source Specifies the path to the source code file for the template. Typically, the source code file is an HTML file. Click the adjacent browse button to display a browse dialog you can use to find the file.

Available Communities Lists all Communities defined in the system that have not been moved to the Visible in these Communities field. You can filter this list by entering a value in the associated Filter field; only Communities that match the Filter criteria will be listed.

Visible in these Communities Lists the Communities to which this Template will be visible.
