Control Properties Editor

The Control Properties editor lets you add properties to a Content Editor control.

To access the Control Properties editor:

  • After choosing a control in the Control field in the Fields and Field Sets editor, click on the Control name and click [...] beside the Control name.

If the control requires the user to choose from a list of predefined choices (for example, a drop list), the Control Properties editor includes two tabs:

  • Control tab
  • Choices tab

If the control requires the user to type in information (for example, a text box), the Control Properties editor includes a single window that is the same as the Control tab.
