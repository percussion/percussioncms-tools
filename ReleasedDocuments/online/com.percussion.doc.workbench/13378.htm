Creating a Resource from Scratch

To create a resource from scratch:

  1. In the Workbench Menu bar, choose Insert. Menu options include Query and Update. Select the type of resource you want to add.
  2. The Workbench adds the resource to the application.
  3. Double-click on the resource to modify it. The Workbench opens the resource in a separate tab.

A resource created from scratch consists only of a Data Pipe. You must add at least a Backend Data Tank, Page Data Tank, and Mapper to the resource.

If you are creating a query resource, you must also add a Selector, and may add a Results Pager.

If you are creating an update resource, you must add an Updater, and may add a Transaction Manager.
