Creating Slots

To create a new Slot:

  1. In the Menu bar, choose File > New > Slot.

    The Workbench displays the New Slot Wizard dialog.

  2. Enter a name for the Slot in the Slot name field. The name must be unique among all of the Slots in the system. Rhythmyx enters the same value into the Label field by default, but you can change the value in the Label field.
  3. Enter a free-form Description of the Slot.
  4. Choose a Content finder.
    1. Choose the Content Finder from the drop list. Options include:
      • None
      • Regular
      • Automatic
      • Managed Navigation

      For details about the Content Finders, see the New Slot Wizard dialog.

    2. Click the browse button next to the Content finder field to display the Extension editor.
    3. Specify values for the Content Finder parameters.
    4. Click the [OK] button to close the Extension editor.
  5. Click the [Next] button.

    The Workbench displays the Slot editor.

  6. The Type field defaults to Regular. If you want to use the Slot for inline links or inline images, change the value to Inline.

    Implementing an Inline Slot usually requires assistance from Percussion Software Professional Services Organization, so you should choose the Regular radio button unless specifically directed otherwise by an employee of Percussion Software.

  7. The Allowed Relationship Type defaults to Active Assembly. You can change this value to Active Assembly Mandatory or the name of a custom Relationship Type in the Active Assembly Category, if your system includes one.
  8. To specify the Content Types allowed in the Slot:
    1. In the first blank row of the Content Type column, click the drop arrow and choose the Content Type you want to assign to the Slot. Options include all Content Types not currently assigned to the Slot.
    2. In the corresponding row of the Template column, click the drop arrow and choose the Template you want to use to format Content Items of the specified Content Type. Options include all Templates and XSL Variants available for that Content Type.
    3. Repeat Steps a and b for each combination of Content Type and Template you want to assign to the Slot.
  9. You can also change the values in the Label and Description fields.
  10. Click the [Save] button to save your changes.
