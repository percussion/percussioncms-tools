Control Properties Editor, Choices Tab

The Choices tab of the Control Properties editor displays the method chosen for filling a control with choices (such as drop list or check box choices) and the settings required for using that method.

Use this tab to:

Choose a method for adding choices and enter settings required to use that method.

Change the method for adding choices or change settings required for using a method.

Field Descriptions

Use a keyword - Option to choose a keyword (category of choices) from the Name drop list.

Name - Drop list of predefined keywords (categories that include choices).

Retrieve from table - Option to retrieve choices from a database table.

Data source - Database connection information.

Table - Name of table that stores choices.

Label column - name of column that stores display value of choice.

Value column - name of column that stores value that Rhythmyx stores when a user selects the choice.

Retrieve from XML application - Option to generate choices using a Rhythmyx application.

URL - The URL of the application.

Define choices for this control only - Option to manually define the choices for the control in the following table.

Choices Label - Display value of choice.

Choices Value - Value that Rhythmyx stores when a user selects the choice.

Default values - Read only. If you choose Use a Keyword or Define choices for this control only option, this field displays the choices associated with that option.

Sort order (choices) - Order in which choices appear in control.

Ascending - Ascending alphanumeric order.

Descending - Descending alphanumeric order.

User specified - Choices appear in the order specified either when a predefined keyword is configured or in the order defined or received for the control or generated by a Rhythmyx application.

Display text for empty entry - When checked, the control displays text that represents an empty entry.

Display text Label - Text that represents the empty entry.

Display text Value - Value that Rhythmyx stores when the empty entry is chosen.

Include - When to include the empty entry. Options are:

  • Always - Always include the empty entry with the other choices.
  • Only if null - Include the empty entry if there are no other choices for the Keyword.

Sort order (empty entry) - Where the empty entry should appear among other choices. Options are:

  • First - Display first in the list of Choices.
  • Last - Display last in the list of Choices.
  • Sorted - Display according to the Sort order chosen.
