Editing a Visibility Rule

When you edit a field visibility rule, you can add, remove, or change the order of any of the Rules listed in the table and change any of the Rule Details that correspond to the Rules.

For a graphic of the Field Visibility Rule editor and definitions of the fields discussed below, see Field Visibility Editor.

To edit a field visibility rule:

  1. In the Field Properties Editor, click [Visibility].

    The Visibility Rule editor opens.

  2. To change the order in which the rules are executed, change the position of the rules in the Rule table. Click on each rule that you want to move and click or .
  3. To add a rule to the Rule table, click on the row where you want to add the rule or click on the rule that follows the rule that you want to add and click to create a new row. Add the rule as specified in Adding a Visibility Rule.
  4. To delete a rule from the Rule table, click on the rule and click .
  5. To change the values in a conditional, click on the conditional rule in the Rule table, and modify any of the values in the conditional table. Click [Apply] to change the existing rule instead of adding a new one.
  6. To change the parameters for an extension, click on the extension in the Rule table, and click [...] under Rule Details to open the the Parameters dialog. After making changes to the parameters, click [Apply] to change the existing rule instead of adding a new one.
  7. Click [OK].
