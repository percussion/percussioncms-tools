Action Panels

Action Panels include Action Menu options and function like Action Menus.

Action menus appear:

  • When a user clicks a Content Item link that has been pasted into a third-party application.
  • When a user clicks the arrow beside a Content Item in the sample Inbox or Search Portlets in any of the Rhythmyx Portal Connectors.

The options in Action Panels function as they do in Action Menus. Customizations that you make to Action Menus in the Rhythmyx Workbench also apply to Action Panels.
