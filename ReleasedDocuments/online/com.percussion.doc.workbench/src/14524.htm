Variant Editor General Tab

The General tab of the Variant Editor contains general data about the Variant:

Screenshot of Variant Editor General Tab

Field Descriptions

Variant name Read-only. Name of the Variant.

Variant label Label displayed for the Variant in Content Explorer.

Description Free-form description of the Variant.

Publish When to publish Variant.

Always

Always publish this Variant.

Never

Never publish this Variant.

Default

Publish this Variant if it is the default Variant of the Content Type.

Active Assembly Format The format type for this variant in Active Assembly. Options are:

Normal

Default. The variant does not require special treatment for Active Assembly.

Auto Index

The variant requires a hidden slot in Active Assembly mode.

Non-HTML

The variant should not be formatted for Active Assembly. Use this with content types that store images, file attachments, JavaScript files, and other file types that are not compatible with Active Assembly.

Location Prefix Optional prefix for the filename of the published Template. Useful when relying on a single Location Scheme, which could result in multiple output pages with the same filename.

Location Suffix Optional prefix for the filename of the published Template. Useful when relying on a single Location Scheme, which could result in multiple output pages with the same filename.

Stylesheet Optional.  The name of the stylesheet that produces the formatting for the Variant.

Output Defines the output form of the Template. Options are

  • Snippet

    Variant outputs a Snippet. Snippets are assembled into other Snippets or into pages; thus, the output of a Snippet Variant must be well-formed XML.

  • Text page

    Variant outputs a formatted HTML page.

  • Binary

    Variant produces a binary output.

URL The URL of the Rhythmyx application that generates the Variant output.
