Extension Data dialog

Use the Extension Data dialog to add extension parameters to the extension registration. To access the Extension Data dialog, on the Extension Registration editor for a Java extension, click the [Extension Data] button.

Field Descriptions

Name Name of the parameter of the extension. The parameter must be defined in the extension code.

Value The Java data type of the parameter. Options are all valid Java data types.

Description Free-form description of the parameter.
