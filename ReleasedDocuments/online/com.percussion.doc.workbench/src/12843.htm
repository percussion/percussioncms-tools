Adding Graphics to Display Formats

The default Display Formats provided by Percussion Software, Inc., do not include graphics, but you can add graphics to your custom Display Formats.

  • For details about adding static graphics (graphics not managed by Rhythmyx) to Display Formats, see the topic "Using Images in Display Formats".
  • For details about adding thumbnail previews of Rhythmyx-managed Image Content Items, see the topics "Adding Thumbnail Previews to Display Formats".
