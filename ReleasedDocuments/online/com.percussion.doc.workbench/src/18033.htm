Template Editor

Use the Template editor to maintain data for a Template.

The Workbench displays the Template editor when you click the [Finish] button after completing the New Template wizard. Other ways to access the Template editor include:

  • In Assembly View, right-click the Template you want to modify and choose Open.
  • In Assembly View, double-click the Template you want to modify.

The Template Editor consists of the following tabs:

  • Velocity
  • General
  • Bindings
  • Slots
  • Sites
  • Target Tables
