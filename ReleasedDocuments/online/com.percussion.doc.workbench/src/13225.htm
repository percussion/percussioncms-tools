Maintaining Keywords

A Keyword defines a category of choices, such as Sizes. Any number of Keyword Choices are defined for the keyword. For example, the Keyword Choices for Sizes could be S, M, L and XL.

When a user assigns a control that requires a list of choices (such as a drop down list) to a field, the user can associate the Keyword with the control to populate the control with the Keyword Choices.

  • Use the New Keyword wizard to create a new Keyword and define Keyword Choices.
  • Use the Keyword editor to edit Keyword Choices.

You can:

  • Create a new Keyword;
  • Add Keyword Choices to the Keyword;
  • Edit the Keyword Choices or
  • Delete the Keyword
