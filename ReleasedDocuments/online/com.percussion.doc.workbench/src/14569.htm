New Shared Field Definition File Wizard

The New Shared Field Definition File wizard allows you to create a new shared field definition file that appears in Content Design view; however the shared field sets in the shared field definition file are entered in the Shared Field editor.

To access the New Shared Field Definition File wizard:

  • From the Menu bar choose File > New > Other. In the Select a Wizard dialog, choose Shared Field File.
  • In Content Design view, right-click the Shared Fields folder or a node under it and choose New > Shared Field File.

    Screenshot of New Shared Field Definition File Wizard

Field Descriptions

Shared field definition file name - Name of the shared field definition file. The suffix .xml is automatically added to the file name.

First group name - Name of the first shared field set in the shared field definition file.
