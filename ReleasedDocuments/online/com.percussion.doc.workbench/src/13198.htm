Variant Editor

Use the Variant editor to maintain data for Variants.

The Workbench displays the Variant editor when you click the [Finish] button after completing the New Variant wizard. Other ways to access the Variant editor include:

  • In the XML Server view, right-click the Variant you want to modify, and choose Open.
  • In the XML Server view, double-click the Variant you want to modify.

The Variant editor includes three tabs:

  • General
  • Slots
  • Sites
