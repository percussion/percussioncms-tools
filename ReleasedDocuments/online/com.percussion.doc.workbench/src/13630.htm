Creating a New Item Filter

To create a new Item Filter:

  1. In the Menu bar, choose File > New > Other.

    The Workbench displays the Eclipse New Object Wizard dialog.

  2. Select Item Filter and click the [Next] button.

    The Rhythmyx Workbench displays the Item Filter Wizard.

  3. Enter the Filter name. Optionally, enter a Description.
  4. Click the [Finish] button.

    The Rhythmyx Workbench displays the Item Filter Editor.

  5. To inherit Rules from another Item Filter, choose the Item Filter from which you want to inherit in the Inherit from drop list. Options include all other Item Filters in the system.
  6. To add a Rule, double-click in an empty row of the Rules table, then click the drop arrow and select the Rule you want to add. Options include all Rules defined in the system.
  7. To add parameters to a Rule:
    1. Double-click on a Rule, the click on the browse button .

      The Rhythmyx Workbench displays the Extension Parameters dialog.

    2. Specify a value for each parameter.
    3. Click the [OK] button to return to the Item Filter dialog.
  8. If the Item Filter will be used for publishing, choose the Associated Authorization Type for the Item Filter. Options include all Authorization Types Keyword Choices. No Authorization Type association is required if the Item filter is used for unpublishing.
  9. On the Button bar of the Rhythmyx Workbench, click the save button to save the Item Filter.
