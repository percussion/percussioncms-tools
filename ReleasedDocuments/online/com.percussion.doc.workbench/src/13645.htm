Rhythmyx Editors

Rhythmyx Workbench editors may consist of a single window or any number of tabs. The editor for an object may duplicate the information entered into the object's wizard, or it may include additional optional or required fields. A box at the top or side of each window may include instructions and display error information if invalid or insufficient data is entered.

Screenshot of a Rhythmyx Editor

An object's editor always opens after you complete the wizard for the object and click [Finish]. At that point, you can enter additional fields in the editor, modify fields, or close the editor without adding or modifying any fields.

To open an object in a Rhythmyx editor:

  • Complete the wizard for the object and click [Finish].
  • Double-click the object in a view that displays it.
  • Right-click the object in a view that displays it and choose Open.
