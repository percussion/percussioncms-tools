Menu and Menu Entry Editor

The Rhythmyx Workbench's Menu and Menu Entry editor allows you to add and edit properties of Menus and Menu Entries, including where they are accessible in Content Explorer.

To access the Menu and Menu Entry editor:

  • After completing the New Menu wizard or the Menu Entry wizard, click [Finish].
  • Right-click on the menu or menu entry object in any view that displays it and select Open.
  • Double-click on the Menu or Menu Entry object in any view that displays it.

The Menu and Menu Entry editor includes four tabs:

  • General tab, which differs for Menus and for Menu Entries
  • Usage tab
  • Command tab
  • Visibility tab
