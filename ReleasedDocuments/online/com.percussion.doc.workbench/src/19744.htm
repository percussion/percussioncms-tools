General Tab

Use the General tab to maintain general data about the Template.

The Workbench includes two different versions of the General tab.

General Tab for Local and Shared Templates

The Workbench displays this version of the General tab if you have specified that the Template is a local or a shared template (in other words, you selected either the Local or Shared radio button on the New Template wizard Output dialog). Use it to specify general data for the Template.

Field Descriptions

Template name Read-only. Displays the name of the Template.

Template label Label displayed for the Template in Content Explorer.

Description Free-form description of the Template.

Assembler Specifies the assembler extension to use when assembling the output. Options include:

  • Velocity Assembler

    Generates text output (including both HTML and XML) using the Apache Velocity technology. For details about Velocity, see http://jakarta.apache.org/velocity, or the following books:

    • Joseph D. Gradeck, Mastering Apache Velocity
    • Rob Harrop, Pro Jakarta Veolcity
  • Binary Assembler

    Extracts binary files from the Repository.

  • Dispatch Assembler

    Used to calculate which Template to use when more than one Template could be used.

Active Assembly format Format type for this Template in Active Assembly. Options are:

  • Normal - Default. The Template does not require special treatment for Active Assembly.
  • Auto Index - Use this option if you are configuring a legacy Auto Index Variant.
  • Non-HTML - The Template should not be formatted for Active Assembly. Use this with Content Types that store images, file attachments, JavaScript files, and other file types that are not compatible with Active Assembly.

    Note: Auto Index Slots are not rendered in Active Assembly regardless of the Active Assembly format chosen.

Output Specifies the output form of the Template. Options include:

  • Snippet

    The Template produces a Snippet, which will be merged into another output.

  • Text Page

    The Template produces a complete text page, such as an HTML page. If you select this option, you must specify the global template option to be used for the output pages. Options include:

    • Default

      Search up the Folder tree and use the first global template found; otherwise, use the global template specified for the Site.

    • Specified

      Always use the specified global template with this Template.

    • None

      Do not use a global template with this Template.

  • Binary

    The Template produces a binary output.

Mime Type Specifies the MIME type of the Template output if the Binary radio button is selected under Output.

Character set Specifies the character set output by the template if either the Snippet or Text Page radio buttons are selected under output.

Location Prefix Optional prefix for the filename of the published Template. Useful when relying on a single Location Scheme, which could result in multiple output pages with the same filename.

Location Suffix Optional suffix for the filename of the published Template. Useful when relying on a single Location Scheme, which could result in multiple output pages with the same filename.

Publish Specifies the circumstances when the Template should be published when publishing to a Site Folder. Options are:

  • Always: Always publish this Template.
  • Default: Publish this Template if it is the Default Template for the Content Type.
  • Never: Never publish this Template.

General Tab for Global Templates

The Workbench displays this version of the General tab if you have specified that the Template is a Global template (in other words, if you selected the Global radio button on the New Template Wizard Output dialog).

Field Descriptions

Global Template name Read-only. Displays the name of the Template.

Global template label Label displayed for the Template in Content Explorer.

Description Free-form description of the Template.

Mime Type Specifies the MIME type of the Template output if the Binary radio button is selected under Output.

Character set Specifies the character set output by the template if either the Snippet or Text Page radio buttons are selected under output.
