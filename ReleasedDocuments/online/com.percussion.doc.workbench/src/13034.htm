Content Type Editor

Use the Content Type editor to

  • enter fields into a new Content Type and assign rules to it.
  • edit an existing Content Type.

To access the Content Type editor:

  • After completing the Content Type wizard, click [Finish].
  • Right-click on the Content Type object in any view and select Open.
  • Double-click the Content Type object in any view.

The Content Type editor includes three tabs:

  • Content Type tab
  • Field Set tabs
  • Properties tab
