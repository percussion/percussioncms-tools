Maintaining Menus and Menu Entries

Menus provide the functions available to Content Items and Folders in various Content Explorer locations or contexts. Menu entries are the choices that are listed in menus. Menu entries are not children of any Menu and can be associated with multiple menus. Typical menu entries enable functions such as editing Content Items, performing Workflow Transitions, and implementing Relationships (for example, creating new Active Assembly Relationships, or new Copies, Versions, or Translation Copies of Content Items). Menus may be nested to any number of layers. Most frequently, menus appear as pop-ups that open when you right-click on a Content Item or Folder or as sub-menus of other menus.

Note: For more information about Relationships and Workflows in Rhythmyx, see the documents Implementing the Relationship Engine and Maintaining Workflows in Rhythmyx.

Create Menus and Menu Entries in the Rhythmyx Workbench:

  • Use the New Menu wizard to create a new menu.
  • Use the New Menu Entry wizard to create a new menu entry.
  • Use the Menu and Menu Entry editor to complete and maintain menus and menu entries.

You can:

  • Create a basic menu object
  • Create a basic menu entry object
  • Add properties to menus and menu entries.
  • Edit properties of menus and menu entries.
  • Delete menus
  • Delete menu entries
