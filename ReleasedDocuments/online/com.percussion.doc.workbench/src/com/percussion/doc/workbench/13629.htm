Item Filter Editor

Use the Item Filter editor to

  • complete an Item Filter
  • maintain Item Filters.

The Workbench displays the Item Filter editor when you click the [Finish] button after completing the New Item Filter wizard. Other ways to access the Item Filter editor include:

  • In Assembly View, right-click the Item Filter you want to modify and choose Open.
  • In Assembly View, double-click the Item Filter you want to modify.

    Screenshot of Item Filter Editor

Field Descriptions

Item filter name Read-only. Name of the Item Filter.

Description Free-form description of the Item Filter.

Inherit from Optional. Specifies another Item Filter from which to inherit rules. The Rules of the specified Item Filter are processed first, then the Rules of the current Item Filter.

Associated Authorization Type Associates the Item Filter with a legacy Authorization Type. Used when publishing Velocity Templates with XSL Variants so the Variants can be processed correctly. Options include all Auth Types defined in the authtypes.properties file (<rhythmyxroot/rxconfig/Server/authtypes.properties).

Rules Table specifying the Rules used in the Item Filter. Rules are processed in the order specified.
