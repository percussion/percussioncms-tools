Variant Editor Sites Tab

Use the Sites tab to specify the Sites on which the Variant can be used.

Screenshot of Variant Editor, Sites Tab

Field Descriptions

Hidden on these Sites column - lists the Sites for which the Variant is not currently available. You can filter this list by entering a string in the Filter field.

Visible on these Sites column - lists the Site for which the Variant is currently available.

Use the arrow buttons to move Sites from one column to the other.
