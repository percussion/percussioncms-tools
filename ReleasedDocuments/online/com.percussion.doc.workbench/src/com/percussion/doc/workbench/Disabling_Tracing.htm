Disabling Tracing

Disabling tracing is stopping tracing on an application that is not currently running.  To disable tracing:

  1. Open the application for which you want to disable tracing.
  2. In the Menu bar, choose Tools > Debugging

    Rhythmyx will display the Trace Setup dialog, with the name of the application in the Title bar, and showing the current settings.

  3. Uncheck the Enable Tracing checkbox.
  4. Click [OK].
  5. Save the application.
