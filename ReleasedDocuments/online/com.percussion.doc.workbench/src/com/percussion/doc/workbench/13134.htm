Creating a New Template

The following procedure assumes that you have already created the HTML file used in your Template.

To create a new Template in Rhythmyx:

  1. In the Menu bar, choose File > New > Template.
  2. The Workbench displays the New Template Type dialog.
    1. To create a Template used by only one Content Type, select the Type-specific radio button. The Workbench enables the Content Type drop list. Choose the Content Type with which you want to associated the Template from the drop list.
    2. To create a Template shared by multiple Content Types, select the Shared radio button.
    3. To create a Global Template, select the Global radio button.
    4. To create a Template for database publishing, select the Database publishing XML radio button.
    5. Click the [Next] button.

      If you selected Local or Shared on the New Template Type dialog, the Workbench displays the New Template Output dialog; if you selected Global, skip to Step 7.

  3. Choose the Assembler to use to generate the output. Options include:
    • Velocity Assembler: generates a text page using Velocity markup to add local content.
    • Binary Assembler: generates a binary output, such as a image file or .pdf file.
    • Dispatch Assembler: calculates which Template to use to generate the output.
  4. Specify the Output of the Template.
    1. If you want the Template to produce an intermediate output that will be assembled with other Content Items to produce a final page, select the Snippet radio button.
    2. If you want the Template to produce a final text page output, select the Text Page radio button. If you select this option, you must also specify the Global Template that will be used with the page. Options are:
      • Default: Use the first Global Template found searching up the Site Folder tree path; if no template is found in this search, use the default Global Template for the Site.
      • Specified: Always use the specified Global Template with this Page Template.
      • None: Do not use a Global Template with this Template.
    3. If you want the Template to produce a binary output, select the Binary radio button.
  5. Click the [Next] button.

    The Workbench displays the New Template Name dialog.

  6. Complete the New Template Name dialog:
    1. Enter a Template name. This value is the name used to identify the Template for internal processing. The Template name must be unique among all Templates in the system.
    2. Enter a Label for the Template. The Content Explorer will display this value as the label for the Template. Multiple Templates can share the same Label.
    3. Optionally, enter a free-form Description of the Template.
    4. To add a Source file for the Template, click the brows button , browse to directory where the source HTML file is stored, and select it, and click the [OK] button.
  7. Specify the Communities that will be able to use the Template:
    1. To make the Template visible to one Community, select the Community and click the [>] arrow .
    2. To filter the list of available Communities, enter a string in the Filter field. The Workbench displays only Communities that match your filter criteria.
    3. To make the Template available to all listed Communities, click the [>>] button.
    4. You can also multi-select Communities by holding down the CTRL key on your keyboard and clicking on the Communities you want to select.
  8. Click the [Next] button.

    The Workbench displays the New Template Slots dialog.

  9. Add Slots to the template.
    1. To add one Slot to the Template, select the Slot you want to add and click the add arrow .
    2. To filter the list of available Slots, enter a string in the Filter field. The Workbench displays only Slots that match your filter criteria.
    3. To add all listed Slots to the Template, click the double-arrow button.
    4. You can also multi-select Slots by holding down the CTRL key on your keyboard and clicking on the Slots you want to select.
    5. Click the [Next] button.
  10. The Workbench displays the New Template Sites dialog.
    1. To add one Site to the Template, select the Site you want to add and click the add arrow .
    2. To filter the list of available Sites, enter a string in the Filter field. The Workbench displays only Sites that match your filter criteria.
    3. To add all listed Sites to the Template, click the double-arrow button .
    4. You can also multi-select Sites by holding down the CTRL key on your keyboard and clicking on the Sites you want to select.
  11. Click the [Finish] button.

    The Workbench displays the Template editor. You can modify the Template, or close the Template editor.
