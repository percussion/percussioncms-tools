XML Server Explorer

The primary dialog used in maintaining XML applications is the Application editor. The following additional dialogs are also used:

  • Data Pipe
  • Result Pager
  • Transaction Manager
  • Page Data Tank (Page Data Tank, XML Application Procedures)
  • Backend Data Tank (Creating a New Application, Backend Datatank)
  • Join Editor
  • Non-text Resource
