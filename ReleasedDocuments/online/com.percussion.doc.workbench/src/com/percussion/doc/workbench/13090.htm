Editing a Field Transform

When you edit a field transform, you can add, remove, or change the order of any of the Rules listed in the table and change any of the Rule Details that correspond to the Rules.

For a graphic of the Field Transform editor and definitions of the fields discussed below, see Field Transform Editor.

To edit a field transform:

  1. In the Field Properties Editor for the field, click [Transform].

    The Field Transform editor opens.

  2. To modify an input transform, click the input tab. To modify an output transform, click the output tab.
  3. To change the order in which the rules are executed, change the position of the rules in the Rule table. Click on each rule that you want to move and click or .
  4. To add a rule to the Rule table, click on the row where you want to add the rule or click on the rule that follows the rule that you want to add and click to create a new row. Add the rule as specified in Adding a Field Transform.
  5. To delete a rule from the Rule table, click on the rule and click .
  6. To change the details (parameters) for a rule, click on the rule in the Rule table and modify the parameters. For information about the parameters, see Default Transforms and their Parameters. To apply the modified details instead of adding a new transform, click [Apply].
  7. To change the Transform failure message, enter a new message.
  8. Click [OK].
