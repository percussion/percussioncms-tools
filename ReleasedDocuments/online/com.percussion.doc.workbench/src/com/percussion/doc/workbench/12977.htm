Connecting to a Rhythmyx Server

Starting a Rhythmyx Workbench session requires opening the Rhythmyx Workbench and connecting to a Rhythmyx Server. Once you are connected to the Server, you can view and edit the objects it contains.

The Workbench can only be connected to one Rhythmyx Server at a time; however, while working in the Rhythmyx Workbench, you can end your connection to one Rhythmyx Server and start a connection to another one.

If you install the Rhythmyx Workbench with the Rhythmyx Server, your default connection is configured to connect to your Rhythmyx Server. The default User and Password are admin1/demo and Make Default Connection is checked. This configuration does not check you in automatically unless you check Automatically connect when Workbench opens in Rhythmyx Preferences.

Screenshot of Manage Connections dialog

If you install the Rhythmyx Workbench separately from the Rhythmyx Server, your default connection is configured to connect to Server localhost and Port 9992. The default User and Password are admin1/demo and Make Default Connection is checked. This configuration does not check you in automatically unless you check Automatically connect when Workbench opens in Rhythmyx Preferences.

To manage connections to Rhythmyx Servers:

  1. To initially open the Manage Connections dialog, open the Rhythmyx Workbench. The Manage Connections dialog automatically opens.

    OR

    Once you are in the Rhythmyx Workbench, to change the Server that you are connected to, in the menu bar, choose File > Connect. The Connections dialog opens.

    Any existing Rhythmyx server configurations appear in the Names list box.

  2. To add a new Rhythmyx server configuration:
    1. Click [New].

      The name New_[ServerName] for the configuration appears in the Name list box and the Name text box. Server is filled with the Rhythmyx Server name for new connections, and Port is automatically set to 9992. The other values are filled in with admin1 and demo. Save password is checked, and Make this the default connection and SSL are not checked.

    2. In Name, optionally enter a new name for the connection.
    3. In Server, enter the name of the Rhythmyx server that you want to connect to, or enter its IP address.
    4. In Port, leave 9992 or enter the Rhythmyx server's port number.
    5. In UID, enter your Rhythmyx user name.
    6. In Password, enter your Rhythmyx password.
    7. By default, SSL is unchecked. To connect to the Server over SSL, check SSL.
    8. If you want the dialog to remember your Rhythmyx password for this server, check Save password.
    9. Click [Apply] to save the configuration without closing the dialog.
  3. To modify an existing Rhythmyx server configuration, click the name of the server configuration in the Names list box and change the values of any of the fields. Click [Apply] to save the changes without closing the dialog.
  4. To delete an existing Rhythmyx server configuration, click the name of the server configuration in the Names list box and click [Delete].
  5. To connect to a configured Rhythmyx server, click the name of the server configuration in the Names list box and click [Connect].

    The Workbench connects to the Rhythmyx server and closes the dialog. The Workbench views now display the objects and settings for the connected server.
