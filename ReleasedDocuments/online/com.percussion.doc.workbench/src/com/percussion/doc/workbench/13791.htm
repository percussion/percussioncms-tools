Search Editor, Custom Search

The Search Editor for custom searches lets you complete or edit a custom search.

To access the Search Editor for custom searches:

  • After completing the New Search wizard (and selecting the Type Custom), click [Finish].
  • Right-click on the custom search object in any view that displays it and select Open.
  • Double-click the custom search object in any view that displays it.

Using the Search Editor for custom searches you can:

  • Complete a custom search
  • Modify a custom search

    Screenshot of Search Editor, Custom Search

Field Definitions

Custom Search Name - Read-only. System name for search query.

Custom Search Label - Required. Name displayed to users for search query.

Description - Description of search.

Custom Search URL - URL of application that includes the custom search query.

Max rows returned - Maximum number of results to return.

Unlimited - Return all results. If checked, any value in Max rows returned is ignored.

Display Format - The Display Format you want to use to display the results of the search.

Search Criteria - The contents of this box vary, depending on whether or not you specify fields for this view, and which fields you specify. Use the [Customize] button to include Content Type fields in the search. The control for the field varies depending on the data type of the field.

drop list - options describing how the entry in Title must match the Content Item title. Options are Starts With, Contains, Ends With, Extracts.

Starts With - The Content Item title must start with the value.

Contains - The Content Item title must start with the value.

Ends With - The Content Item title must end with the value.

Extracts - xxxxx

Title - exact title or part of title to search for.

[Customize] - Opens the Field Selection editor for further customizing the search.
