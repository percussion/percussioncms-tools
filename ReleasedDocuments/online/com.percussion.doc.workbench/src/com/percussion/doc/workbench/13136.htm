Deleting a Template

If you delete a Snippet Template or a Binary Template, assembly of any Content Item that uses that Template will fail, but the remainder of the assembly process will be executed. Rhythmyx will return an error in the log stating that assembly of the Content Item failed because the Template could not be found. The page will be published, however, with the failed template excluded.

If you delete a Page Template or a Global Template, assembly of the page will fail. An error will be reported to the log stating that assembly of the page failed because the required Template could not be found. No page will be produced.

To delete a Template, in the Assembly Design View, right-click on the Template you want to delete and from the popup menu, choose Delete.
