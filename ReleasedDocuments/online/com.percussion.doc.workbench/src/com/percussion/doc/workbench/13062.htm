Including Workflows

For a graphic of the Properties tab and definitions of the fields discussed below, see Properties tab.

  1. To add Workflows and Transform and Validation rules to a Content Type click the final Properties tab on the Content Type editor.

    The Content Type name, Label, Description, and Allowed workflows are entered from the Rhythmyx repository. The Content Type name cannot be changed.

    1. By default, Enable searching for this Content Type is checked. If you do not want the contents of this Content Type indexed for searching, uncheck the check box.
    2. All available Workflows for the Content Type are listed with check boxes next to them under (Allowed) Workflows. Workflows that you selected as Allowed Workflows in the Content Type Wizard are checked. Check or uncheck the Workflows to allow for the Content Type. You must check at least one Workflow to save your edits. To check all of the Workflows, click [Select All]. To unselect all of the Workflows, click [Unselect All].
    3. Select a Default Workflow for the Content Type. If the Default Workflow is not already checked under Allowed Workflows, it is automatically checked when you select it. If you uncheck the Default Workflow, the first checked Workflow becomes the Default Workflow. One Allowed Workflow and a Default Workflow must exist before you can save the information on the screen.

Now you can:

  • Go to the tabbed Item box to perform item input transforms, item output transforms, and validations.
  • Close the Content Editor.
