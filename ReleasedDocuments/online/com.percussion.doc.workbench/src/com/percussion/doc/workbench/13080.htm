Field Validation Editor

The Field Validation editor allows you to add a field validation to a field and maintain the validation.

To access the Field Validation editor:

  • After selecting a row for a field in a Field and Field Sets editor, click the [Validation] button.
  • When creating or editing a field in the Field Properties editor, click the [Validation] button

    Screenshot of Field Validation Editor

Field Definitions

[Prerequisites] - Button that opens a dialog for specifying conditions for performing the validations.

Validation on insert or update - List of validations selected in Validation type drop list under Rule Details and operators that indicate the relationship between the validations. Validations are listed in the order in which they are selected.

Rule - Extension or condition that performs a specific validation on a field.

Boolean - Operator such as AND or OR that specifies the relationship between this rule and the next one.

Rule Details - Section listing the validation type and the values of parameters, if any, associated with the validation type. For information about specific validations see Default Validations and their Parameters.

Validation type - Default and user-defined extensions, conditions, and JEXL expressions that define validation rules.

Validation failure message - Message that appears in field in Content Editor if validation fails.
