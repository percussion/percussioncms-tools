System Functions

Binding variables that begin with the prefix $rx are system functions.

Note that you can also write your own binding functions, which must use the prefix $user. For details, see the Rhythmyx Technical Reference Manual.

  • $rx.asmhelper: provide data for use in assembly
  • $rx.codec: encode and decode data
  • $rx.cond: evaluate conditional statements
  • $rx.db: Database Publishing
  • $rx.doc: process XML and HTML documents
  • $rx.ext: call a Rhythmyx extension
  • $rx.guid: retrieve GUIDs
  • $rx.i18n: internationalization
  • $rx.keyword: provide access to Keyword data
  • $rx.link: manipulate links
  • $rx.location: generate hypertext links
  • $rx.nav: Managed Navigation processing
  • $rx.string: manipulate string values
