Setting New Searches for Communities

For graphics of the editors and definitions of the fields discussed below, see the Configure New Searches for Communities Dialog.

To set default new searches in general and for Communities:

  1. In UI Elements Design view, right-click the Searches folder and choose Assign New Searches by Community.
  2. In New CX Search choose a search in the drop list. This search will open when users double-click the New Search node in the navigation pane of Content Explorer.
  3. In New AA Search choose a search in the drop list. This search will open when users choose to search for a related content item during Active Assembly or Active Assembly for documents.
  4. If you want to use different default searches for any of the Communities in your system, in the Community Specific Searches table, in the row for the Community choose a different New CX Search or New AA Search.
  5. Save and close the editor.
