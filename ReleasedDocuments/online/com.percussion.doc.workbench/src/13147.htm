Template Editor Slots Tab

Use the Slots tab to specify the Slots that are available to use in the Template.

Screenshot of Template Editor, Slots Tab

Field Descriptions

Available slots column - displays all Slots in the system not current in use by the Template. You can filter this list by entering a string in the Filter field.

Contained slots column - lists Slots that are assigned to the Variant.

Use the arrow buttons to move Slots from one column to the other.
