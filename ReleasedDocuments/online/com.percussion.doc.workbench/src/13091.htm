Removing a Field Transform

Since a field transform is a set of rules that are applied to a field, you do not delete the field transform, but remove all of the transform rules applied to the field.

For a graphic of the Field Transform editor and definitions of the fields discussed below, see Field Transform Editor.

To remove a field transform:

  1. In the Field Properties Editor for the field, click [Transform].

    the Field Transform editor opens.

  2. To remove an input transform, click the input tab. To remove an output transform, click the output tab.
  3. Click on each Rule in the Rule table and click .
  4. Click [OK].
