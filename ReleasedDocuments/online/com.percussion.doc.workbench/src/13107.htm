Maintaining Slots

A Slot defines a location within a Content Item that can contain related content. It also specifies the Content Types that can be included in that location.

Any Slot can be used with either a Template or an XSL Variant.

For detailed information about implementing Slots, see "Creating Slots" in the Rhythmyx Implementation Guide.

Use the New Slot Wizard to create a new Slot.

Use the Slot editor to complete a new Slot or to maintain an existing Slot.

You can

  • create new Slots;
  • modify existing Slots; or
  • delete Slots.
