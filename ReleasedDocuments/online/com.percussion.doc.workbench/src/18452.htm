Modifying a Template

When you modify a Template, you can modify the Velocity template or XSL file associated with the Template, modify general Template data, add or modify bindings, and add or remove Slots and Sites.

To modify a Template:

  1. Right-click on the Template you want to modify and from the popup menu, choose Open.

    The Workbench displays the Template Editor.

  2. On the Velocity tab, you can modify the Velocity Template. For details about adding Velocity macros to the Velocity template, see the Rhythmyx Implementation Guide.
  3. On the General tab, you can:
    • change the Label or Description of the Template;
    • change the Publish option;
    • add or modify a Location Prefix or Location Suffix;
    • change the Active Assembly format
    • change the MIME Type; or
    • change the Character set.
  4. On the Bindings tab, you can add or change bindings.
  5. On the Bindings tab, you can copy bindings from one Template and paste them to another. To copy and paste bindings:
    1. Select the binding you want to copy.
    2. Right-click and from the popup menu, choose Copy.
    3. Open the Template to which you want to paste the binding and select the Bindings tab.
    4. Right-click in an empty row and from the popup editor, choose Paste.
  6. On the Slots tab, you can add or remove Slots.
  7. On the Sites tab, you can add or remove Sites.
  8. When you have finished your modifications, click the [Save] button to save your changes.
