Adding a Sub-Folder in CMS Navigator

You can create user-defined sub-folders under many of the existing nodes and folders in the CMS Navigator object trees.

To create a user-defined folder in a CMS Navigator view:

  1. Open the view.
  2. Right-click on the folder in the view that will be the parent of the new folder. The parent folder can be the main folder in the view or a sub-folder.
  3. In the menu, choose New > Folder.

    A folder named New Folder appears under the selected node:

  4. Right-click on the new folder.
  5. In the drop menu, choose Rename.
  6. Type in a new name for the folder.
