Binding Variable Properties Dialog

Use the Binding Variable Properties dialog to specify a binding variable and the JEXL expression used to generate a value for that variable.

To access the Binding Variable Properties dialog,

  • double-click on a row in the Bindings editor; or
  • select an existing binding where you want to add a new binding; the new binding will be inserted between the selected row and row immediately above it.

Field Descriptions

Variable Name The binding variable you want to define. All binding variables should begin with the "$" character. If you do not prefix the variable with this character, Rhythmyx will add it automatically.

Expression Editor The JEXL expression that generates the value for the variable. Any expression valid in JEXL version 1.1 can be specified.
