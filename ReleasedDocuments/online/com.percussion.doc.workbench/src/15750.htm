Exporting and Importing Objects

The same procedure is used for exporting and importing objects.

  1. In the Rhythmyx Workbench, from the File menu, choose File > Export (to export and object) or File > Import (to import an object.

    The Rhythmyx Workbench displays the Select dialog of the Import/Export wizard.

  2. Select the type of object you want to import or export. Options include:
    • Content Types
    • Templates
  3. Click the [Next] button

    The Rhythmyx Workbench displays the Import/Export wizard Object dialog.

  4. Enter the path to the directory to which you want to export or from which you want to import.

    Click the [Browse] button to display a browse dialog you can use to browse the file system for the location.

    To create a new directory, browse to the directory where you want to create the new directory, then click the [New Directory] button. Enter a name for the directory in the Folder field.

  5. Check the boxes of the objects you want to export or import. (Note that if you are importing, no objects will be listed until you specify a directory that contains export files (files with the extension contentType or template).
  6. Click the [Finish] button.

    Rhythmyx performs the action you requested.
