Saving New and Edited Objects

To save a new or edited object and leave editor open:

  • In the Menu bar choose File > Save.

OR

  • In the Tool bar click .

To save a new or edited object and close the editor:

  1. In the Menu bar choose File > Close.

    The Save Resource dialog prompts you to save your changes:

  2. Click [Yes].

    The editor closes and your new object or changes are saved.

OR

  1. Click the X in the editor tab.

    The Save Resource dialog prompts you to save your changes:

  2. Click [Yes].

    The editor closes and your new object or changes are saved.
