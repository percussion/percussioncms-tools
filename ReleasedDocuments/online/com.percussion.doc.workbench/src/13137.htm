New Template Type Dialog

The New Template Type dialog is the first dialog of the New Template Wizard. Use this dialog to specify whether the new Template is a Type-specific Template, a Shared Template (used by two or more Content Types), a Global Template, or a Database publishing XML Template .

Screenshot of New Template Type dialog

Field Descriptions

Content Type drop list - only available if the Content-specific radio button is selected. Options for this list include all Content Types currently defined in the system.
