Import/Export Wizard

You export and import objects using the same wizard; the difference is in the processing. The import/export wizard consists of two dialogs.

The first dialog is the Select dialog. Use this dialog to specify the type of object you want to import or export. Options include all objects eligible to be exported.

Screenshot of Import/Export Wizard Select Dialog

The second dialog of the wizard is the Objects dialog. This dialog displays the objects eligible for export.

Screenshot of Import/Export Wizard Object Dialog

In the Export wizard, use the To directory field to specify the directory where you want to save the object export file. In the Import wizard, use the From directory field to specify the directory from which the object export file should be imported.

The Objects field lists the objects eligible to be exported or imported. Check the boxes of the objects you want to import or export.
