XML Application Dialogs

The primary dialog used in maintaining XML applications is the Application editor. The following additional dialogs are also used:

  • Data Pipe
  • Mapper
  • Selector
  • Result Pager
  • Transaction Manager
  • Page Data Tank
  • Backend Data Tank
  • Updater Properties
  • Join Editor
  • Non-text Resource
  • Web Page Properties dialog
  • Cache Settings dialog
