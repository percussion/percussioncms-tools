Starting Tracing on an Application that is Running

Starting tracing is setting tracing on an application when it is already running.  To start tracing:

  1. In the Rhythmyx Workbench, go to the XML Server View.
  2. Select the application whose tracing settings you want to maintain, right-click, and from the popup menu choose Debugging.

    The Rhythmyx Workbench displays the Trace Setup dialog, with the name of the application in the Title bar.

  3. Check the boxes for the trace option you want to monitor.  When you check the box, the dialog displays a description of what that trace monitors and records.  See also Tracer Types and Flags.  

    Note that modifications to the Output Column Width and Enable timestamp only tracing fields do not affect tracing started manually.  To modify these options you must click [OK], save the application, then restart it.

  4. Click [Start Tracing].

These changes are temporary unless you save the application.
