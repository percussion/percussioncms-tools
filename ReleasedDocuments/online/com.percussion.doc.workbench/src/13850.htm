Modify a Standard Search

You can modify any of the information for a standard search accessible through the Search Editor for standard searches.

For graphics of the editors and definitions of the fields discussed below, see Standard Database Search Editor and Standard Full Text Search Editor.

To modify a standard search:

  1. Change any of the following:

    Standard Search label

    Description

    Display Format

    Initial display mode

    User customizable

    Add, remove, or change the value entered Title, or click [Customize] to add or remove custom fields that you have included under Search criteria.

  2. If your system uses the Full Text Search engine, you can also change the values of the following fields:

    Search for

    Mode. If you change the Mode to Concept, choose an Expansion value. If you change the Mode from Concept, the Expansion field is no longer used.

    Filter with

    Check or uncheck Unlimited or change the maximum results returned in Max rows returned.

  3. Save and close the editor.
