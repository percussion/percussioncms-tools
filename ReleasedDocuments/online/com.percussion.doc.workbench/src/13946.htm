New Content Type Communities Dialog

The Communities dialog is the first dialog of the New Content Type Wizard. Use this dialog to specify basic properties of the Content Type and the Communities that it is visible to.

Content Type Wizard, Communities dialog

Field Descriptions

Content Type name - Required. System name for Content Type.

Label - Name displayed to users for Content Type.

Description - Description of Content Type.

Available Communities - Communities that may have access to Content Type.

Visible to these communities - Communities given access to Content Type.

Filter - Pattern that each Community displayed under Available Communities must match.
