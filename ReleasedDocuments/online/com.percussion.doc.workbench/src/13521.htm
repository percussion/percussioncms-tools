Editing Fields and Field Sets

You can edit local, shared, and system fields and field sets in the Field and Field Sets editor.

To edit a field or field set:

  1. In the Content View, either
    • right-click the Content Type, Shared Field Object, or Content Types Global Configuration node and choose Open

      OR

    • double-click the Content Type, Shared Field Object, or Content Types Global Configuration node
  2. You can modify local and shared fields in the the Fields and Field Sets table, but not system fields.
    1. To modify a local or shared field click on its row.
      • Click [Move Up] or [Move Down] to change its location in the Content Type.
      • Click [Delete] to delete a local or shared field. The following message appears:

      Click [Yes] to remove the field from the table and the file that defines the shared field set or the Content Type. Click [No] to only remove the field from the file that defines the shared field set or the Content Type.

      If you select multiple fields to delete, the behavior you choose will apply to all of the fields. If you do not want it to apply to all of the fields, perform at least two delete actions, one in which you choose all of the fields that you want to remove from the table and click [Yes], and one in which you choose all of the fields that you do not want to remove from the table and click [No].

      If you remove a shared field that is used in a Content Type, you will receive an error that the field is not found when attempting to open the Content Type. If you proceed to open the Content Type, it will not display any fields below the missing field.

      • You cannot change the Name of a field or field set.
      • To change the Label, click in the cell and retype the Label.
      • To change the Control, click in the cell and select a different Control from the drop list. If you have originally chosen an array control (sys_CheckBoxGroup, sys_CheckBoxTree, sys_DropDownMultiple), only one of these control types is available in the drop list for changing the control. If you have originally chosen a non-array control (all other standard controls except for sys_table), only non-array control types are available in the drop list for changing the control.
    2. To insert a field or field set in the row above an existing field, click in field's row and click [+].

      A blank row is inserted. Either select a shared or system field or field set and click [>] to move it into the blank row, or click in the Name cell and type in the name for a local field. Define the field or field set.

  3. Under Field Properties:
    • You can modify all properties of local and shared fields.
    • You cannot modify the Data type and Storage size of system fields, but you can modify the other properties.

    Click on a field or field set row to view and change the values of its properties under Field Properties. If changing the value may cause a loss of data, the editor warns you and asks you to confirm the change.

  4. To edit field validations click [Validations].
  5. To edit additional search and display properties of a field selected in the table, click [All Properties].
