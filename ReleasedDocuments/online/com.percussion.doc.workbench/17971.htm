Quick Field Creation

Using quick field creation, you only have to enter a field name and then use the other property defaults that the Workbench fills in for you.

To create a field using quick field creation, use the Fields and Field Sets editor.

For a graphic of the editor, instructions for accessing it, and definitions of the fields discussed below, see Field and Field Set Editor.

To create a field using quick field creation:

  1. In the Fields and Field Sets table click the next empty Name cell and type in the name for the field. It is best practice to enter the field in lower case. (NOTE: Field names that use an initial lower-case letter followed by an upper-case letter are not valid in Rhythmyx; in other words, aBcd would not be a valid field name, but abcd or even ABCD would be valid.)
  2. Press <Enter>.
  3. Defaults for the other properties are entered for you as follows. Properties that are not listed are left empty.

    Label: Same as Name, except capitalized and followed with :

    Control: sys_EditBox

    Source: Local

    Data type: text

    Storage size: 50

    Mime type mode: Default

    Enable searching for this field: checked

    Show in preview: checked

    Error label: Same as Name, except capitalized and followed with :

    Include in full text multi-field query: checked

  4. When you save the Content Type, you save the field. At any time, you can edit its properties.
