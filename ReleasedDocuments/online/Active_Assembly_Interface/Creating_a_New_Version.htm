Creating a New Version

You can create a new version of the main item or a Snippet in a Slot by choosing Tools > Promotable Version from the menu bar. By default, you can only create versions of Content Items that are checked in and in a Public state.

A new version has the Community of the user who creates the new version, not the Community of the content item originator.

To create a new version of the main item or a Snippet:

  1. Select the main item or the Snippet
  2. In the menu bar, choose Tools > Promotable Version.

    Rhythmyx creates a new version of the Content Item, assigns a unique ID to the new Content Item, and opens the new Content Item in a Content Editor. In front of the title it appends PV Copy(#) of.

  3. Edit the fields you want to modify for the new version.  
  4. Click the [Update] button to save the version.  

    If you do not click [Update], Rhythmyx will save the new item but not the changes you made to it.

  5. Click [Close].

    Rhythmyx saves the new version, but does not display it in Active Assembly. In Content Explorer, you can find it in the folder of the content item from which you copied it (the owner content item).

    When the Version becomes public, Rhythmyx archives the owner content item.
