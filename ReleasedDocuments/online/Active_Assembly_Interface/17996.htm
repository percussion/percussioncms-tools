Using the Content Browser

In Active Assembly, you use the Content Browser to insert content into Slots and to insert inline objects into rich text fields.

When you right-click a Slot icon and choose Insert Snippet, or when you click a button in a rich text editor to insert an inline link, inline image, or inline Template, the Content Browser opens.

The Content Browser has three tabs, Sites, Folder and Search. In the Sites and Folders tabs, you can browse to items located in your Content Explorer Sites or Folders. In the Search tab, you can perform a query style search to locate an item.

When the Content Browser initially opens, the Sites tab is selected, and Path displays the path of the Content Explorer Site you are currently working in. The content area holds the Site's top-level folders and content items.

Depending on the way that Content Types are configured in your system, the Content Browser may display custom Content Type-specific icons or the same page icon beside each content item. By default, the Content Browser displays the following icons for the FastForward Content Types:

You can use the Content Browser in two different ways:

  • If want to look for content in a specific folder or find a content item whose location you know, use the Content Browser to browse to the correct site or folder path.
  • If you want to look for content that matches certain criteria, use the search option of the Content Browser to enter search parameters and get results that match the criteria you have entered.
