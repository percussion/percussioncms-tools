Performing Workflow Transitions

The Workflow button appears in the Menu Bar when you have selected the main item, a Snippet in a Slot, or a field.

The Workflow option appears in the Context Menu of the main item or a Snippet in a Slot or a field in Outline view or Page view.

When you click Workflow, the interface opens the Workflow dialog:

The Workflow dialog displays the Workflow Transition options for the content item in the Action drop list. They are always the same options that appear in Content Explorer when you choose the Workflow option in the Action menu. The Workflow dialog may also include a Comments box which functions in the same way as the Comments box that appears when you Transition an item in Content Explorer. If comments are required, a red asterisk appears beside the Comments box.

If the "to" State of the Transition allows Ad Hoc assignment (the assignment of an item to a specific individual who may or may not be assigned to one of the States to which the item can Transition). The Workflow dialog displays an Ad Hoc Assignees box with a [Search] button for locating possible Assignees.

To perform a transition on an item in Active Assembly:

  1. Choose the main item or a Snippet in a Slot.
  2. In the menu bar, click Workflow, or in the context menu, choose Workflow.

    The Workflow dialog opens.

  3. Choose a Transition in the Action drop list.
  4. If the Comments box appears: if required, enter a comment; if not required, optionally enter a comment.
  5. If an Ad-hoc Assignees box appears, and you want to assign an Ad-hoc assignee:
    1. click [Search].

      A Search dialog opens.

    2. In the Role drop list, choose a Role.
    3. In the Filter field, optionally enter a name search string.
    4. Click [Go].

      The search returns a list of Ad Hoc users who match the search criteria.

    5. Check one or more users. They will be assignees of the content item in the "to" State.
    6. To add the assignees and close the dialog, click [Add]. To close the dialog without assigning the users as Ad Hoc assignees, click [Close].
  6. In the Workflow dialog, click [Submit].

    The item is transitioned and automatically checked in.
