Creating a Copy of the Main Item

To create a copy of the main item:

  1. From the menu bar, choose Content > Copy.

    Rhythmyx opens the Create Copy dialog.

  2. In Title, enter a unique title for the copy of the item.

    The title that you enter becomes the sys_title and the displaytitle, if the Content Type includes one.

  3. Click [Create].

    Rhythmyx copies the main item, including all of its dependencies and other relationships. The only difference is the Title.

    Active Assembly now displays the copied item as the main item.
