Using the Menu Bar and Context Menus

The Menu Bar at the top of Active Assembly includes buttons with commands and drop lists of actions. The buttons that appear are different depending on whether the main item, a Slot, a Snippet in a Slot, or a Field is selected, and an appropriate icon appears before the buttons indicating whether the selected object is the main item, a Slot, a Snippet in a Slot, or Field.

Many of the Menu Bar options are also available in context menus when you right click on an item in Outline view or in Page view. To right click on an item, follow the instructions for selecting it in Selecting a main item, a Slot, a Snippet in a Slot, or a Field, but instead of clicking on it, right-click on it.

Displays the selected item's URL, or if a field is selected, displays the URL of the item storing the field. This enables you to copy the URL and paste it in another application so another user can access the page (or Snippet).
