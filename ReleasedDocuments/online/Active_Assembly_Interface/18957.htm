Publishing an Item

In Active Assembly, you can publish the main item immediately to your Web Site. This option is useful if you want to make changes to the item in Active Assembly and have them appear on your Web Site as soon as possible.

To publish the main item immediately:

  1. Select the main item, one of its fields, a Snippet in one of its Slots, or one of the Snippet's fields.

    Note that you always publish the main item, even if you select a Snippet or one of its fields.

  2. In the menu bar, choose Tools > Publish Now.

    An edition containing the item is queued, and Rhythmyx displays the following dialog. The time it takes the system to publish the item depends on how full your publishing queue is. In some cases, the item is published immediately.

  3. To verify that the item published, go to Content Explorer and check your publishing logs.
