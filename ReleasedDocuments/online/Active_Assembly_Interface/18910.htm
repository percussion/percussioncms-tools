Creating a New Main Item

To create a new main item:

  1. From the menu bar, choose Content > New.
  2. Rhythmyx opens the Create Item dialog.
  3. In Title, enter a title for the new item.

    The title that you enter becomes the sys_title and the displaytitle, if the Content Type includes one.

  4. In Type, choose a Content Type from the drop list or leave the default value.

    If you choose a different Type than the default value, the large image may appear as a different Template, and different thumbnails may be listed below it.

  5. Choose another thumbnail to change the Template or leave the default Template.
  6. Click [Create].

    If any validation errors exist, you are prompted to open the content editor to correct them:

    1. Click [OK] to open the content editor and correct the error(s).
    2. Save and close the content editor.
  7. Click [Create] again.

    Active Assembly now displays the new item as the main item.

    You must edit the new content item to enter or change values in fields. In Content Explorer it is located in the path listed in the Create New Item dialog.
