Create Item Dialog

When you choose Content > New from the menu bar or right-click on an empty Slot and choose New Snippet, the Create Item dialog opens. (If you right-click on a Slot with one or more Snippets in it and choose New Snippet, first the Choose Where to Place the Snippet dialog opens and after you click [Next], the Create Item dialog opens).

The Create Item dialog only fills in fields with default values (and the Title that you specify). If there are any validation errors, including required fields without default values, you are prompted to open the content editor and correct the errors.

Fields in the Create Item Dialog

Title - The title you give to the new item. This value is entered in the item's sys_title field. If the item has a displaytitle field (all FastForward Content Types include a displaytitle field), the displaytitle field is also filled in with the Title.

Type - Content Type. Default value is:

  • Content Type of the current main item, if you are creating a new main item;
  • Content Type of the Snippet selected in the Choose Where to Place the Snippet dialog, if you are creating a new Snippet in a Slot containing one or more Snippets;
  • The first available Content Type for the Slot alphabetically, if you are creating a new Snippet in a Slot that does not contain other Snippets.

You can choose any of the Content Types in the drop list to change the Content Type.

Template - The large image shows the Template that will format the new item. The large image and the thumbnails below it are sample images used to show how the Templates available for the new item appear; the content in them bears no relationship to the content of the item being created. Initially the thumbnail Templates are shown in alphabetical order. To select one of the thumbnails to replace the large image as the Template that will be used to format the new item, click on the thumbnail image.

or - Arrows that appear beside the thumbnail images of Templates. Click each to see more thumbnails in the direction the arrow indicates.

Folder - Folder path of the main item. This cannot be changed, and becomes the folder path of the new item whether it is a new main item or a Snippet placed in a Slot.
