Inserting Existing Content in Slots

To add an existing Snippet to a Slot:

  1. Right-click on the Slot in Outline view or Page view and choose Insert Snippet

    OR

    Select the Slot in Outline view or Page view, and in the Menu Bar click Insert Snippet.

  2. If there are no Snippets already inserted in the Slot, skip to step 5. If one or more Snippets are already inserted in the Slot, Rhythmyx displays the Choose Where to Place Snippet dialog.
  3. Highlight a Snippet on the dialog and choose the Before, After, or Replace radio button under Place Where. If there are many Snippets in the Slot, you may choose to click [Show Titles] so you can see all of them in the dialog. To return to the original view, click [Show Snippets].
  4. Click [Next].

    The Content Browser opens.

  5. Use the Content Browser to:
    • browse for a content item and insert it;
    • enter a search query for locating a content item and insert it.
