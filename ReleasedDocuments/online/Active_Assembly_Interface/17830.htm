Inserting an Inline Template into a Content Editor

Use an inline Template when you want to insert formatted content from another Content Item into the text of the Content Item you are editing. (For more details, see Using Inline Links.) Inline Templates are only available in Content Editor fields that use rich text editors (not text boxes).

To insert an inline Template:

  1. Place the cursor in the location where you want to insert the formatted content from another Content Item.
  2. Click the inline Template button .

    Rhythmyx displays the Content Browser.

  3. Browse or search for the content item you want to add to the content item you are editing.
  4. In the list of results, click the item that you want to add as an inline Template.

    Only Snippet Templates are available for inline Templates.

  5. Choose the Template for the content item you want to add, and click [Select].

    Rhythmyx adds the formatted content to your Content Item and closes the Content Browser.
