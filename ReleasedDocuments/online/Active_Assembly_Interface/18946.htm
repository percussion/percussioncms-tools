Sending a Content Item's URL to Another User

If you want another user who has access to the Rhythmyx Server to view one of your Content Items, you can copy its URL in Rhythmyx and paste it into email or another application that the other user can access. Note: The application must be able to communicate with Rhythmyx and the user must have authorization to view the Content Item to access it.

To send a Content Item's URL to another user:

  1. Select the main item or a Snippet in a Slot.
  2. In the menu bar, choose Tools > Link to Page.

    Rhythmyx displays a Paste link dialog with the item's URL.

  3. Click CTRL-C.
  4. In the user's application (for example, IM or an email application), paste the URL into an appropriate location.
  5. The user may click on the URL to access the content item in its Active Assembly format. A dialog prompts for username and password if the user has not yet logged in to Rhythmyx.
