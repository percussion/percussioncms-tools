Creating Translation Content Items

Translation content items are duplicates of content items that you translate into other languages. When you create a Translation content item, Rhythmyx creates a duplicate copy of the original content item, its links, and its related content items, and gives the Translation content item and its related content items different content item IDs. The Translation content item proceeds through the Workflow independently from the original content item.

The text in the Translation content item is the same as the text in the original content item; Rhythmyx does not automatically translate it. The text is in the same language as the original until you manually edit it.

For example, you may require two different sites for the Canadian market—one site in Canadian English and one site in Canadian French.  You could begin by creating the site in Canadian English, and then could create duplicate Canadian French pages by creating Translation content items of each English page through the Action Menu. The original content items and the new Translation content items would proceed through Workflow independently.  

Rhythmyx refers to the Translation languages available to you as Locales. For each content item, you can create one Translation item per Locale in your system.

When you create a Translation Content Item for a specific Locale, Rhythmyx may already have created Translation content items for the same Locale for some of the Snippets in its Slots. In this case, Rhythmyx does not create additional Translation content items for those Snippets.

If you have more than one Locale in your system, Tools > Create Translation Item appears as an option on the menu bar when you have the main item, a field, or a Snippet in a Slot selected.

To create a Translation item of a content item:

  1. If the content item that you want to translate is not already checked in, check it in. (If you attempt to create a translation of an item that is not checked in, a dialog will prompt you to check it in.)
  2. Select the main item or the Snippet that you want to copy as a Translation item, or select one of their fields.
  3. In the menu bar, choose Tools > Create Translation Item.
  4. Rhythmyx displays the New Translation Version Properties window.

    If you cannot create a Translation Content Item, the window displays a message explaining why. Otherwise, the window displays a field in which you can specify the Locale of the new Translation Content Item.

  5. In the Locale drop list, select the Locale of the Translation content item. If a Translation content item already exists for the original content item in a Locale, that Locale does not appear in the Locale drop list.
  6. Click [Create] to save the Translation content item. Rhythmyx saves the Translation content item and Translation content items of the Snippets in its Slots. (If Translation content items already exist for any of the Snippets, Rhythmyx does not create them again).

    NOTE: Click [Close] to return to the previous window without saving the Translation item.

    If you click [Create], the Results window pops up. If the item is created successfully, the window displays a [Close] button:

  7. Click [Close] to return to the previous Active Assembly window. The window will not display the Translation item, but you can see it in Content Explorer under the same folders as the original item.

    Rhythmyx treats a Translation content item like other content items, but you cannot create a Translation content item of a Translation content item.
