Creating New Items

In Active Assembly, you have several options for creating new content items. You can:

  • Create new content items to insert into Slots;
  • Create copies of the main item; and
  • Create new main items.

When you create new content items to insert into Slots or create new main items, you enter information into the Create Item dialog.

When you copy the main item, you enter information into the Create Copy dialog.
