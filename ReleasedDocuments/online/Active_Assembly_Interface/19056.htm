Selecting a main item, a Slot, a Snippet in a Slot, or a Field

In the Active Assembly interface, you can select either the entire main item (which may be a Page or a Snippet depending on the Template chosen), a Slot, a Snippet in a Slot, or a field in an item, to edit or preview. When you initially open the Active Assembly interface, the item that you have chosen for active assembly is selected; once you have selected another component, you must select the original item again to make it the current component. You can select an object in Page view or Outline view, but it always becomes selected in the other view as well. In Page view, a selected item is bordered (or, if you have enabled Show Icons/Borders, the border becomes thicker); in Outline view, a selected item is highlighted.

To select an object in Outline view:

  1. Click the name or icon for the object (main item , Slot , Snippet , or field ) in Outline view; the object is also automatically selected in Page view.

    The following graphic shows a user clicking on the Title field to select it in Outline view. It becomes highlighted to show that it is selected. The title About Enterprise Investments also becomes bordered with a dotted line in Page view to show that it is selected.

You select objects in Page view differently if you have Show Icons/Borders or Hide Icons/Borders enabled.

To select an object in Page view when you have Hide Icons/Borders enabled:

  1. In Page view, scroll your cursor over the object and allow the cursor to hover in place for a few seconds.

    The icon for the object and the icons for its parent objects become visible in a line or a cluster depending on where your cursor is situated. Click the icon for the object (always the first icon); the object is also automatically selected in Outline view.

    The following graphic shows that a user has hovered the cursor over the Title field in the List Slot and selected the Field icon. The Title field is highlighted in Outline view, indicating that it is also selected in Outline view.

    Notice that in this example, all four icons appear because the field is contained in a Snippet, which is contained in a Slot, which is contained in the main Page. In addition, the four icons appear in a row because the user rested the cursor in a place that allowed enough room for a row.

    In the next example, the user hovers the cursor over the edge of the Snippet, so there is only room for the icons to appear vertically. Although the mouse hovered over the Snippet, the user can click any of the visible icons to select the object that the particular icon represents. In this case, the user clicks the Slot icon to select the Slot object, which becomes outlined. In addition the Slot becomes highlighted in Outline view.

To select an object in Page view when you have Show Icons/Borders enabled:

  1. Click the icon representing the object in Page view. The icon appears above or beside the object. The object is automatically selected in Outline view.

    In the following example, the user selects the icon for the Title field About Enterprise Investments. The field becomes bordered with a heavier dotted line. In Outline view, the field is highlighted.

Note: If you select a field with a link in it, you can follow the link in Internet Explorer or Mozilla Firefox by pressing CTRL-click.
