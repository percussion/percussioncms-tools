Using In-place Editing

In-place editing opens a field in the body of the page or Snippet for editing. It is used for the one-line edit box.

To use in-place editing:

  1. Open a one-line text field.

    You now have the ability to edit the field where it appears in Page view. [Update] and [Cancel] buttons appear below the field.

  2. Make changes to the field.
  3. Click [Update] to update the field on the parent page and anywhere else it appears.

    The in-place editor closes.

  4. Click [Cancel] to close the editor without making any changes.
