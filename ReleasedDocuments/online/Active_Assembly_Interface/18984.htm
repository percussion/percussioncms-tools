Choose Where to Place the Snippet Dialog

The Choose Where to Place the Snippet dialog opens if there are already Snippets in a Slot when you choose an Insert Snippet or New Snippet command. The dialog lets you choose where you want to add a Snippet in relation to the existing Snippets.

Fields in Choose Where to Place the Snippet dialog

Snippets - List of Snippets in the order that they are in the Slot. Clicking on a Snippet selects it.

Place Where - Where to place the Snippet that you are adding to the Slot in relationship to the selected Snippet under Snippets. Choices are:

Before - Add the Snippet before the selected Snippet.

After - Add the Snippet after the selected Snippet.

Replace - Replace the selected Snippet with the Snippet that you are adding.

[Show Titles] - Click this button to show just the titles of the Snippets in the Snippets box. This is useful when the Snippets are too large to fit in the window. Once [Show Titles] is selected, the button changes to [Show Snippets].

[Show Snippets] - The dialog displays this button when it is showing only the titles of the Snippets in the Slot. Show Snippets causes the dialog to show the fully formatted Snippets. Once [Show Snippets] is selected, the button changes to [Show Titles].

[Next] - Takes you to the next screen in the sequence, either the Content Browser or the Create Item dialog.

[Cancel] - Closes the dialog without doing anything.
