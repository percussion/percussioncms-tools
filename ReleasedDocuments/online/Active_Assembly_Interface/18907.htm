Create Copy Dialog

When you choose Content > Copy from the menu bar the Create Copy dialog opens. The Create Copy dialog prompts you to enter a name for the copy of the main item that you are creating. Once you click [Create], the main item is copied with all of its dependencies and other relationships. The only difference is the Title.

Fields in the Create Copy Dialog

Title - The title you give to the new copied item. This value is entered in the item's sys_title field. If the item has a displaytitle field (all FastForward Content Types include a displaytitle field), the displaytitle field is also filled in with the Title.
