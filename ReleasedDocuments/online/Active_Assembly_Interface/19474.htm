Changing an Item's Position in a Slot in Page View

To change an item's position in a Slot in Page view using drag and drop:

  1. Click on the item, and drag it above or below the item in the Slot that you want it to precede or follow.

  2. When a gray bar appears above the item if you are moving it up or below the item if you are moving it down, drop it.

    The Snippet moves to the new location.

To change an item's position in a Slot in Page view using Menu Bar or Context Menu options:

  1. Select the Snippet that you want to move.

    The Snippet is selected.

  2. In the Menu Bar click the [Up] or [Down] button

    OR

    Right-click on the Snippet and choose Up or Down.

    The Snippet moves up or down in the Slot.
