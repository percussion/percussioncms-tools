The Comparison Page

You can compare two revisions of the main content item or a Snippet in a Slot by using the Tools > Compare option in the menu bar. When you select Compare, Rhythmyx displays a Comparison Page, which has three frames. The upper left frame displays the selected Content Item, the lower left frame lets you select a revision of the Content Item, and the right frame has Item tabs that display the items assembled using selected Templates and a Compared tab that compares the text in the items. The Compared tab displays common text in both items as plain text, and displays new, deleted, and modified text with highlighting and strikethrough marks:

  • A blue strikeout indicates a deletion or change between revisions.
  • A red strikeout indicates a deletion or change between Content Items assembled using different Templates.
  • Green indicates an addition or change.

The upper and lower left frames contain the following fields and buttons:

Item - Content ID of the item displayed in this frame.

Title - Display title of the item.

Rev - Revision ID of the item.

Date - Revision Date.

Who - User who created the revision.

State - Current state of the revision.

Comment - Link to workflow comment for the revision.

Select Template - Template that Item tabs use to display Content Item. By default, the Template used in Select Template is the first Template alphabetically for the Content Item.

[Select Revision] - Opens the Revision List for the Content Item to allow you to select a different revision for comparison.
