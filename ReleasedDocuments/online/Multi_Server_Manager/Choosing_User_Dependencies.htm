Choosing User Dependencies

After you click the [User Dependencies] button in the Include Element Dependencies or Ancestors dialog, a User Dependencies dialog opens:

To add user dependencies:

  1. In the file tree under Rhythmyx server file structure, navigate to a file that you want to add as user dependency.
  2. Select the file.

    The [Add] button becomes enabled.

  3. Click [Add].
  4. The User Dependencies box displays the file name.
  5. Repeat steps 1 through 4 for each file that you want to add as a user dependency.
  6. Click [OK].

    The dialog closes and the Dependency Tree in the Include Element Dependencies or Ancestors dialog displays the user dependencies that you have added.
