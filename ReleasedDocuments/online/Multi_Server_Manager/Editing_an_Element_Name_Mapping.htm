Editing an Element Name Mapping

To edit an element name mapping:

  1. Follow steps 1 through 3 of Mapping Elements.
  2. Select a row in the Source to Target Mapping table and click [Edit].

    The Multi-Server Manager displays the Edit/Add Mapping dialog with the values from the selected mapping entered in the fields.

  3. Select a new Target Element or select blank.

    If you select blank, the Multi-Server Manager automatically checks the Add to server checkbox to add the Source Element to the target server.

  4. Click [OK].

    The Multi-Server Manager displays the edited mapping in the Source to target mapping table.
