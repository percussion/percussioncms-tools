Repository Alias

When you enter a new server registration in the Edit/New Rhythmyx Server Registration dialog and when you change the server repository through the Repository Change dialog, the Multi-Server Manager displays the Repository Alias dialog, which prompts you to enter an alias name. The alias name is the identifier for the repository that the Multi-Server Manager uses in the Server Groups Tree.

Repository Alias dialog fields

Alias Name - the repository alias name.
