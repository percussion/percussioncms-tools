Specifying ID Types

To identify ID Types:

  1. Either:
    • Right-click the Server Repository Node and select Identify ID Types.

    OR

    • Select the Server Repository Node and select Edit > Identify ID Types in the Menu Bar.

    The Multi-Server Manager displays the Identify ID Types dialog. The dialog displays a tree of all Rhythmyx elements in installable server objects that have literal (hard-coded) identifiers.

    NOTE: You can access a similar mode of this dialog in the Create Archive/Descriptor Wizard.

  2. Follow the steps in Using the Identify ID Types dialog.
  3. Click [OK].

    Rhythmyx stores all of the ID Types and will send them to the target server when you install the archive.
