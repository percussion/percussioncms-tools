Editing a Datasource Mapping

To edit datasource mappings:

  1. Follow step 1 and step 2 of Mapping Datasources.
  2. Select a mapping in the Map source to target table and click [Edit].

    The Multi-Server Manager displays the Edit/Add Datasource Map dialog.

  3. Change the source and/or target datasource.
  4. Click [OK].

    The Multi-Server Manager displays the changed mapping in the Map source to target table.
