Choose Installation Mode Dialog

The Choose Installation Mode dialog prompts you to specify whether you want to proceed through the Wizard in Typical or Custom mode. Typical mode only takes you to the Transforms dialog if mappings are not complete. Custom mode always takes you to the Transforms dialog to review and modify your mappings.

To complete the Choose Installation Mode dialog:

  1. Choose Typical to proceed in the typical mode.

    OR

    Choose Custom to proceed in the custom mode.

  2. Click [Next>>] to proceed to the Add Packages to Install dialog of the Wizard.
