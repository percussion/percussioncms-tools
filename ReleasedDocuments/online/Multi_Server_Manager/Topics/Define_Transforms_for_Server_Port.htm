Define Transforms for <Server:Port>

The Rhythmyx Multi-Server Manager provides the Define Transforms dialog for creating maps of Rhythmyx elements and of non-default datasources specified in applications. This dialog is available from the Multi-Server Manager Window and the Install Archive Wizard. It has two tabs: the Datasource Maps tab and the Element Maps tab.

The Multi-Server Manager saves the maps on the target server and uses them during installation.
