Mapping Source and Target Datasources

Rhythmyx applications may reference datasources other than the default datasource used by the source server. You must map the non-default datasources in the applications to the datasources they should reference on the target server. The Multi-Server Manager provides the Datasource Maps tab of the Define Transforms dialog for you to add, edit, or delete datasource mappings.
