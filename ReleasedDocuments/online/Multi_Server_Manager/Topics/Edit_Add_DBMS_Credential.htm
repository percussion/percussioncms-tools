Edit/Add Datasource

In the Datasource Maps tab of the Define Transforms dialog, when you click [Add] or click a row in the source to target mappings and click [Edit], the Multi-Server Manager displays the Edit/Add Datasource Map dialog. This dialog lets you insert or change a datasource mapping.

Edit/Add Datasource Map fields

Source - drop list containing all registered datasources on the source server

Target - drop list containing all registered datasources on the target server
