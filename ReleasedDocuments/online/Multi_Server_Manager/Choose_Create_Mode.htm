Choose Create Mode

The Choose Create Mode dialog prompts you to specify whether you want to proceed through the Create/Archive Wizard in Typical or Custom mode. Typical mode only takes you to the Identify ID Types dialog if hard-coded identifiers in server objects that you are installing are unidentified. Custom mode takes you to the Identify ID Types dialog to review and modify your ID types whenever you are installing server objects with hard-coded identifiers

To complete the Choose Create Mode dialog:

  1. Choose Typical to proceed in the typical mode.

    OR

    Choose Custom to proceed in the custom mode.

  2. Click [Next>>] to proceed to the Add Elements to the Archive dialog of the Wizard.
