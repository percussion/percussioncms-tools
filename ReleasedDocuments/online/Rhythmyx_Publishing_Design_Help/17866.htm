Creating a Content List

To define a Content List

  1. While displaying either version of the List Content Lists dialog, from the Menu bar, choose Action > Create Content List.

    Rhythmyx displays the Content List editor with default data.

  2. The value in the Name field defaults to ContentList_nnn, where nnn is the system-generated ID of the Content List. Values begin from 1 (thus, the first Content List you create will have the name "ContentList_1"). Change this value to a meaningful name.
  3. Optionally, enter a free-form Description of the Content List.
  4. Select the Content List Type. Options include:
    • Normal

      A Normal Content List clears the Site record, removes all existing Content Items from the target location, and publishes all Public Content Items in the Site.

    • Incremental

      An Incremental Content List removes any expired Content Items from the Site, updates any Content Items that have changed since the last publishing run, and publishes any Content Items that became Public since the last Publishing run.

  5. Select the Delivery Type of the Content List. Options include all Delivery Types defined in the system (by default, File System, FTP, SFTP, Database).
  6. Select the Item Filter to filter the initial generated list of Content Items to produce a final list of Content Items to publish.
  7. Select the Generator to generate the initial list of Content Items to publish. In most cases, when you select a Generator, the Content List editor will display one or more fields for Generator parameters. Enter the appropriate parameter values in these fields. The specific parameters depend on the Generator selected.
  8. Select the Template Expander to determine the Templates that will be used to format Content Items in the list. In most cases, when you select a Template Expander, the Content List editor will display one or more fields for Template Expander parameters. Enter the appropriate parameter values in these fields. The specific parameters depend on the Template Expander selected.
  9. To enter data in advanced fields:
    1. Click the Show Additional Details link.
    2. If you want to use an alternate Content List servlet, enter the Base Resource Path to the servlet. The value defaults to the path of current servlet
    3. If you want to add additional parameters to the Content List request, enter them in the Extra Parameters field. These parameters must be correctly formed HTML parameters.
  10. To save the Content List, in the Menu bar, click [Save].

Note: When a new Content List is created, it is not associated with any Edition. To access the new Content List, use the Content Lists node at the root of the Navigation pane to access the List Content Lists dialog. See Editing an Edition for details about associating a Content List with an Edition.
