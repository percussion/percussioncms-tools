Modifying a Content List

To modify a Content List

  1. Access the content List Editor for the Content List:
    • If the Content List is associated with an Edition, expand the Sites Node, expand a Site with which the Content List is associated, expand the Content Lists node, and double-click on the Content List you want to modify.
    • If the Content List is not associated with an Edition, expand the Content Lists node and double-click on the Content List you want to modify.
  2. You can modify the value of any field. If you change the Generator or the Template Expander, you must enter values for the associated parameters.
  3. To save your changes, in the Menu bar, choose Save.
