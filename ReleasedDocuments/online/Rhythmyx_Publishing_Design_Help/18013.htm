Modifying a Delivery Type Registration

If you modify the Spring Bean Name, the bean you specify must exist. The bean is validated when you save the Delivery Type registration.

To modify a Delivery Type Registration:

  1. In the Navigation pane, expand the Delivery Types node and double-click on the Delivery Type registration you want to modify.

    Rhythmyx displays the Delivery Type editor in the View and Edit pane.

  2. You can change the value of any field. (Note: The value in the Spring Bean Name field will be validated on save to ensure that the specified bean exists.).
  3. Click the [Save] button to save your changes.
