Removing a Location Scheme from a Context

When you remove a Location Scheme from a Context, that Location Scheme is deleted and cannot be recovered.

To remove a Location Scheme:

  1. In the Navigation pane, expand the Contexts node and double-click on the Context with which the Location Scheme you want to delete is associated.

    The View and Edit pane displays the Context editor.

  2. In the Context editor, select the Location Scheme you want to remove.
  3. In the Menu bar, choose Action > Delete Selected Location Scheme.
  4. Rhythmyx returns a confirmation page. To confirm the delete action, in the Menu bar, click Delete. To cancel the delete action, in the Menu bar, click Cancel.
  5. To save your change, in the Menu bar of the Context Editor, click Save.
