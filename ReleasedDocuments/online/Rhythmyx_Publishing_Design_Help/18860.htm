Add Context Variable Page

Use the Add Context Variable page to add Context Variables to the Site. To access the Add Context Variable page, on the Site Editor, on the Menu bar, choose Action > Add Context Variable.

Field Descriptions

Context Variable Name Name of the Context Variable. Context names must be begin with a letter, and can contain any alphanumeric characters, underscores, hyphens, or dots (periods).

Context Variable Value Value of the Context Variable. Can contain any alphnumeric characters. The value of a Context Variable is usually a path, and will often be included in a URL, so characters that are not valid in a path or URL should be avoided.

Context Drop List. Context with which the Context Variable is associated.

All Context Variables Read-only table of all Context Variables currently defined in the system.
