Associate Content List

Use the Add/Edit Content Lists dialog to add Content Lists to an Edition or to modify the data of existing associations.

To access the Add/Edit Content List dialog, from the Edition editor Menu bar, choose Action > Add Content List Association or Action > Edit Content List Association.

Field Descriptions

Filter Content Lists String to use to filter the list of Content Lists in the Content Lists field. Only Content Lists whose names match the filter are listed in the Content Lists field.

Limit to Site and Unassigned Check this box to filter the list in the Content Lists field to Content Lists already associated with the Site to which the Edition is associated and Content Lists not currently associated with any Site.

Content Lists Table of Content Lists that match the filter criteria. Select the check box of the Content Lists you want to add or change.

Assembly Context The Context to use when defining links between assembled Content Items when publishing the Edition.

Delivery Context The Context to use when determining the location to which to deliver the published output.

Authtype Used in legacy systems originally implemented in Rhythmyx Version 5.7 or earlier. The Auth Type to use when publishing the Edition.
