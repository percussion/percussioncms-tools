Editing a Site

To edit a Site:

  1. On the Navigation tab, expand the Sites node and double-click on the Site you want to edit.

    Rhythmyx displays the Sites editor in the View and Edit pane.

  2. You can modify any field, including the value of any Context Variable.
  3. To add a Context Variable:
    1. In the Menu bar, choose Action > Add Context Variable.

      Content Explorer displays the New Context Variable editor.

    2. Enter the Context Variable Name.
    3. Enter the Context Variable Value.
    4. Choose the Context to which the Value will apply.
    5. To copy an existing Context Variable, click the copy button in the row of the Context Variable you want to copy. Rhythmyx copies the data for the Context Variable into the appropriate fields. Change the Context Variable Value and Context as needed. Note that the same Context Variable Name may be used several times with different values for different combinations of Site and Context, so you may want to to copy the instance that most closely matches the values you need.
    6. In the Menu bar, click Done.
  4. To delete a Context Variable, select the Context Variable you want to delete and from the Menu bar choose Action > Delete Selected Context Variable.
  5. To save your changes, in the Menu bar, choose Save.
