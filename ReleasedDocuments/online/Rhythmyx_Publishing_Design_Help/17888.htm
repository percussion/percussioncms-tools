Maintaining a Legacy Location Scheme

To maintain a legacy Location Scheme:

  1. In the Navigation pane, expand the Contexts node and double-click on the Context with which the Location Scheme you want to modify is associated.

    If the Location Scheme is a legacy Location Scheme, the View and Edit pane displays the Legacy Scheme editor. (otherwise it displays the Scheme editor, and you are maintaining a standard Location Scheme).

  2. You can modify the Name or Description of the Location Scheme, choose a different Content Type or Template.
  3. To add a new parameter:
    1. In the Menu bar, choose Edit > Add Parameter.

      Rhythmyx adds a row to the parameter table.

    2. Enter the Name, Type, and Value of the parameter.
  4. To modify a parameter, change the Name, Type, or Value of the parameter.
  5. To change the order of parameters, select a parameter you want to move, and in the Menu bar choose Edit > Move Selected Parameter Up or Edit > Move Selected Parameter Down.
  6. To delete a parameter, select the parameter you want to delete and in the Menu bar, choose Edit > Remove Selected Parameter.
  7. Click the [OK] button to save your changes.
