Creating a Context

To create a Context:

  1. In the Navigation pane, select the Contexts node and from the Menu bar, choose Action > Create Context.

    The View and Edit pane displays the Context editor.

  2. Enter a Name for the Context. Context names must be begin with a letter, and can contain any alphanumeric characters, underscores, hyphens, or dots (periods). Optionally, enter a free-form Description.
  3. To add a Location Scheme:
    1. In the Menu bar, choose Action > Add Location Scheme.
    2. The View and Edit pane displays the Scheme editor.
    3. Enter a Name for the Location Scheme. Location Scheme names must be begin with a letter, and can contain any alphanumeric characters, underscores, hyphens, or dots (periods). Optionally, enter a free-form Description.
    4. Select the Content Type and Template to which the Location Scheme will be applied.
    5. Enter the JEXL Expression to generate the location. For details about the syntax of a JEXL expression, see the Apache JEXL Syntax documentation. (Note: requires registration on the Rhythmyx forums.) Rhythmyx also includes an extensive array of JEXL variables and functions.

      You may want to test the JEXL expression to ensure that it works correctly and returns the desired results. To test the expression, click the Show Test Panel link to display the test panel. Choose the Site you want to use to test the expression. Enter the Item Path to the Content Item you want to use to text the expression. Click the button to access the Browse Site Root Path dialog to browse to the Content Item. Enter any Extra Parameters required by the expression; parameters should be expressed as name=value pairs, pairs separated by an ampersand character ("&"); for example: sys_contentid=301&sys_revision=1&sys_slotid=501.. When you have defined this data, click the [Evaluate JEXL Expression] button. The Result is returned (either the path generated by the expression or the error returned by it), with an indication of the Status (Success or Failure).

    6. Click the [OK] button to save the Location Scheme.

      The View and Edit pane displays the Context editor.

    7. Repeat steps a through f to add more Location Schemes.
  4. Click the [OK] button to save the Context.
