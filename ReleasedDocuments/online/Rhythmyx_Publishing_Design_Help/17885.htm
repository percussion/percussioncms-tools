Context Editor

Use the Context editor to create and maintain Contexts.

To access the Context editor:

  • in the Navigation pane, select the Contexts node and in the Menu bar, choose Action > Create Context;
  • In the Navigation pane, expand the Contexts node, select the Context you want to edit, and in the Menu bar, choose Action > Edit Selected Context;
  • in the Navigation pane, expand the Contexts node and double-click on the Context you want to edit;
  • in the Contexts List, double-click on the Context you want to edit.

Field Descriptions

Name Name of the Context. Context names must be begin with a letter, and can contain any alphanumeric characters, underscores, hyphens, or dots (periods).

Description Free-form description of the Context.

Default Scheme The Location Scheme to use when no Location Scheme matches the combination of Content Type and Template being processed.

Location Schemes Table:

Name Name of the Location Scheme

Used By The combination of Content Type and Template to which the Location Scheme can be applied.

Description The text description of the Location Scheme.
