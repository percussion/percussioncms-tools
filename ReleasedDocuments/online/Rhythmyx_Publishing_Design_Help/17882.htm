Modifying a Location Scheme

To modify a Location Scheme:

  1. In the Navigation pane, expand the Contexts node and double-click on the Context with which the Location Scheme you want to modify is associated.

    The View and Edit pane displays the Scheme editor.

  2. You can modify the Name or Description of the Location Scheme, choose a different Content Type or Template, or modify the JEXL Expression. For details about the syntax of a JEXL expression, see the Apache JEXL Syntax documentation. (Note: requires registration on the Rhythmyx forums.) Rhythmyx also includes an extensive array of JEXL variables and functions.

    If you modify the JEXL expression, you may want to test it to ensure that it works correctly and returns the desired results. To test the expression:

    1. Click the Show Test Panel link to display the test panel.
    2. Choose the Site you want to use to test the expression.
    3. Enter the Item Path to the Content Item you want to use to text the expression. Click the button to access the Browse Site Root Path dialog to browse to the Content Item.
    4. Enter any Extra Parameters required by the expression. Parameters should be expressed as name=value pairs, pairs separated by an ampersand character ("&"); for example: sys_contentid=301&sys_revision=1&sys_slotid=501.
    5. Click the [Evaluate JEXL Expression] button. The Result is returned (either the path generated by the expression or the error returned by it), with an indication of the Status (Success or Failure).
  3. Click the [OK] button to save your changes.
