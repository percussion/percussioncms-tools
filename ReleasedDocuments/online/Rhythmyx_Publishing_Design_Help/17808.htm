Site Editor

Use the Site editor to create and maintain Sites.

To access the Site editor:

  • in the Navigation pane, select the Sites node and in the Menu bar, choose Action > Create Site;
  • in the Navigation pane, expand the Sites node, select the Site you want to edit, and in the Menu bar, choose Action > Edit Site;
  • in the Navigation pane, expand the Sites node and double-click on the Site you want to edit;
  • in the Sites List, double-click on the Site you want to edit.

Field Descriptions

Name Name of the Site. Site names must be begin with a letter, and can contain any alphanumeric characters, underscores, hyphens, or dots (periods). Defaults to Site_0.

Description Optional free-form text description of the Site.

Rhythmyx Path The source Folder under the //Sites node in Content Explorer within which the Sites Folders and content reside. The Site path must begin with the //Sites node, but need not end as an immediate child of that node (in other words, both //Sites/EnterpriseInvestments and //Sites/Investments/EnterpriseInvestments are valid Site root paths). The path specified must exist in the Navigation Pane of Site Explorer. If the specified path does not exist, you cannot save the Site. Use the Browse button to access the Browse Site Root Path page, which you can use to browse the //Sites node to find the root location you want to want to use.

Global Template The Global Template used by default to format pages in the Site. Options include all Global Templates defined in the system. The Global Template may be overridden for a specific Folder, or for a specific Page Template.

Published URL The URL used to navigate to the Site when it is published.

Published Path (Optional) The root or base file path to which to publish the Site content.

Site Details

FTP Server IP Address (Optional; used when specifying an FTP or SFTP delivery type.) The IP address used to establish a connection to an FTP or SFTP server.

FTP Port (Optional; used when specifying an FTP or SFTP delivery type.) The port used to connect to the FTP or SFTP server.

User Name (Optional; used when specifying an FTP or SFTP delivery type.) The user name to use when authenticating to the FTP or SFTP server.

Password (Optional; used when specifying an FTP or SFTP delivery type.) The password to use when authenticating to the FTP or SFTP server.

Unpublish Flags Character (case-insensitive) used to flag a Workflow State for Content to be unpublished; multiple characters can be entered separated by commas. The Publishable Keyword must include a Keyword Choice with a Value matching each character specified. For more details see "Configuring Unpublish Flags" in the Rhythmyx Technical Reference Manual. Workflow States must be assigned one or more of these Publishable options to unpublish. Defaults to u, which is the Keyword Choice Value of the Archive Keyword Choice.

Nav Theme Theme indicator for stylesheet coding in Managed Navigation. Used in legacy XSL-based Managed Navigation implementations for Rhythmyx Version 5.7 and earlier.

Allowed Namespaces Comma-separated list of XML namespaces to preserve when cleaning up namespaces in rich-text fields when assembling content for the Site. Entries in this list should have a corresponding entry in the file <Rhythmyxroot> /rxconfig/server/stylesheetCleanupFilter.xml. If a namespace specified for the Site does not have a corresponding entry in the that file, it will be filtered out.

Context Variables

Name Name of the Property.

Context Read-only. Output Context with which the associated value will be used.

Value Value of the Context Variable for the specified Context.
