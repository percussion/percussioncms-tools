Server Properties Dialog - Logging Properties

This section of the Server Properties Dialog contains fields that define the logging characteristics of the Rhythmyx server. The initial values for these fields are defined during the installation of Rhythmyx.

Name

Description

Default

Logging Type

Logging database type. Choose either DBMS or File.

Log File Url

URL of the file used when the Logging Type is "File".
