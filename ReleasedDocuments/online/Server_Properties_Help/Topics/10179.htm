Server Properties Dialog - SSL Properties

This section of the Server Properties Dialog allows you to enable or disable Secure Socket Layer (SSL) support in the Rhythmyx server. When the SSL KeyStore File parameter has a value, SSL support is enabled; when the parameter does not have a value, SSL support is disabled. By default, the SSL KeyStore File parameter and the other SSL parameters are not defined and SSL support is disabled.

Parameter

Description

Default

SSL KeyStore File

Name and location of the keystore file that contains the server's public and private key and digital certificates.  The path specified must be relative to the directory AppServer/server/rx/conf.

None

SSL Key Password

The encrypted password for validating the key specified in SSL KeyStore File.

Required when SSL KeyStore File is defined.

None

Confirm SSL Key Password

The SSL Key Password reentered, to ensure that you entered the intended password in SSL Key Password.

None

Allowed SSL Ciphers

 

A list of allowed cipher suites (encryption protocols). Supported cipher suites are:

SSL_DHE_DSS_WITH_DES_CBC_SHA

SSL_DHE_DSS_WITH_3DES_EDE_CBC_SHA

SSL_DHE_DSS_EXPORT_WITH_DES40_CBC_SHA

SSL_RSA_WITH_RC4_128_MD5

SSL_RSA_WITH_RC4_128_SHA

SSL_RSA_WITH_DES_CBC_SHA

SSL_RSA_WITH_3DES_EDE_CBC_SHA

SSL_RSA_EXPORT_WITH_RC4_40_MD5

For descriptions of these cipher suites, see http://www.mozilla.org/projects/security/pki/nss/ssl/fips-ssl-ciphersuites.html.

Not required when SSL KeyStore File is defined.

SSL_RSA_WITH_RC4_128_MD5;
SSL_RSA_WITH_RC4_128_SHA; SSL_DHE_DSS_WITH_3DES_EDE_CBC_SHA

 

NOTE: Rhythmyx uses these three cipher suites as defaults if you do not select any from the Allowed SSL Ciphers field. If you select one or more, your selections are used instead of the defaults.

SSL Port

Port that receives requests that require an SSL connection.

 Not required when SSL KeyStore File is defined.

9443

To test whether you have implemented SSL correctly in the Rhythmyx server, enter in the browser:

https://<rhythmyxhostname>:9443/Rhythmyx

where <rhythmyxhostname> is the name of the machine where your Rhythmyx server resides and 9443 is the SSL port you specified when you configured your server to use SSL. If your browser presents the login dialog, SSL is working correctly.
