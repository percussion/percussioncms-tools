Maintaining Server Properties

During installation, Rhythmyx defines the default parameters and values used by the Rhythmyx server in the following file:

"./rxconfig/Server/server.properties

To view or modify this file, use the Rhythmyx Server Properties editor. Using this tool, you can view and modify the following groups of Rhythmyx server properties:

  • General
  • Logging
  • Applications
  • SSL
  • Application Server

CAUTION: Do not save server properties while Rhythmyx is running. Be sure to shut down Rhythmyx first. Otherwise, your server would be in an inconsistent state, possibly causing many errors.

To modify server properties:

  1. Shut down the Rhythmyx Server.
  2. To open the Server Properties Dialog, use one of the following methods:
    • In Windows, run the file RhythmyxServerPropertiesEditor.exe in the Rhythmyx root directory.
    • On the command line, enter the following command at the root directory:

      RhythmyxServerPropertiesEditor

      If you are on a server with no GUI, edit the server.properties file directly using a text editor such as Notepad or vi.

  3. Make your changes in the Server Properties Dialog and click the [Save] button to update your server properties.

    You may now restart the Rhythmyx Server.
