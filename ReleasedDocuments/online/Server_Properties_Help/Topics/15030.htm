Server Properties Dialog - Application Server Properties

This section of the Server Properties dialog contains fields that specify ports used internally by the Rhythmyx application server. In general, you should not need to modify these fields after installation unless you install another product later that uses one of these ports.

The details about the use of each port are specific to the application server. The purpose of this section of the dialog is to allow you to change these ports in event of a port conflict with another application.
