Editing a Connection Configuration

To edit a Connection configuration:

  1. On the Connections subtab of the Datasource tab, select the Connection configuration you want to edit and click the [Edit] button.

    The Server Administrator displays the Connection Configuration dialog.

  2. You can modify the data in any field.
  3. Click the [OK] button to save your changes.
  4. On the Server Administrator, click the [Apply] button to apply your changes to the server.

You must restart the Rhythmyx server before your changes will take effect.
