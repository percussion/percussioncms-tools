Driver Configuration Dialog

Use the Driver Configuration dialog to create and maintain database driver definitions.

To access the Driver Configuration dialog:

  • on the Drivers subtab of the Datasource tab, click the [Add] button
  • on the Drivers subtab of the Datasource tag, select a driver definition and click the [Edit] button.

Field Descriptions

Driver Name of the JDBC driver

Class Fully-qualified name of the driver class.

Container Type Mapping Dataset used to connect and interface with a specific version of the specified RDBMS. Drop list includes all currently-supported RDBMS versions, but you can enter a dataset name manually.

Database Dialect Specifies the dialect to use in the data persistence layer to communicate with the RDBMS.
