Enabling and Disabling the Pause Option

The pause option is on by default.  

To disable the pause option:

  1. Open the Server.Properties file.
  2. Insert the line noPauseOnExit=TRUE.  

To turn the pause option back on, set noPauseOnExit=FALSE.
