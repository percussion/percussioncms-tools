Stopping Rhythmyx Server as a Daemon in a UNIX Environment

To stop Rhythmyx server when running as a daemon, change to the Rhythmyx bin directory and enter the following:

./RhythmyxDaemon stop <Rhythmyx root directory>

When you press <Enter>, the Rhythmyx server will shut down.
