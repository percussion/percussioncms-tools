Ordinarily, you should not need to modify these definitions. You might need to modify a configuration if you are upgrading your RDBMS.

Use the Driver Configuration dialog to define new definitions and edit existing definitions.

You can add new Driver definitions or edit existing definitions.

To delete a driver definition, select the definition you want to delete and click the [Delete] button. Be sure you are not using the driver definition before deleting it. Rhythmyx cannot validate that a specific definition is in use in your system.
