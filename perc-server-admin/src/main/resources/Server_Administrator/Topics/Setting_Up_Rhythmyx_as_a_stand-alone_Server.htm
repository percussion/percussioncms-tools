Setting Up Rhythmyx as a stand-alone Server

You can reconfigure Rhythmyx as a stand-alone server even if you installed Rhythmyx on a machine that has a web server on it.  As a stand-alone server, Rhythmyx doesn't use any APIs to hook natively into the existing web server to process URLs.  Instead, all URLs go directly to the Rhythmyx server based on the address (either the port number or the IP address/host address).

Exercise caution when configuring Rhythmyx to port 80 when an existing web server is already using that port.
