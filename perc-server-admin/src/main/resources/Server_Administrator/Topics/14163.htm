Editing a JNDI Datasource Configuration

To edit a JNDI Datasource configuration:

  1. On the JNDI subtab of the Datasources tab, select the Datasource Configuration you want to modify and click the [Edit] button.

    The Server Administrator displays the JNDI Datasource Configuration dialog JNDI Datasource Configuration dialog with the data for the Datasource configuration you selected.

  2. If the Datasource is used on a Connection Configuration, you cannot change the value in the Name field. Otherwise, you can modify the data in any field.
  3. When you have made your modifications, click the [OK] button to save your changes.
  4. On the Server Administrator, click the [Apply] button to apply your changes to the server.

You must restart the Rhythmyx server before your changes will take effect.
