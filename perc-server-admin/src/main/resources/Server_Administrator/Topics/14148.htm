Maintaining Connection Configurations

A Datasource Connection defines a connection to a specific database or schema. At least one of the Connections must be designated as the CMS Repository. If the specified Connection does not include data from the Rhythmyx Repository, the server will fail when started.

Use the Connection Configuration dialog to create new Connection configurations and to modify existing Connection configurations.

You can create new Connection Configurations and modify existing Connection configurations.

To delete a Connection configuration, select the Connection configuration you want to delete and click the [Delete] button.

To designate a connection configuration as the CMS repository, click in the CMS Repository radio button of the row of the Connection configuration you want to designate. Note that if the designated Connection does not contain CMS data, the Rhythmyx server will fail to start.
