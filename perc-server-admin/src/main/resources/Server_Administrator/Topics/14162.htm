Adding a JNDI Datasource Configuration

To add a JNDI Datasource configuration:

  1. On the JNDI subtab of the Datasources tab, click the [Add] button.

    The Server Administrator displays the JNDI Datasource Configuration dialog.

  2. In the Name field, enter a name for the Datasource configuration AFTER the jdbc/. (Note: the convention when naming Datasource configurations is to prepend this string to the name.)
  3. In the Driver drop list, choose the driver used to connect to your RDBMS.
  4. In the Server field, enter the name of the host of your database server (MS SQL Server ) or the Connect string for the server (Oracle).
  5. Enter the Userid and Password Rhythmyx will use to authenticate with the database server.
  6. Optionally, modify the values in the Minimum Connections, Maximum Connections, and Idle Timeout fields.
  7. Click the [OK] button to save your configuration.
  8. On the Server Administrator, click the [Apply] button to apply your changes to the server.

You must restart the server before your changes will take effect.
