Adding a Connection Configuration

To add a new Connection Configuration:

  1. On the Connections subtab of the Datasources tab, click the [Add] button.

    The Server Administrator displays the Connection Configuration dialog.

  2. Enter a Name for the Connection configuration.
  3. In the JNDI Datasource drop list, choose the Datasource to connect to your RDBMS server.
  4. If your RDBMS is MS SQL Server:
    1. Enter the name of the Database to which you want to connect.
    2. Enter the Origin of the database to which you want to connect.
  5. If your RDBMS is Oracle, enter the Schema you want the Connection configuration to use.
  6. Click the [OK] button.
  7. On the Server Administrator, click the [Apply] button to apply your changes to the server.

You must restart the Rhythmyx server before your changes will take effect.
