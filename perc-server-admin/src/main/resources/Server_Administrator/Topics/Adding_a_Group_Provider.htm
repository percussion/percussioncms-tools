Adding a Group Provider

To add a group provider:

  1. Add a Directory Configuration or Edit a Directory Configuration.
  2. Click the Insert New Entry button   to the right of the "Group Providers to Make Availa..." row near the bottom of the dialog.

    Rhythmyx displays a drop list showing all existing group providers. You can select an existing provider by double-clicking on its name.  If the drop list is empty, click the arrow and click on Create new to create a new group provider.

    Rhythmyx displays the JNDI Group Provider Details dialog.

  3. Enter information in the dialog and optionally, edit the default information.
  4. Click [OK] to save the group provider.
  5. Click [Apply] to commit the changes to the Rhythmyx server.
