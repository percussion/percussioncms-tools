Editing a Database Driver Definition

To edit a Driver definition:

  1. On the Drivers subtab of the Datasources tab, select the driver definition you want to edit.
  2. Click the [Edit] button.

    The Server Administrator displays the Driver Configuration dialog.

  3. If a Driver definition is used in a Datasource configuration, the value in the Name field cannot be changed. Otherwise, you can modify any field.
  4. Click the [OK] button to save your edits.
  5. On the Server Administrator, click the [Apply] button to apply your changes to the server.

You must restart your server before your changes can take effect.
