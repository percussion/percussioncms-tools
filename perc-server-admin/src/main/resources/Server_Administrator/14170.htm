Datasource Connection Configuration Dialog

Use the Connection Configuration dialog to add new Datasource Connection configurations or to modify an existing Connection configuration.

To access the Connection Configuration dialog:

  • on the Connections subtab of the Datasources tab, click the [Add] button.
  • on the Connections subtab of the Datasources tab, select the Connection configuration you want to modify and click the [Edit] button.

Field Descriptions

Name Name of the Connection configuration.

JNDI Datasource Datasource to use for this connection. Options include all Datasources defined in the system.

Database Name of the database. Required for MS SQL Server. Leave blank for Oracle.

Schema/Origin For MS SQL Server, origin of the database specified in the Database Name field. For Oracle, the schema of the Connection configuration.
