Maintaining Datasources

A datasource defines the data Rhythmyx uses to connect to a RDBMS installation, and to the specific database or schema in that installation.

You can

  • maintain driver definitions

    Driver definitions specify the database drivers to use to connect to databases of a specific types. Rhythmyx is shipped with a set of pre-defined driver definitions. Under normal circumstances, you should not have to modify driver definitions.

  • maintain JNDI Datasource configurations

    JNDI Datasource configurations define the data specifying the connection to a specific RDBMS installation, the authentication data used to connect to that data, and connection-management information.

  • maintain Connection configurations

    Connection configurations specify the database or schema to which Rhythmyx can connect. At least one connection must be defined as the CMS Repository.
