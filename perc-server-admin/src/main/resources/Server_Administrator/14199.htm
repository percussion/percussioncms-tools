Modifying a DBMS Table Security Provider

To modify a DBMS Table security provider:

  1. Select the security provider you want to modify and click the [Edit] button.
  2. If the security provider is a DBMS Table security provider, The Server Administrator displays the DBMS Table Security Property Details dialog.
  3. You can change any value on the Provider Properties, Backend Connections, and Authentication tabs.
  4. To add a new attribute mapping:
    1. Select the Attributes tab.
    2. To add all columns as attributes, click the [Add All] button.
    3. To add a single attribute, click the new button .

      The Server Administrator opens a new line on the tab.

    4. use the following code to associate a user attribute with a column name:

      User/Attribute/attributename=columnname

  5. Do delete an attribute mapping, select the attribute mapping you want to delete and click the delete button .
  6. Click the [OK] button
