Starting Rhythmyx Server as a Windows Service in Windows NT

To start Rhythmyx server as a Windows service in Windows NT:

  1. Click the [Start] button, then select Settings > Control Panel. In the Control Panel, double-click the Services icon.

      Rhythmyx will display the Services dialog.

  2. Select the Percussion Rhythmyx Service.
  3. Click [Start].
