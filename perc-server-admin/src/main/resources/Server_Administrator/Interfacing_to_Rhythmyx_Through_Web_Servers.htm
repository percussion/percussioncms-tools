Interfacing to Rhythmyx Through Web Servers

You can interface to Rhythmyx as a servlet on J2EE web and web application servers and to Microsoft IIS as an ISAPI extension.

You must interface to Rhythmyx through a web server if you want to use the Web Server security Provider.

When connecting to Rhythmyx through a webserver, use the URL:

http://<server>:<port>/Rhythmyx/<appname.....>

If you to not enter the correct URL, the Rhythmyx server will not respond correctly.
