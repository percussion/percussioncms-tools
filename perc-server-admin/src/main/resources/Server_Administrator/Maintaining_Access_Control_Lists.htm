Maintaining Access Control Lists

Access control allows a server to give different permission levels to different users.  Rhythmyx assigns access control permissions only after a security provider authenticates the user.  An Access Control List (ACL) describes each possible set of users and their respective access levels.

There are two levels of ACLs:

Application ACL - Defined by designers, each application has its own ACL for users and their permissions for a specific application.

Server ACL -  Defined by administrators, provides ACL for the Rhythmyx server.
