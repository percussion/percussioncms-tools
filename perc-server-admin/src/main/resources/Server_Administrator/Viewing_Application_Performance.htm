Viewing Application Performance

You can view statistics and details about any active application currently residing on the Rhythmyx server.

To view application statistics:

  • In the Status window, select the application and click Details.

You see the following server information:

Update Interval:

Period of time (in milliseconds) between Event Counter and Event Timer updates.

Uptime:

Length of time the application has been active on the server.

Event Counters:

Number of Processed, Failed, and Pending information transfers during uptime.

Event Timers:

 Minimum, Maximum, and Average event time (in milliseconds) during uptime.
