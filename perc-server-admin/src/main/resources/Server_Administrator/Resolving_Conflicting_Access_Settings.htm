Resolving Conflicting Access Settings

To choose how an access level is selected for users in multiple groups or Roles:

In the Server ACL sub-tab, for the If multiple group matches found drop list, choose one of the following options:

  • Merge entries, Max. access - Give user all selected access levels among all entry instances.
  • Merge entries, Min. access - Gives user only the redundant access levels among all entry instances.
  • Max. access - Give user highest level of access among all entry instances.
  • Min. access - Give user lowest level of access among all entry instances.
