Inserting Inline Images into a Content Editor

For details about inline images, see Using Inline Links.

To insert an inline image into a Content Editor:

  1. Place the cursor in the location where you want to insert the formatted content from another Content Item.
  2. Click the inline image button .

    Rhythmyx displays the Content Browser.

  3. Browse or search for the Image Content Item you want to add to the Content Item you are editing and select an item.
  4. Choose the Template for the content item to which you want to link and click [Select].

  5. The image is added, and the Content Browser automatically closes.
