Using the Rich Text Editor Control

Most Rhythmyx content editors include at least one field that uses a rich text editor control. When you open a content editor that uses a rich text editor control, you see a read-only placeholder box instead of the rich text editor control:

To access the control for a field, click on the placeholder box. The rich text control loads for that field. If your content editor includes multiple fields that use rich text editor controls, you can only activate one of the fields at a time. When you click on a rich text editor field, it becomes activated, but the previously activated field, if any, becomes a read-only placeholder box.
