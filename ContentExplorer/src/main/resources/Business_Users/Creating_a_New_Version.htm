Creating a New Version

You can create new Versions of Content Items from the Action Menu. By default, you can only create Versions of Content Items that are checked in and in a Public state.

A new Version has the Community of the user who creates the new Version, not the Community of the Content Item originator.

To create a new Version of a Content Item from the Action Menu:

  1. In the Action Menu select Create > Promotable Version.

    Rhythmyx creates a new Version of the Content Item, assigns a unique ID to the new Content Item, and opens the new Content Item in a Content Editor.

  2. Edit the fields you want to modify for the new Version.  
  3. Click the [Update] button to save the Version..  

    If you do not click [Update], Rhythmyx will save the new item but not the changes you made to it.

  4. Click [Close].

    Rhythmyx saves the new Version. You can find it in your Views, but not in the folder of the owner Content Item. If you want to move it into the owner's folder, copy it from a Search or a View.

    When the Version becomes public, Rhythmyx archives the owner Content Item.
