Fields

When you enter a term into Search for, you can specify a specific field in a content item. Otherwise, the search looks through all non-binary fields in the content item that are enabled for searching.

You can search any field by typing the field name followed by a colon ":" and then the term you are looking for.

As an example, let's assume a content item contains two fields, title and text. If you want to find the document entitled "The Right Way" which contains the text "don't go this way", in Search for, you can enter:

title:"The Right Way" AND text:go

or

title:"Do it right" AND right

The field indicator is not required if you want to search all fields enabled for searching.

Note: The field is only valid for the term that it directly precedes, so the query

title:Do it right

Will only look for "Do" in the title field. It will look for "it" and "right" in all fields enabled for searching.

Fields are enabled for searching when they are created or modified in the Rhythmyx Workbench Field and Field Sets editor. See the Rhythmyx Workbench online help for more information.
