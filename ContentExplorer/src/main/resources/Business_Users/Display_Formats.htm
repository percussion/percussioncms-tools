Display Formats

A display format defines the columns included in the Display pane when you select a view, folder or search node in the Navigation pane. It also defines the categories, if any, displayed below the view, folder or search node in the Navigation Tree.

In the Display pane, one display format could display many columns:

and another could display just a few columns:

In the Display pane, each display format has a default sort, but you can sort by any of the columns by clicking on the column header. Successive clicking on the column header toggles between an ascending and a descending sort. If you want to make a column wider or narrower, drag the vertical division line between that column's header and the adjacent column header.

Under the node in the Navigation Tree, the display format determines categories that appear assubnodes. In the following graphic, the subnodes under Inbox and Outbox are determined by their Display Formats.

You can change the display format through the Change Display Format option in the right-click menu.

To change a display format by choosing a new format:

  1. In the Navigation pane, select the node for the folder, view, or saved search whose display format you want to change, and in the Menu Bar, choose [node name] > Change Display Format > [display format name].

    OR

    Right-click the node for the folder, view, or saved search whose display format you want to change, and in the drop menu, choose Change Display Format > [display format name].

    Rhythmyx applies the new display format to the folder, view, or saved search.
