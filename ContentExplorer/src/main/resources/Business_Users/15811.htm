Customizing a Search When Using the Database Search Engine

The Search dialog includes a default set of fields when you first call the search. If you want to search on different fields, you must customize the search dialog. To customize the search dialog:

  1. Start a search by double-clicking the New Search node, choosing Content > Search from the menu bar, or right-clicking in the Navigation Tree pane and choosing Content > Search from the pop-up menu.
  2. At the bottom of the search dialog, click the [Customize] button.

    Rhythmyx displays the Field Selection Editor.

  3. In the Category drop list, choose the category of Content Editor fields that you want to search on:
    • System – system Content Editor fields
    • Shared – shared Content Editor fields
    • <Content Type> – local Content Editor fields for the Content Type
    • All – all Content Editor fields
  4. After you choose a category the Available Fields table displays all fields available as a selection criteria in the category.
  5. To see only fields within the selected Category that match a certain string, enter a string in Field Filter. The string matches with the Display Label column and is case-sensitive. To apply the filter to the Available Fields, click .
  6. To include a search field, click the row for the field in the Available Fields table and click .. Include any number of fields.

    The Selected Fields table displays the fields that you include. It also displays the fields that are included by default.

    Note that some fields cascade. In other words, the values available in one field depend on the value you specify in another field. Cascading fields may be optional or required. If a cascading field is required, Rhythmyx includes it automatically. If the field is optional, Rhythmyx displays a message notifying you about the dependency and offering to include the dependency.

  7. To remove a search field after selecting it, choose . You can remove any filed except Title (sys_title).
  8. To change the order in which the Content Search dialog displays the custom search fields, click on a field and click to move the field up, or click to move the field down.
  9. Click [OK].

    The dialog closes and Rhythmyx displays the selected fields in the Search Criteria section of the Content Search box.

    Each time you create and run a new search, Rhythmyx saves the search query in the Search Results node. If you do not run additional searches, the search query in Search Results is saved for the current session only; you may rerun it during the current session by clicking Search Results.

    You can save your search query to make it available in future sessions and to other users.

When entering criteria for a Search, the values available in the Content Type field depend on the Community selected. If no Community is specified, all Content Types are available. However, if you specify a Community, only Content Types associated with that Community are available.
