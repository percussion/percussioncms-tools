Adding Related Content Using the Document Assembly Window

You can add related content to a Slot in a Content Item through the Document Assembly Window by:

  • searching for related content and selecting it;
  • dragging and dropping the Content Item's existing related content;
  • copying and pasting the Content Item's existing related content;
  • creating a new Content Item in the Slot.
