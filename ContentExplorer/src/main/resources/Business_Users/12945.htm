Lifecycle Analysis

If you have report administrative privileges for Rhythmyx content, you can retrieve Lifecycle Analysis reports on the content in your system. (Contact your system administrator to find out whether you have report administrative privileges.)

Percussion Software provides the following reports with Rhythmyx:

Full name

Name on Excel Tab

Description

Stale Content

Stale Content

Detailed listing of Content Items that have not yet been Public and have not been modified in the last seven days, sorted by the number of days since each Content Item was last touched; shows the date last modified, the last user that checked out the Content Item, the user that created the Content Item, and the Community of the Content Item.

Content Inventory by Community and Workflow

Content Inventory C-W

Pivot table view of the count of Content Item of each Content Type in each Community; you can filter this report by Content Type or by Community.

Content Community Details

Content Community Details

Listing of Content Items by Community, with metadata for each Content Item.

Content Inventory by Content Type and Workflow

Content Inventory CT-W

Pivot table view of the count of Content Items of each Content Type in each Workflow State; you can filter this report by Content Type, by Workflow, and by Workflow State.

Content Inventory Details

Content Inventory Details

Detailed listing of Content Status data for all Content Items in the system

Publish Status Summary by Community and Content Type

Publish Status Summary C-CT

Pivot table view of the count of the published Content Items in each Community, broken down by Publication ID, Content Type and publishing status. By default, this report shows data for only the last three Editions that were published. Your system administrator can modify the configuration of the report to modify the number or Editions whose data will be returned. (NOTE: If you publish the same Content Item to multiple Folders using the same Template, the report only lists the Content Item Template once. The report lists the data for the last instance of the Content Item Template published.)

Publish Status Details

Publish Status Details

Detailed listing of the publishing data for all published Content Items.

Your system administrator can develop additional reports if you need them.

By default, Rhythmyx delivers report results using Microsoft Excel. The minimum supported version is Excel XP. Earlier versions of Excel do not support the macro code required for Lifecycle Analysis reporting. Note that this workbook file requires at most Medium macro security on your Excel installation. Medium macro security will allow Excel to run macros, but it will ask you for permission to run them when you open the workbook. High security disables all macros, so Excel will not be able to retrieve data from the Rhythmyx server.

If you prefer to use a different reporting tool, contact your system administrator.
