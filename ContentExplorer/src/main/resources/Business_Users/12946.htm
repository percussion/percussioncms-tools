Retrieving Lifecycle Analysis Reports

The Rhythmyx Excel file uses macros to communicate with the Rhythmyx server. You must set the security on Excel to Medium at most to be able to retrieve Rhythmyx data. Medium macro security will allow Excel to run macros, but it will ask you for permission to run them when you open the workbook. High security disable all macros, so Excel will not be able to retrieve data from the Rhythmyx server.

To retrieve Lifecycle Analysis reports:

  1. In Content Explorer, open a Site Folder.
  2. Right-click anywhere in the Content pane and from the popup menu choose Lifecycle Analysis.

    Rhythmyx launches a new browser session to contain the Microsoft Excel workbook. This browser session displays a download dialog.

  3. On the download dialog, choose to open the file from its current location.

    The browser displays a login dialog.

  4. Enter your Rhythmyx User name and Password.

    MS Excel opens in the browser with the current data for all reports. Each report is displayed on its own spreadsheet (tab) of the workbook.

  5. You may want to save the workbook to provide a snapshot of the current data, or to make the workbook available for later use. To save the workbook, in the Menu bar, choose File > Save As, and choose the location where you want to save the workbook.
