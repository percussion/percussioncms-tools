Creating a Test Site

To copy a Site folder tree:

  1. Right-click the Site folder that you want to copy and choose Copy.
  2. Right-click on the root Site Folder and choose Paste > As Copy.

    The Copy Site wizard opens.

  3. Press [Next].

    The wizard moves to the next page. The New Folder Name and New Site Name fields are empty.

  4. Enter names in New Folder Name and New Site Name.

  5. Click [Next].

    The wizard moves to the next page.

  6. Click one of the following options:
    • Copy Site Folders - copy the folders but none of their content items
    • Copy Site Folders with Navigation - copy the folders and their navigation content items
    • Copy Site Folders, Navigation and Content - copy the folders and all of their content items
  7. Click [Next].

    The wizard moves to the page below if you have selected Copy Site Folders, Navigation and Content; otherwise it moves to the page shown in the next graphic.

  8. Click one of the following options:
    • Copy Content as link - For each non-navigational content item, creates a link to the original item.
    • Copy Content as new copy - For each non-navigational content item, creates a new copy of the original item.
  9. Click [Next].

    The wizard moves to the next page, which allows you to choose different Communities for your copied folders and content.

  10. By default, the Target Community column displays the same Community as the Source Community column. If you want to choose a different Target Community, click in the Target Community row and access the drop list. The drop list includes all Communities that include the same visible Content Types as those in the source Community (and additional Content Types, if any).

  11. Click [Next].

    The wizard moves to the next page, which confirms your choices.

  12. Click [Next].

    After the wizard completes its processing, it closes. Content Explorer displays your new Site folder. Note that the content in this folder is all in the Draft State and is created by the user rxserver.

NOTE: If you have Site-specific Action Menu entries, such as Publish Now or Active Assembly Preview, you will need to create new Action Menu Entries, specifying the new Site ID on the Command tab of the Action Menu Entry. If you have any Community-specific Action Menu Entries and you move the copied Site to a new Community, you will need to create new Action Menu Entries specifying the new Community on the Command tab of the Action Menu Entry. For details, see Customizing the Active Assembly and Content Explorer Interfaces.
