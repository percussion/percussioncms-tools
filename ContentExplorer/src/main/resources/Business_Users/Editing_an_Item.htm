Editing an Item

When a Content Item is opened via the Action Menu Edit option, Rhythmyx displays the item in Edit mode within the Content Editor. A Content Editor, in Edit mode, makes the fields available for editing, and may display links to Active Assembly Templates.

If a Content Item is available to you to edit, double-clicking on it opens it in a Content Editor in Edit mode; if the Content Item is only available to you to view, double-clicking on it opens it in a Content Editor in View mode.

At the top of the Content Editor is a Properties link that takes you to the Properties page for the Content Item.

To edit content in a Content Editor:

  1. Click the Action Menu and choose Edit or double-click on the Content Item.

    If you have not already checked out the Content Item, Rhythmyx checks it out to you now. Rhythmyx displays all of the Content Item's fields in a Rhythmyx Content Editor.

  2. Change any of the accessible fields. The fields with an asterisk (*) next to them are required.
  3. If available, click a Template link to edit the item in Active Assembly.
  4. If available, click [Edit All] to modify related content in the Editor Assembly window.
  5. If a Detail Editor is associated with the Content Editor, click the [Edit] button to activate the Detail Editor and manage its content.
  6. Click [Update] when you have finished making changes.

    Rhythmyx saves your edits to the Content Item and refreshes the screen.

  7. Click [Close].

    If you do not click [Update] before clicking [Close], a dialog displays asking you if you want to save your changes. If you click [OK], Rhythmyx saves your changes before closing the pop-up. The changes are now visible on the current page or item and in any other places where the Content Item appears.
