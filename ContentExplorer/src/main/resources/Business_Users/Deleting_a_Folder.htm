Deleting a Folder

If you have Admin permission for a folder, you may delete its sub-folders. No user has permission to delete the main Sites or main Folders node in the Content Explorer navigation panel.

To delete a folder:

  1. Select the folder to delete.
  2. In the Menu Bar, choose Content > [folder name] > Delete.

    OR

    Right-click in the Navigation Tree pane and choose Delete in the drop menu.

    Rhythmyx deletes the folder and the content links in the folder.
