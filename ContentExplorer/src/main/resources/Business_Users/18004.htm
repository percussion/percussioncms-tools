Customizing Advanced Searches When Using the Full-text Search Engine

You can customize the fields included in an advanced search. To customize advanced search

  1. Click the [Customize] button on the Content Search dialog. Rhythmyx displays the Field Selection Editor.

  2. In the Category drop list, choose the category of Content Editor fields that you want to search on:
    • System - system Content Editor fields
    • Shared - shared Content Editor fields
    • Local - local Content Editor fields
    • All - all Content Editor fields

    After you choose a category the Available Fields table displays all fields available as a selection criteria in the category.

  3. To see only fields within the selected Category that match a certain string, enter a string in Field Filter. The string matches with the Display Label column. To apply the filter to the Available Fields, click .
  4. To include a search field, click the row for the field in the Available Fields table and click . Include any number of fields.
  5. The Selected Fields table displays the fields that you include. It also displays the fields that are included by default.

    Some fields cascade. In other words, the values available in one field depend on the value you specify in another field. Cascading fields may be optional or required. If a cascading field is required, Rhythmyx includes it automatically. If the field is optional, Rhythmyx displays a message notifying you about the dependency and offering to include the dependency.

  6. When entering criteria for a Search or View, the values available in the Content Type field depend on the Community selected. If no Community is specified, all Content Types are available. However, if you specify a Community, only Content Types associated with that Community are available.
  7. To remove a search field after selecting it, click . You can remove any filed except Title (sys_title).
  8. To change the order in which the Content Search dialog displays the custom search fields, click on a field and click to move the field up, or click to move the field down.
  9. Click [OK].

    The dialog closes and Rhythmyx displays the selected fields in the Search Criteria section of the Content Search dialog.
