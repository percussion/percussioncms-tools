Content Explorer and Popup Blockers

Note that several windows used in Content Explorer, such as Content Editors, search windows, and Active Assembly windows, are displayed as popups from Content Explorer. If you have a popup blocker installed, it may prevent the display of these dialogs. You will need to configure your popup blocker to allow display of popups created by Rhythmyx.

Note that Microsoft Windows XP Service Pack 2 includes an integrated popup blocker. If you have been running an earlier version of Windows and upgrade to this operating system, you may find that some Rhythmyx functionality is blocked. You will need to configure the integrated firewall in Windows XP SP 2 to allow the display of Rhythmyx popups.
