Ad Hoc Assignment

A normal Transition assigns a content item to the Role associated with the new State of the content item.  However, you may want to assign an item to a specific individual, possibly an individual that is not in one of the States to which the item can Transition.  Use ad hoc assignment to assign a content item to a specific individual when you Transition it.  

To make an ad hoc assignment:

  1. Select the Workflow transition from the Action Menu for the Content Item.

    If Rhythmyx displays a pop-up with an Ad Hoc Assignees field, you may assign an Ad Hoc Assignee.

  2. Click the button under the Ad Hoc Assignees box.

    Rhythmyx displays the User Search page.

  3. Select the Role to which the user to whom you want to assign the content item belongs.
  4. Enter a few characters of the name of the user to whom you want to assign the content item.
  5. Click [Search].

    Rhythmyx displays User Search Results page showing all users that meet the criteria you specified.

  6. Check the user to whom you want to assign the content item.
  7. Click [OK].
