Creating New Folders

If you have Admin permissions to a folder, you can create sub-folders. (Only your System Administrator has Write access to the main Folders and Sites nodes and can create new folders under these nodes.) When you create a folder, you are automatically assigned Admin permission for the folder, and other Security settings are inherited from its parent folder.

To create a new folder:

  1. In the Menu Bar, choose Content > [folder name] > New Folder.

    OR

    Right-click a Folder in the Navigation Tree pane and choose New Folder in the drop menu.

    Rhythmyx opens the Folder Properties dialog.

  2. On the General tab:
    1. In Folder Name, enter the name of the new folder.
    2. In Folder Community, choose the Community that has access to the folder. To give all Communities access to the folder, choose All.
    3. Optionally, in Description, enter a description of the folder.
    4. In Locale, choose a Locale from the drop list.
    5. In Default Display Format, choose the default Display Format in the drop list.
    6. If you want to use a Global Template other than the default Template for the Site when assembling Content Items in the Folder, choose it in the Global Template drop list. Leave Use Default as the value if you want to use the Site Global Template for the Folder.
    7. If you want to flag the Folder for special Publishing processing, check the Publish Only in Special Edition checkbox. For more details about special Edition processing, see "Setting Up Mixed-mode Publishing" in Configuring FastForward for Web Content Management.
  3. To change the folder's security settings:
    1. Click the Security tab.
    2. Optionally, choose entries in the Names list and modify the Permissions checked for the entries.
    3. Optionally, choose [Add] to access the Folder ACL List Entry Editor and add additional entries in the Names list.
    4. Optionally, select entries in the Names list and choose [Remove] to remove their access to the folder.
  4. To add a property:
    1. Click the Properties tab.
    2. Enter the Name and Value for the Property. If you want the name of the Folder on the Site to be different from the name of the Folder in Rhythmyx, click in the Name cell and in the drop list choose sys_pubFileName. In Value, enter the name that you want the Folder to have on the Site. NOTE: Do not add properties unless directed by your system administrator.
  5. Click [OK].

    Rhythmyx closes the dialog and displays the new folder under the selected folder in the Navigation Tree.

    If you open the new folder, you will see that a Navon content item with the same name as the folder is automatically created in it. Navon's are used to create page navigation and only users in Admin Roles can modify them.
