Pasting: Copies Versus Links

Rhythmyx has copy and paste functions that let you copy a Content Item and paste it into a Content Explorer folder as a new Copy of the Content Item or as a link to the original Content Item.

To use copy and paste, right-click on a Content Item in the display pane (in a folder, search, or view) and choose Copy in the Action Menu. Then right-click on a folder in the Navigation Tree. Rhythmyx displays a Paste menu that includes the options:

  • Move
  • As Link
  • As New Copy

In most cases, you simply want to make the Content Item available from the folder, so you will choose Paste > As Link to create an icon in a Content Item folder that links to the Content Item that you selected in the Display pane. Paste > Move performs the same action, but it also deletes the Content Item icon from the original folder.

When you choose Paste > As New Copy, Rhythmyx creates a new Copy of the Content Item. The new Copy has its own content ID and moves through its Workflow separately from the original Content Item. See Adding Content to Folders for copy and paste instructions.
