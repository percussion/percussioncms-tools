Adding Related Content to a Document Slot by Copying and Pasting

To create a new Copy of a Content Item in a Slot by copying and pasting in a Document Assembly window:

  1. Display the Content Item to copy in the Display pane by selecting its Slot in the Navigation Tree.
  2. Right-click the item to copy in the Display Pane.

    Rhythmyx displays a drop menu.

  3. Choose Copy in the drop-menu.
  4. Right-click the target Slot in the Navigation tree to display a drop menu

    OR

    In the Menu Bar, choose Content > [slot type].

  5. To add the Content Item to the Slot, select Paste > As Link > [Template] or Paste> Move > [Template]. Note: If you have chosen a Slot that does not hold the Content Type or Template, the menu displays Paste > No Entries.

    If you choose Paste > As Link, Rhythmyx inserts a node for the Content Item in the Slot and leaves it in the original slot. If you choose Paste > Move, Rhythmyx inserts a node for the Content Item in the Slot and deletes it from its original slot. The node is actually a link to the Content Item in its Rhythmyx directory.

For more information see Pasting: Links versus Copies.
