Content Item Icons

An icon appears In front of each content item listed in Content Explorer's display pane.

Depending on the way your system implementer configures your Content Types, the icon or a custom icon may represent each Content Type. There are three possible choices for the icon that represents each Content Type in your system:

  • The default or non-custom icon:
  • A custom icon associated with the Content Type, such as those shown above for the FastForward Content Types Generic (EI 12 Easy Steps . . . ) and Contact (EI CEO Contact Information).
  • A custom icon associated with the type of file stored in one of the fields in the Content Type. If this choice is used, different icons may represent the same Content Type. For example, in the above graphic there are two File Content Types. The EI Sample PDF.pdf File item uses a PDF icon, and the EI Sample Word Document.doc item uses a Word icon.

Your system may use the icon to represent all Content Types, some of the FastForward icons that are appropriate for your Content Types, or additional custom icons. It is possible that your system may use a combination of these, since the icon for each Content Type is specified individually. Your Rhythmyx administrator will tell you which icons represent which Content Types.
