Saving a Search Query

To save a search query:

  1. In the Navigation Pane, click the Search Results node and in the Menu Bar, select Content > Search Results > Save As

    OR

    Right-click in the Navigation Tree pane on the Search Results node and choose Save As in the drop menu.

    The Search Results node is not present until the user creates a user-defined search.

    Rhythmyx displays the Save Search dialog.

  2. In the Search Name field, enter a name for the search.
  3. Optionally, in the Description field, enter a description for the search.
  4. Beside Show to, click the radio button that specifies to whom you want to make the search available. Options are:
    • Current user: only the user that saved the search can see and use this saved search query.
    • Current community: Only members of the Community the current user is logged into can see and use the query.
    • All Communities: All Rhythmyx users in all Communities in the system can see and use the search.
  5. Click [OK].

The Navigation Tree displays a node for the saved search under Searches. You may click this node to rerun the Search. The Search is available to other users depending on their access.
