Editing Content in a Public State

Once content has reached a public State, you cannot use a Check Out transition to check it out and edit it. However, once the item is public, the Quick Edit option is available from Action Menu for the item, and the Ignore Publish transition is available from the Workflow sub-menu.

Quick Edit checks out the item and puts it in a Quick Edit State, which makes it available to edit. Quick Edit also checks out and opens the item in its Content Editor so you may make the required edits.

The Ignore Publish transition moves the item from the public State to the Quick Edit State. The Ignore Publish transition does not check out or open the Content Item as the Quick Edit action does.

From the Quick Edit State, the only transition available other than Check In or Check Out is a publish Transition, which puts the Content Item back in the public State.
