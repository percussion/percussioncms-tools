Keyboard Shortcuts Available in Rhythmyx

When you copy and paste Content Items in Rhythmyx folders and other nodes, you can use keyboard shortcuts:

  • Copy - CTRL-C
  • Paste - CTRL-V

For more information about copying and pasting in Rhythmyx, see Pasting: Copies versus Links.
