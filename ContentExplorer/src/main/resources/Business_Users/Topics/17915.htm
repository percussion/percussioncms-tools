Grouping

Rhythmyx's full-text search supports using parentheses to group clauses to form sub queries. This can be very useful if you want to control the boolean logic for a query.

To search for either "jakarta" or "apache" and "website" in Search for, enter:

(jakarta OR apache) AND website

This eliminates any confusion and makes sure that website must exist and either term jakarta or apache may exist.
