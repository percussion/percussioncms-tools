Creating Inline Links in a Content Editor

Use an inline link when you want to add a link to text that exists in a content item. (For more details, see Using Inline Links.) Inline links are only available in content editor fields that use rich text editors (not text boxes).

To create an inline link:

  1. Select the text you want to use as the hotspot for the inline link.
  2. Click the inline link button .

    Rhythmyx displays the Content Browser.

  3. Browse or search for the content item to which you want to link.

  4. Click an item to select it.

    The Select a Template page opens. Only Page Templates are available for inline links.

  5. Choose the Template for the content item to which you want to link and click [Select].

    Rhythmyx creates a link on the selected text. (NOTE: If the target Content Item is not Public, or if a link to the target page cannot be created for some other reason, no link will be published.)

    The Content Browser automatically closes.
