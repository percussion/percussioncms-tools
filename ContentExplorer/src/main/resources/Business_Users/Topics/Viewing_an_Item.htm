Viewing an Item

When you open a Content Item using the Action Menu View > Content option, Rhythmyx displays it in a Content Editor in View mode. In View mode, you cannot edit fields.

To view content in a Content Editor:

  1. Click the Action Menu and choose View > Content (or double-click on it).

    NOTE: If a Content Item is available to you to edit, double-clicking on it opens it in a Content Editor in Edit mode; if the Content Item is only available to you to view, double-clicking on it opens it in a Content Editor in View mode.

    Rhythmyx displays all content fields on the item. None of the fields are accessible for edit. The Content Editor may display a view-only Editor Assembly window at the bottom for viewing the Content Item's related content.

    At the top of the Content Editor is a Properties link that takes you to the Properties page for the Content Item.
