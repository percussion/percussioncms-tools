Using Active Assembly to Produce Documents

To produce documents, you can use the Document Assembly window, which functions like the Content Explorer interface, but displays related content and Slots for a single Content Item. The Navigation Tree displays a main node representing the Content Item with sub-nodes for Slots. Each Slot may contain sub-node(s) for the Content Item(s) it contains, and each of these Content Items contains sub-node(s) for Slot(s), and so on. The Navigation Tree displays the Slots from the Template of the Content Item selected in the Action Menu. In each Slot, you can only insert Content Types formatted using Templates defined for that Slot.

The Navigation Tree also displays a main node for Search Results. The Search Results node contains searches for Content Items, just as it does in Content Explorer, but each search finds content that can be inserted into the selected Slot only.

A Menu Bar at the top of the Document Assembly Window has Content, View, and Help menus. The Content Menu has options similar to the Content Menu in Content Explorer; the View and Help Menus have the same options as the View and Help Menus in Content Explorer. For information about the View menu options, see Refreshing the Page and Changing Content Explorer Options.

In the Document Assembly Window Navigation Tree, you can work with Slots in the same way that you work with folders in the Content Explorer Navigation Tree. You can drag and drop or copy and paste content into Slots and access right-click menus with options for searching for, copying, and creating Content Items for a specific Slot. Depending on your system configuration, the items listed in the interface may be preceded by custom icons instead of the standard content item icon appearing in the graphic above.

Content Items also function as they do in Content Explorer. You can drag and drop Content Items from the Display pane into Slots in the Navigation Tree. You can also right-click on Content Items to access Action Menus with Preview, Copy, Edit, View, Workflow, and Preview options that perform the same functions on Content Items as they do in Content Explorer. The Action Menus in Document Assembly also include an Arrange option that lets you change an item's position in a Slot or remove it from the Slot, a Change Template option that lets you choose another Template allowed in the Slot for the Content Item, and a Help option for accessing online Help.

By default, you cannot perform Active Assembly on documents that are in a public State.
