Escaping Special Characters

Some characters have special meanings in Lucene query syntax. For example + indicates that the following term is required. However, you may want to search for a + character and not use its special meaning. In this case, you can "escape" the special meaning of the character.

The current list of special characters in Lucene are

+ - && || ! ( ) { } [ ] ^ " ~ * ? : \

To escape these character use the \ before the character. For example to search for (1+1):2 use the query:

\(1\+1\)\:2
