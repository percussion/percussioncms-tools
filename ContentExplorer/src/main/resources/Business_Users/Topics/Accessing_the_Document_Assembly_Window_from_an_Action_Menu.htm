Accessing the Document Assembly Window from an Action Menu

To access the Document Assembly Window from an Action Menu:

  1. In the Display pane, right-click the Content Item that holds the document. The Content Item for a document usually has a Template that includes the document's structure.

    Rhythmyx opens an Action Menu.

  2. In the Action Menu, select Active Assembly for Documents > [page Template].

    Rhythmyx opens the Document Assembly window for the Template of the Content Item.
