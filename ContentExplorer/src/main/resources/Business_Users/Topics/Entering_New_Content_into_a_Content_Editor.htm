Entering New Content into a Content Editor

To add new content into a Content Editor:

  1. Open a Content Editor for a new Content Item through an Action Menu in Content Explorer or Active Assembly.

  2. Enter the data for the new Content Item. Fields that have an asterisk (*) next to them are required.

    Note that the required field URL has a dummy default value to prevent an error message from interfering with content creation in Active Assembly.

  3. To save the content item, click [Insert].

    An [Update] button replaces the [Insert] button. You must click [Update] to save changes that you make after clicking [Insert]. If there are fields that you must enter through a Table Editor, the Content Editor now displays a control for adding them (in the graphic below, this control is the [Add new item] button).

  4. Enter any fields that have become available for inserting after clicking [Insert].
  5. Click [Update] to save all of your changes.
  6. Click [Close].

    If you do not click [Update] before clicking [Close], a dialog confirms that you want to save your changes. If you click [OK], Rhythmyx saves any changes you have made after clicking [Insert] before closing the Content Editor.
