Creating New Items in a Content Explorer Folder

To create new content in a folder:

You can create a new Content Item in any folder except the Main folder node. The new Content Item is actually located in a the Rhythmyx repository, and the node in the folder is a link to it.

To create new content in a folder:

  1. Select the folder and choose Content > New > Item > [Content Type] in the Menu bar

    OR

    Right-click on the folder in the Navigation Tree pane and choose New Item > [Content Type] in the drop menu.

    Rhythmyx opens a Content Editor for the selected Content Type.

  2. Create the Content Item using the Content Editor.

    When you save the Content Item, the folder displays a new node for it. The new node is a link to the new Content Item stored in the Rhythmyx repository.
