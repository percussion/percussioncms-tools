Using an Advanced Full-text Search

When you click a Slot and choose the Insert > Snippet option or when you choose to insert an inline object in a rich text editor, the Content Browser dialog opens and gives you different options for finding a content item to insert.

To access the advanced full-text search, open the simple full-text search and click . In addition to the fields included on the simple full-text search, the default version of the advanced search includes the following Content Type fields:

  • Title
  • Content ID
  • Community
  • Content Type
  • Created by
  • Start Date

as well as the Expand Query with Synonyms option.

Enter the values you want to use to search in the fields you want to search. For example, if you wanted to search for a specific Content Item and you knew its ID, you could enter the ID in the Content ID field and the search engine would return that specific Content Item.

If you want to find items with synonyms of the Search for term, check Expand Query with Synonyms. If, for example, you want to find content items that refer to the subway, you could enter the word "subway" and the search might return content items that include the words "subway", "train", and "metro". By default, Expand Query with Synonyms is not checked.
