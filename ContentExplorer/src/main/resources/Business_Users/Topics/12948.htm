Drilling into Report Data

If Lifecycle Analysis data is displayed in a pivot table, you can view the content details behind any count. To drill into the data, double-click on the cell that summarizes the data you want to view. MS Excel creates a new spreadsheet in the workbook with the detailed data for that cell.

You can also access a specific Content Item from any sheet that lists Content Items. To access a Content Item, select the row of the Content Item, then in the Menu bar choose Rhythmyx > Action Panel. This action displays the Rhythmyx Action Panel for that Content Item, which allows you to work with the Content Item in Rhythmyx.
