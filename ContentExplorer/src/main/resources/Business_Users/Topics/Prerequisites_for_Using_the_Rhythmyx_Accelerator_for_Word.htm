Prerequisites for Using the Rhythmyx Connector for Word

When you use a Content Editor that uses the Rhythmyx Connector for Word, an Active X control uploads content written in MS Word to Rhythmyx and downloads Word files from Rhythmyx to MS Word. If the Active X control is not installed, the system prompts you to install it. The OCS requires the MSXML4.dll. Click [Yes] when prompted to install the MSXML4.dll.

A certificate for this OCX is stored on the Rhythmyx server. During the download process, your browser will display a security warning, with a link to the certificate. Click on this link to start the certification process. Accept the default options on the Install Certificate dialog to install the certificate. To disable this dialog and automatically install the certificate, see the procedure below.

.

When certificate is installed, refresh your browser. If the following security warning appears, you should modify your security settings; otherwise you will see this message every time you open a Word-based Content Editor.

To Modify Security Settings in your browser:

This procedure uses Internet Explorer 5.5 as the browser; if you are using a different browser, set the corresponding settings as specified in Step 5.

  1. From the Menu bar, select Tools > Internet Options.  

    Internet Explorer displays the Internet Options dialog.

  2. Select the Security tab.

  3. In the upper portion of the dialog, select Trusted Sites.
  4. Click [Sites].

    Internet Explorer displays the Trusted sites dialog.


  5. Enter your Rhythmyx address in the Add this Web site to the zone field and click [Add].
  6. Uncheck the Require server verification check box.
  7. Click [OK].

    Internet Explorer returns to the Security tab.

  8. Click the [Custom Level] button.  

    Internet Explorer displays the Security Settings dialog.

  9. Under ActiveX Controls and Plugins, change the settings to the following:
    • Download signed ActiveX controls -- Enable (to avoid seeing the initial security warning)
    • Download unsigned ActiveX controls -- Prompt
    • Initialize and script ActiveX controls not marked as safe -- Enable (To avoid warnings for unsafe ActiveX controls)
    • Run ActiveX controls and plug-ins -- Enable
    • Script ActiveX controls marked safe for scripting -- Enable
  10. Click [OK].
