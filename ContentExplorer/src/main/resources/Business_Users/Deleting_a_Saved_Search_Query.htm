Deleting a Saved Search Query

To delete a saved search query:

  1. Right-click the Search Query node for the saved search under Searches in the Navigation Tree, and in the Menu Bar, select choose Content > [search name] > Delete.

    OR

    Right-click on the Search Query node and in the drop menu choose Delete.

    Rhythmyx deletes the search query.
