Connecting to a Server

To connect to a server, in the Menu bar, click Servers. The popup menu displays the last five connections used. If the connection you want to use is listed, click on it. If the connection is not listed, click More Connections. The Package Builder displays a popup menu with all defined server connections. Click on the connection you want to use.

If you did not save the credentials when you defined the connection, the Package Builder displays an authentication dialog. Enter the user name and password, then click [OK] to log in to the server.

If you saved the credentials, you will be logged in automatically.
