Descriptor Definition Dialog

Use the Descriptor Definition Dialog to created and modify descriptors.

To access the Descriptor Definition dialog:

  • on the Package Descriptor List dialog, click the [New] button; or
  • in the Menu bar, choose Package > New; or
  • on the Package Descriptor List dialog, select the descriptor you want to modify and click the [Edit] button; or
  • on the Package Descriptor List dialog, select the descriptor you want to modify and in the Menu bar, choose Package > Edit.

The Descriptor Definition dialog includes three tabs:

  • General tab
  • Selection tab
  • Dependencies tab
