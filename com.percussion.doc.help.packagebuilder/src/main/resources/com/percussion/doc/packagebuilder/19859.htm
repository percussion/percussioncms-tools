Modifying a Server Connection

To modify a server connection:

  1. On the Connection Manager dialog, choose Server > Connections Manager.

    The Package Builder displays the Connections Manager dialog.

  2. Select the connection you want to modify.

    The Package Builder adds the data for the connection you selected to the fields in the dialog.

  3. You can modify the data in any field. If you change the Host or Port, a new connection will be created.
  4. To save the connection, click the [Save] button. To save the connection and connect to the server, click the [Save and Connect] button.
