Building Packages

To build a package

  1. Start the process. Two options are available.
    • On the Package List dialog, in the Menu bar, choose Package > Build.
    • On the Descriptor dialog, click the button.

    Regardless of the way you start the process, the Package Builder displays the Select Target Directory dialog.

  2. Enter the directory where you want to build the package. You can use the browse button [...] to open a browse dialog you can use to choose the directory.
  3. Click the [OK] button.

    The Package Builder builds the package. When the process is complete, the Package Builder displays a completion dialog.
