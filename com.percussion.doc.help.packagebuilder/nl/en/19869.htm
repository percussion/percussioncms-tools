Defining Descriptors

A descriptor defines:

  • the design objects that will be included in a package. including any dependencies (design objects required to support the specified design objects);
  • the files used to configure the design objects in the package;l and
  • supporting files required for the design object to function.

Use the Descriptor Definition dialog to define descriptors. You can create a new descriptor, modify an existing descriptor, or delete a descriptor. You can also build a package from the descriptor.
