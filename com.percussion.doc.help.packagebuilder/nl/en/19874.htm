Descriptor Definition Dialog Selection Tab

Use the Selection tab to maintain the following features of the descriptor:

  • design elements
  • configuration files
  • supporting files

Field Descriptions

Design objects List of design objects available in the Percussion CMS Server. Check the design objects you want to add to the package. You can filter the list of design objects by entering a filter string in the field above the list. You can use the "?" and "*" characters as wildcards in the filter string. The "?" character matches one character, while the "*" character matches zero or more characters.

File Resources List of additional files required to support the design objects in the package.

Configuration Definition File The configuration definition file for the package. Optional, but if you add a configuration definition file to the descriptor, you must also add a default configuration file.

Default Configuration File The default configuration file for the package. Optional, but if you add a default configuration file to the descriptor, you must also add a configuration definition file.

Selected Only Check this box to filter the list of design objects to only those that have been selected to be included in this package.
